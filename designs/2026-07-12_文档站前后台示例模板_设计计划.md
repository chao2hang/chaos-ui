# 文档站前后台示例模板 设计计划

## 概述

在文档站增加「示例模板」入口与页面：以内嵌近全屏可交互预览的方式，展示**前台网站**与**后台管理**的精简标杆组合，并附场景说明、关联组件链接与关键接入代码。

## 背景与动机

- 现有文档以单组件详情为主，多组件组合能力散落在组件页与 Storybook。
- Header 仅有「文档」「组件」，缺少 Examples IA。
- 目标：打开 `/examples` 即可预览前后台模板组合。

## 目标与非目标

### 目标（首版）

- 文档站内两者一体：总览 + 可交互模板 + 组合说明
- 内嵌近全屏预览
- 精简标杆场景

### 非目标

- 独立 starter 包、真实鉴权/API
- 完整 ERP、`?scene=` 深链（二期）
- 改包组件 breaking API

## 设计方案

### 路由

| 路由 | 用途 |
|------|------|
| `/examples` | 总览两张主卡 |
| `/examples/admin` | 后台模板 |
| `/examples/public` | 前台模板 |

Header：示例 → `/examples`

### 场景

**后台：** login / dashboard / list / detail  
**前台：** home / article / auth  

场景切换为预览内 state，不改 URL。

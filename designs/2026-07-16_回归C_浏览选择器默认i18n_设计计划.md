# 回归 C：UserBrowse / DepartmentBrowse 默认 i18n（#19 同类）

## 概述

通用选择器默认 UI 串仍为英文（placeholder / DialogTitle / search）。对齐 ExpenseLineEditor：`useSafeTranslation` + `defaultValue` 中文，并写入 `lib/i18n/resources/*/ui.json`。

**不**在本波重开 OrderLineEditor 全量 i18n（#42 产品边界）。

## 实施清单

- [ ] `lib/i18n/resources/zh|en|ja|ko/ui.json` — userBrowse.* / departmentBrowse.*
- [ ] `components/ui/user-browse.tsx` — t() + defaultValue
- [ ] `components/ui/department-browse.tsx` — 同上
- [ ] 测试：默认中文或 key 解析；传 placeholder 仍覆盖
- [ ] `lib/i18n/config.test.ts` 如需补 key 抽样

## Keys（建议）

```
userBrowse.placeholder / title / searchPlaceholder / remove
departmentBrowse.placeholder / title / searchPlaceholder / remove
```

## 验收

- [ ] 无 i18n 时 defaultValue 中文可见
- [ ] vitest 不因文案硬断言英文失败（更新断言）

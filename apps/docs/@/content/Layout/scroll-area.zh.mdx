# ScrollArea 滚动区域

> 布局组件

自定义样式的滚动区域,支持横/纵向滚动条。

## 何时使用

需要滚动区域功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { ScrollArea, ScrollBar } from "@/components/ui/scroll-area"

export function Basic() {
  return (
    <div className="space-y-4">
  <ScrollArea />
  <ScrollBar />
    </div>
  )
}`} lang="tsx" />


## API / Props

### ScrollArea

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# ScrollArea 滚动区域

> 布局组件

自定义样式的滚动区域,支持横/纵向滚动条。

## 何时使用

当你需要「滚动区域」相关能力时使用：自定义样式的滚动区域,支持横/纵向滚动条。用于组织页面结构与区域划分；先定布局再堆业务模块。

## 代码示例

### 基础用法

<CodeBlock code={`import { ScrollArea, ScrollBar } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <ScrollArea />
  <ScrollBar />
    </div>
  )
}`} lang="tsx" />

## API / Props

### ScrollArea

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# ScrollArea

> Layout primitives for structuring pages.

Custom-styled scroll area supporting horizontal/vertical scrollbars.

## When to Use

Use ScrollArea when you need: Custom-styled scroll area supporting horizontal/vertical scrollbars. Use to structure page regions before filling in business modules.

## Examples

### Basic Usage

<CodeBlock code={`import { ScrollArea, ScrollBar } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <ScrollArea />
  <ScrollBar />
    </div>
  )
}`} lang="tsx" />

## API / Props

### ScrollArea

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

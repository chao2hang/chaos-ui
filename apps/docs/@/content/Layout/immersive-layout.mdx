# ImmersiveLayout 沉浸布局

> 布局组件 / Layout primitives for structuring pages.

沉浸布局 — 沉浸式全屏布局，隐藏侧栏/顶栏/标签页，适用于数据大屏。
ImmersiveLayout — Immersive full-screen layout that hides sidebar/header/tabs for dashboards.

## 何时使用 / When to Use

需要沉浸布局功能时使用本组件。

Use when you need a immersivelayout.

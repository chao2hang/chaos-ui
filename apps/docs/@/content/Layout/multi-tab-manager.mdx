# MultiTabManager 多标签管理

> 布局组件 / Layout primitives for structuring pages.

多标签管理 — 多标签管理器，支持右键菜单（关闭其他/右侧/刷新）与缓存策略。
MultiTabManager — Multi-tab manager with right-click menu — close others/right/refresh and cache strategy.

## 何时使用 / When to Use

当你需要「多标签管理」相关能力时使用：多标签管理器，支持右键菜单（关闭其他/右侧/刷新）与缓存策略。用于组织页面结构与区域划分；先定布局再堆业务模块。

Use MultiTabManager when you need: Multi-tab manager with right-click menu — close others/right/refresh and cache strategy. Use to structure page regions before filling in business modules.

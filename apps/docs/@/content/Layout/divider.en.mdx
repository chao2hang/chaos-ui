# Divider

> Layout primitives for structuring pages.

Content separator; supports horizontal/vertical and labels.

## When to Use

Use Divider when you need: Content separator; supports horizontal/vertical and labels. Use to structure page regions before filling in business modules.

## Examples

### Basic Usage

<CodeBlock code={`import { Divider } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Divider>Default</Divider>
}`} lang="tsx" />

## Style Variants

**orientation**

| orientation  | Description | 默认 |
| ------------ | ----------- | ---- |
| `horizontal` |             | ✓    |
| `vertical`   |             |      |

**Variant**

| Variant   | Description   | 默认 |
| --------- | ------------- | ---- |
| `default` | Default style | ✓    |
| `muted`   |               |      |
| `primary` | Primary style |      |
| `dashed`  |               |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |

## API / Props

### Divider

| Prop            | Type                            | Default  | Description                                                   |
| --------------- | ------------------------------- | -------- | ------------------------------------------------------------- |
| `className`     | `string`                        | —        | Custom CSS class                                              |
| `label`         | `React.ReactNode`               | —        | Optional text in the middle                                   |
| `labelPosition` | `"left" \| "center" \| "right"` | `center` | Text alignment                                                |
| `decorative`    | `boolean`                       | `false`  | If true, renders as a decorative separator (no semantic role) |
| ...rest         | `React.ComponentProps<"div">`   | —        | Inherits all `<div>` attributes                               |

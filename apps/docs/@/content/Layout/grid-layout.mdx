# GridLayout 栅格布局

> 布局组件 / Layout primitives for structuring pages.

栅格布局 — Chaos UI 的布局组件,用于组织页面区域结构。
GridLayout — a layout primitives for structuring pages. for structuring page regions.

## 何时使用 / When to Use

需要栅格布局功能时使用本组件。

Use when you need a grid layout.

## 代码示例 / Examples

### 基础用法 / Basic Usage

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(GridItem),使用组合模式按需导入。

# GridLayout 栅格布局

Chaos UI 栅格布局组件，基于 CSS Grid 实现，支持响应式列数配置。
A CSS Grid-based layout component with responsive column configuration.

## 何时使用 / When to use

用于页面和区块的栅格化布局，支持响应式列数调整。适合仪表盘、卡片列表、表单布局等需要多列排列的场景。

Use for grid-based page layouts with responsive columns. Ideal for dashboards, card lists, and form layouts.

## 代码示例 / Examples

### 基础用法 / Basic

<GridLayout columns={3} gap={16}>
  {Array.from({ length: 6 }).map((_, i) => (
    <GridItem key={i}>
      <div style={{ background: "#e2e8f0", padding: 20, borderRadius: 8, textAlign: "center" }}>
        Item {i + 1}
      </div>
    </GridItem>
  ))}
</GridLayout>

## API

### GridLayout

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| columns | `number` | `1` | 默认列数 | Default column count |
| gap | `number` | `16` | 列间距 (px) | Gap between items in px |
| responsive | `{ sm?: number; md?: number; lg?: number; xl?: number }` | — | 响应式断点列数 | Responsive breakpoint columns |
| alignItems | `'start' \| 'center' \| 'end' \| 'stretch'` | `'stretch'` | 垂直对齐方式 | Vertical alignment |
| className | `string` | — | 自定义样式类名 | Custom CSS class |
| children | `React.ReactNode` | — | 子项（通常为 GridItem） | Children (usually GridItem) |

### GridItem

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| span | `number` | `1` | 跨列数 | Column span count |
| rowSpan | `number` | `1` | 跨行数 | Row span count |
| start | `number` | — | 起始列位置 | Start column position |
| end | `number` | — | 结束列位置 | End column position |
| className | `string` | — | 自定义样式类名 | Custom CSS class |
| children | `React.ReactNode` | — | 子内容 | Child content |

## 注意事项

- **响应式断点**：`responsive` 使用 Tailwind 默认断点 — `sm: 640px, md: 768px, lg: 1024px, xl: 1280px`。未指定断点的列数回退到 `columns` 属性。
- **跨列优先级**：`start`/`end` 优先级高于 `span`，同时指定时以 `start`/`end` 为准。
- **无障碍**：GridLayout 和 GridItem 均使用纯 CSS Grid，无特殊 ARIA 需求。若用于列表场景，建议在外层包裹 `role="list"` 并在 GridItem 上加 `role="listitem"`。

# Grid 栅格（Row / Col）

> Layout · 24 列栅格系统

包导出的是 **`Row` + `Col`**（没有单独的 `<Grid />` 组件）。`sourcePath` 为 `components/ui/grid.tsx`。

## 何时使用

需要按 24 列划分页面区域、控制 gutter / 对齐 / 响应式跨度时使用。

## 基础用法

<CodeBlock code={`import { Row, Col } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Row gutter={16}>
      <Col span={12}>左半</Col>
      <Col span={12}>右半</Col>
    </Row>
  )
}`} lang="tsx" />

## 响应式

`Col` 支持 `xs` / `sm` / `md` / `lg` / `xl` / `xxl` 断点对象：

```tsx
<Col xs={{ span: 24 }} md={{ span: 12 }} lg={{ span: 8 }} />
```

## API

### Row

| Prop      | Type                                                                      | Default | 说明                        |
| --------- | ------------------------------------------------------------------------- | ------- | --------------------------- |
| `gutter`  | `number \| [number, number]`                                              | `0`     | 水平（或 [水平, 垂直]）间距 |
| `align`   | `top \| middle \| bottom \| stretch`                                      | `top`   | 垂直对齐                    |
| `justify` | `start \| end \| center \| space-around \| space-between \| space-evenly` | `start` | 水平对齐                    |
| `wrap`    | `boolean`                                                                 | `true`  | 是否换行                    |

### Col

| Prop       | Type                         | Default | 说明         |
| ---------- | ---------------------------- | ------- | ------------ |
| `span`     | `1–24`                       | —       | 占据列数     |
| `offset`   | `number`                     | —       | 左侧偏移列数 |
| `order`    | `number`                     | —       | 排序         |
| `xs`…`xxl` | `{ span?, offset?, order? }` | —       | 响应式       |

## 注意

# SplitPane 分割面板

> 布局组件

可拖拽分割线的双栏/多栏布局容器。

## 何时使用

需要分割面板功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { SplitPane } from "@/components/ui/split-pane"

export function Basic() {
  return <SplitPane />
}`} lang="tsx" />

## API / Props

### SplitPane

| Prop          | Type                         | Default      | 说明              |
| ------------- | ---------------------------- | ------------ | ----------------- |
| `className`   | `string`                     | —            | 自定义样式类名    |
| `direction`   | `"horizontal" \| "vertical"` | `horizontal` | 排列方向          |
| `defaultSize` | `number`                     | `50`         |                   |
| `minSize`     | `number`                     | `20`         |                   |
| `maxSize`     | `number`                     | `80`         |                   |
| `onResize`    | `() => void`                 | —            |                   |
| `first`       | `React.ReactNode`            | —            |                   |
| `second`      | `React.ReactNode`            | —            |                   |
| `resizer`     | `boolean`                    | `true`       |                   |
| ...rest       | `React.HTMLAttributes`       | —            | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

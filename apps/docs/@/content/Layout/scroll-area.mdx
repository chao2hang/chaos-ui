# ScrollArea 滚动区域

Chaos UI 滚动区域组件，基于 Base UI ScrollArea，为内容溢出提供美观的自定义滚动条。
A Base UI-powered scrollable container with custom-styled scrollbars.

## 何时使用 / When to use

当内容超出容器高度时，使用 ScrollArea 替代原生 overflow 滚动，提供统一的自定义滚动条样式。适合侧边栏、对话框内容区、日志面板等。

Use instead of native overflow scrolling when you need custom scrollbar styling. Ideal for sidebars, dialog content areas, and log panels.

## 代码示例 / Examples

### 基础用法 / Basic

<ScrollArea style={{ height: 180, width: "100%", border: "1px solid #e2e8f0", borderRadius: 8, padding: "0 12px" }}>
  <div style={{ paddingRight: 16 }}>
    {Array.from({ length: 20 }).map((_, i) => (
      <p key={i} style={{ padding: "8px 0", borderBottom: "1px solid #f1f5f9" }}>
        Item {i + 1} — 滚动查看内容 / Scroll to see more content.
      </p>
    ))}
  </div>
</ScrollArea>

## API

### ScrollArea

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| hideScrollbar | `boolean` | `false` | 隐藏滚动条（仍可滚动） | Hide scrollbar (still scrollable) |
| className | `string` | — | 自定义样式类名 | Custom CSS class |
| style | `React.CSSProperties` | — | 自定义行内样式 | Custom inline style |
| ...rest | `ScrollAreaPrimitive.Root.Props` | — | 透传 Base UI 属性 | Passthrough Base UI props |

### ScrollBar

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| orientation | `'horizontal' \| 'vertical'` | — | 滚动条方向 | Scrollbar orientation |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

## 注意事项

- **高度限制**：ScrollArea 需要显式设置高度（通过 `style` 或 `className`），否则不会出现滚动。
- **base-ui 原生行为**：Base UI ScrollArea 的 `ScrollBar` 一般由 ScrollArea 内部自动管理，无需手动使用。仅在需要自定义滚动条行为时才需直接使用 `ScrollBar`。
- **无障碍**：Base UI ScrollArea 默认不增加额外 ARIA 属性——原生滚动区域本身对屏幕阅读器友好。`hideScrollbar` 仅视觉隐藏，不影响键盘和屏幕阅读器操作。
- **性能**：大量列表项建议搭配虚拟列表（如 `VirtualList`），ScrollArea 本身不优化渲染性能。

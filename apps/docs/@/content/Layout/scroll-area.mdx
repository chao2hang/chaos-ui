# ScrollArea 滚动区域

> 布局组件 / Layout primitives for structuring pages.

自定义样式的滚动区域,支持横/纵向滚动条。
Custom-styled scroll area supporting horizontal/vertical scrollbars.

## 何时使用 / When to Use

需要滚动区域功能时使用本组件。

Use when you need a scroll area.

## 代码示例 / Examples

### 基础用法 / Basic Usage

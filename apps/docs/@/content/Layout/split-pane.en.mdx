# SplitPane

> Layout primitives for structuring pages.

Multi-pane layout container with a draggable divider.

## When to Use

Use when you need a split pane.

## Examples

### Basic Usage

<CodeBlock code={`import { SplitPane } from "@/components/ui/split-pane"

export function Basic() {
  return <SplitPane />
}`} lang="tsx" />

## API / Props

### SplitPane

| Prop          | Type                         | Default      | Description                       |
| ------------- | ---------------------------- | ------------ | --------------------------------- |
| `className`   | `string`                     | —            | Custom CSS class                  |
| `direction`   | `"horizontal" \| "vertical"` | `horizontal` | Direction                         |
| `defaultSize` | `number`                     | `50`         |                                   |
| `minSize`     | `number`                     | `20`         |                                   |
| `maxSize`     | `number`                     | `80`         |                                   |
| `onResize`    | `() => void`                 | —            |                                   |
| `first`       | `React.ReactNode`            | —            |                                   |
| `second`      | `React.ReactNode`            | —            |                                   |
| `resizer`     | `boolean`                    | `true`       |                                   |
| ...rest       | `React.HTMLAttributes`       | —            | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

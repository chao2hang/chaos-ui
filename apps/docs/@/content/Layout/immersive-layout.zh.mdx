# ImmersiveLayout 沉浸布局

> 布局组件

沉浸布局 — 沉浸式全屏布局，隐藏侧栏/顶栏/标签页，适用于数据大屏。

## 何时使用

需要沉浸布局功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { ImmersiveLayout } from "@chaos_team/chaos-ui/layout"

export function Basic() {
  return <ImmersiveLayout />
}`} lang="tsx" />

## API / Props

### ImmersiveLayout

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

# Space 间距

> 布局组件

间距 — Chaos UI 的布局组件,用于组织页面区域结构。

## 何时使用

当你需要「间距」相关能力时使用：间距 — Chaos UI 的布局组件,用于组织页面区域结构。用于组织页面结构与区域划分；先定布局再堆业务模块。

## 代码示例

### 基础用法

<CodeBlock code={`import { Space } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Space>Default</Space>
}`} lang="tsx" />

## 样式变体

**方向**

| direction    | 说明 | 默认 |
| ------------ | ---- | ---- |
| `horizontal` |      | ✓    |
| `vertical`   |      |      |

**尺寸**

| Size | 说明 | 默认 |
| ---- | ---- | ---- |
| `xs` | 超小 |      |
| `sm` | 小号 | ✓    |
| `md` |      |      |
| `lg` | 大号 |      |
| `xl` | 超大 |      |

**wrap**

| wrap    | 说明 | 默认 |
| ------- | ---- | ---- |
| `true`  |      | ✓    |
| `false` |      |      |

**对齐**

| align      | 说明 | 默认 |
| ---------- | ---- | ---- |
| `start`    |      |      |
| `center`   |      | ✓    |
| `end`      |      |      |
| `baseline` |      |      |

## API / Props

### Space

| Prop        | Type                          | Default | 说明                         |
| ----------- | ----------------------------- | ------- | ---------------------------- |
| `className` | `string`                      | —       | 自定义样式类名               |
| `split`     | `React.ReactNode`             | —       | Split items with a separator |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性        |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

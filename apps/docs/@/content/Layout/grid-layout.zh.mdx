# GridLayout 栅格布局

> 布局组件

栅格布局 — Chaos UI 的布局组件,用于组织页面区域结构。

## 何时使用

当你需要「栅格布局」相关能力时使用：栅格布局 — Chaos UI 的布局组件,用于组织页面区域结构。用于组织页面结构与区域划分；先定布局再堆业务模块。

## 代码示例

### 基础用法

<CodeBlock code={`import { GridLayout, GridItem } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <GridLayout>
      <GridItem>Content</GridItem>
    </GridLayout>
  )
}`} lang="tsx" />

## API / Props

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(GridItem),使用组合模式按需导入。

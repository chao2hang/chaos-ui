# Divider 分割线

> 布局组件

区隔内容的分割线,支持水平/垂直、含文字。

## 何时使用

需要分割线功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { Divider } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Divider>Default</Divider>
}`} lang="tsx" />

### 变体

<CodeBlock
  code={`{/* 展示 variant 变体 */}
<div className="flex flex-wrap items-center gap-3">
  <Divider variant="default">Default</Divider>
  <Divider variant="muted">Muted</Divider>
  <Divider variant="primary">Primary</Divider>
  <Divider variant="dashed">Dashed</Divider>
</div>`}
  lang="tsx"
/>

## 样式变体

**方向**

| orientation  | 说明 | 默认 |
| ------------ | ---- | ---- |
| `horizontal` |      | ✓    |
| `vertical`   |      |      |

**变体**

| Variant   | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认样式 | ✓    |
| `muted`   |          |      |
| `primary` | 主要样式 |      |
| `dashed`  |          |      |

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |

## API / Props

### Divider

| Prop            | Type                            | Default  | 说明                                                          |
| --------------- | ------------------------------- | -------- | ------------------------------------------------------------- |
| `className`     | `string`                        | —        | 自定义样式类名                                                |
| `label`         | `React.ReactNode`               | —        | Optional text in the middle                                   |
| `labelPosition` | `"left" \| "center" \| "right"` | `center` | Text alignment                                                |
| `decorative`    | `boolean`                       | `false`  | If true, renders as a decorative separator (no semantic role) |
| ...rest         | `React.ComponentProps<"div">`   | —        | 透传 `<div>` 原生属性                                         |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

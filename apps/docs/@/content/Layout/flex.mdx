# Flex 弹性布局

> 布局组件 / Layout primitives for structuring pages.

弹性布局 — Chaos UI 的布局组件,用于组织页面区域结构。
Flex — a layout primitives for structuring pages. for structuring page regions.

## 何时使用 / When to Use

需要弹性布局功能时使用本组件。

Use when you need a flex.

## 代码示例 / Examples

## API / Props

### Flex

| Prop        | Type                          | Default | 说明 / Description                                          |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# PageContainer 页面容器

> 布局组件 / Layout primitives for structuring pages.

页面容器 — Chaos UI 的布局组件,用于组织页面区域结构。
PageContainer — a layout primitives for structuring pages. for structuring page regions.

## 何时使用 / When to Use

需要页面容器功能时使用本组件。

Use when you need a page container.

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 2 个子组件(PageHeader, PageContent),使用组合模式按需导入。

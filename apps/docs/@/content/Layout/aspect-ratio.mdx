# AspectRatio 宽高比

> 布局组件 / Layout primitives for structuring pages.

锁定内容宽高比的容器组件,常用于图片/视频占位。
Container that enforces an aspect ratio; useful for image/video placeholders.

## 何时使用 / When to Use

当你需要「宽高比」相关能力时使用：锁定内容宽高比的容器组件,常用于图片/视频占位。用于组织页面结构与区域划分；先定布局再堆业务模块。

Use AspectRatio when you need: Container that enforces an aspect ratio; useful for image/video placeholders. Use to structure page regions before filling in business modules.

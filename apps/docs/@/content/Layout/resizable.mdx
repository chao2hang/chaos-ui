# Resizable 可调整大小

> 布局组件 / Layout primitives for structuring pages.

可拖拽调整子面板尺寸的布局容器。
Layout container whose panels can be resized by dragging.

## 何时使用 / When to Use

需要可调整大小功能时使用本组件。

Use when you need a resizable.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <ResizablePanelGroup />
  <ResizablePanel />
  <ResizableHandle />
    </div>
  )
}`} lang="tsx" />

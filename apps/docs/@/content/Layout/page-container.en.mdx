# PageContainer

> Layout primitives for structuring pages.

PageContainer — a layout primitives for structuring pages. for structuring page regions.

## When to Use

Use PageContainer when this capability is required. Use to structure page regions before filling in business modules.

## Examples

### Basic Usage

<CodeBlock code={`import { PageContainer, PageHeader, PageContent } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <PageContainer>
      <PageContent>Content</PageContent>
    </PageContainer>
  )
}`} lang="tsx" />

## Style Variants

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `sm`      | Small        |      |
| `default` | Default size | ✓    |
| `lg`      | Large        |      |
| `full`    |              |      |
| `none`    |              |      |

**padding**

| padding   | Description | 默认 |
| --------- | ----------- | ---- |
| `none`    |             |      |
| `sm`      |             |      |
| `default` |             | ✓    |
| `lg`      |             |      |

**center**

| center  | Description | 默认 |
| ------- | ----------- | ---- |
| `true`  |             | ✓    |
| `false` |             |      |

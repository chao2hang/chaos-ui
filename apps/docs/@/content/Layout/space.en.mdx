# Space

> Layout primitives for structuring pages.

Space — a layout primitives for structuring pages. for structuring page regions.

## When to Use

Use Space when this capability is required. Use to structure page regions before filling in business modules.

## Examples

### Basic Usage

<CodeBlock code={`import { Space } from "@chaos_team/chaos-ui/ui"

  return <Space>Default</Space>

## Style Variants

**direction**

| direction    | Description | 默认 |
| ------------ | ----------- | ---- |
| `horizontal` |             | ✓    |
| `vertical`   |             |      |

**Size**

| Size | Description | 默认 |
| ---- | ----------- | ---- |
| `xs` | Extra small |      |
| `sm` | Small       | ✓    |
| `md` |             |      |
| `lg` | Large       |      |
| `xl` | Extra large |      |

**wrap**

| wrap    | Description | 默认 |
| ------- | ----------- | ---- |
| `true`  |             | ✓    |
| `false` |             |      |

**align**

| align      | Description | 默认 |
| ---------- | ----------- | ---- |
| `start`    |             |      |
| `center`   |             | ✓    |
| `end`      |             |      |
| `baseline` |             |      |

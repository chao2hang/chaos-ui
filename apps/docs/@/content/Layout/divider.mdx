# Divider 分割线

> 布局组件 / Layout primitives for structuring pages.

区隔内容的分割线,支持水平/垂直、含文字。
Content separator; supports horizontal/vertical and labels.

## 何时使用 / When to Use

需要分割线功能时使用本组件。

Use when you need a divider.

## API / Props

### Divider

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `label` | `React.ReactNode` | — | Optional text in the middle |
| `labelPosition` | `"left" \| "center" \| "right"` | `center` | Text alignment |
| `decorative` | `boolean` | `false` | If true, renders as a decorative separator (no semantic role) |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

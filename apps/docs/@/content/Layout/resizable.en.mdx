# Resizable

> Layout primitives for structuring pages.

Layout container whose panels can be resized by dragging.

## When to Use

Use Resizable when you need: Layout container whose panels can be resized by dragging. Use to structure page regions before filling in business modules.

## Examples

### Basic Usage

<CodeBlock code={`import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <ResizablePanelGroup />
  <ResizablePanel />
  <ResizableHandle />
    </div>
  )
}`} lang="tsx" />

## API / Props

### SplitPane

| Prop          | Type                         | Default      | 说明 / Description                                        |
| ------------- | ---------------------------- | ------------ | --------------------------------------------------------- |
| `className`   | `string`                     | —            | 自定义样式类名 <br/> Custom CSS class                     |
| `direction`   | `"horizontal" \| "vertical"` | `horizontal` | 排列方向 <br/> Direction                                  |
| `defaultSize` | `number`                     | `50`         |                                                           |
| `minSize`     | `number`                     | `20`         |                                                           |
| `maxSize`     | `number`                     | `80`         |                                                           |
| `onResize`    | `() => void`                 | —            |                                                           |
| `first`       | `React.ReactNode`            | —            |                                                           |
| `second`      | `React.ReactNode`            | —            |                                                           |
| `resizer`     | `boolean`                    | `true`       |                                                           |
| ...rest       | `React.HTMLAttributes`       | —            | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

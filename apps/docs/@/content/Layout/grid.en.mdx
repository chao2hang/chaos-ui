# Grid (Row / Col)

> Layout · 24-column grid system

The package exports **`Row` + `Col`** (there is no standalone `<Grid />`). Source: `components/ui/grid.tsx`.

## When to use

Use the 24-column Row/Col system to structure content with gutters and responsive spans.

## Basic

<CodeBlock code={`import { Row, Col } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Row gutter={16}>
      <Col span={12}>Left</Col>
      <Col span={12}>Right</Col>
    </Row>
  )
}`} lang="tsx" />

## API

### Row

| Prop      | Type                                 | Default | Description                   |
| --------- | ------------------------------------ | ------- | ----------------------------- |
| `gutter`  | `number \| [number, number]`         | `0`     | Horizontal (or [h, v]) gutter |
| `align`   | `top \| middle \| bottom \| stretch` | `top`   | Vertical align                |
| `justify` | `start \| end \| center \| space-*`  | `start` | Horizontal align              |
| `wrap`    | `boolean`                            | `true`  | Wrap columns                  |

### Col

| Prop       | Type     | Description                  |
| ---------- | -------- | ---------------------------- |
| `span`     | `1–24`   | Columns to span              |
| `offset`   | `number` | Offset columns               |
| `xs`…`xxl` | object   | Responsive span/offset/order |

## Note

Responsive span, offset, and order use package CSS-variable rules and do not depend on consumer Tailwind scanning runtime classes. Storybook: `Components/Grid`.

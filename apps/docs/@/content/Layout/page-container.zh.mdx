# PageContainer 页面容器

> 布局组件

页面容器 — Chaos UI 的布局组件,用于组织页面区域结构。

## 何时使用

当你需要「页面容器」相关能力时使用：页面容器 — Chaos UI 的布局组件,用于组织页面区域结构。用于组织页面结构与区域划分；先定布局再堆业务模块。

## 代码示例

### 基础用法

<CodeBlock code={`import { PageContainer, PageHeader, PageContent } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <PageContainer>
      <PageContent>Content</PageContent>
    </PageContainer>
  )
}`} lang="tsx" />

### 尺寸

<CodeBlock
  code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <PageContainer size="sm">Sm</PageContainer>
  <PageContainer size="default">Default</PageContainer>
  <PageContainer size="lg">Lg</PageContainer>
</div>`}
  lang="tsx"
/>

## 样式变体

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `sm`      | 小号     |      |
| `default` | 默认尺寸 | ✓    |
| `lg`      | 大号     |      |
| `full`    |          |      |
| `none`    |          |      |

**padding**

| padding   | 说明 | 默认 |
| --------- | ---- | ---- |
| `none`    |      |      |
| `sm`      |      |      |
| `default` |      | ✓    |
| `lg`      |      |      |

**center**

| center  | 说明 | 默认 |
| ------- | ---- | ---- |
| `true`  |      | ✓    |
| `false` |      |      |

## API / Props

### PageContainer

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 2 个子组件(PageHeader, PageContent),使用组合模式按需导入。

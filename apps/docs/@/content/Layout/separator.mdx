# Separator 分隔符

> 布局组件 / Layout primitives for structuring pages.

对内容区进行视觉分隔,通常为细线。
Visual separator between content areas, typically a thin line.

## 何时使用 / When to Use

在视觉上分隔内容区域或菜单项。

Visually separate content regions or menu items.

# Button 按钮

> 通用基础组件

触发操作或事件,如提交表单、打开对话框、取消操作等。

## 何时使用

点击触发操作,如提交表单、打开弹窗、确认/取消操作。

### 基础用法

<CodeBlock code={`import { Button } from "@chaos_team/chaos-ui/ui"

## 样式变体

**变体**

| Variant       | 说明             | 默认 |
| ------------- | ---------------- | ---- |
| `default`     | 默认样式         | ✓    |
| `outline`     | 描边样式         |      |
| `secondary`   | 次要样式         |      |
| `ghost`       | 幽灵(无背景)样式 |      |
| `destructive` | 危险/删除样式    |      |
| `link`        | 链接样式         |      |

**尺寸**

| Size      | 说明       | 默认 |
| --------- | ---------- | ---- |
| `default` | 默认尺寸   | ✓    |
| `xs`      | 超小       |      |
| `sm`      | 小号       |      |
| `lg`      | 大号       |      |
| `icon`    | 图标正方形 |      |
| `icon-xs` | 超小图标   |      |
| `icon-sm` | 小图标     |      |
| `icon-lg` | 大图标     |      |

## API / Props

### Button

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

## API / Props

### InfiniteScroll

| Prop              | Type                          | Default        | 说明 / Description                                                                                       |
| ----------------- | ----------------------------- | -------------- | -------------------------------------------------------------------------------------------------------- |
| `className`       | `string`                      | —              | 自定义样式类名 <br/> Custom CSS class                                                                    |
| `onLoadMore`      | `() => void`                  | —              | Called when the sentinel enters the viewport and more data is available / 哨兵进入视口且有更多数据时调用 |
| `hasMore`         | `boolean`                     | `true`         | Whether more data exists / 是否还有更多数据                                                              |
| `loading`         | `boolean`                     | `false`        | Whether currently loading / 是否正在加载                                                                 |
| `loader`          | `React.ReactNode`             | —              | Custom loading indicator / 自定义加载指示器                                                              |
| `threshold`       | `string`                      | `200px`        | Distance from viewport bottom to trigger (CSS margin string, default "200px") / 触发距离                 |
| `scrollContainer` | `HTMLElement \| null`         | —              | Custom scroll container element / 自定义滚动容器                                                         |
| `endMessage`      | `React.ReactNode`             | `No more data` | Text shown when no more data / 无更多数据时显示的文本                                                    |
| ...rest           | `React.ComponentProps<"div">` | —              | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes                                              |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

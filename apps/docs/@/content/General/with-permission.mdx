## API / Props

### WithPermission

| Prop          | Type                   | Default | 说明 / Description                                        |
| ------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`   | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `permissions` | `string[]`             | —       | 当前用户拥有的权限列表 / current user's permissions       |
| `require`     | `string \| string[]`   | —       | 需要的权限 / required permission(s)                       |
| `mode`        | `"any" \| "all"`       | `any`   | 校验模式：any=任一满足，all=全部满足 / "any" \| "all"     |
| `fallback`    | `React.ReactNode`      | `null`  | 不满足时渲染的兜底内容 / fallback content                 |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

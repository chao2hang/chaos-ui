# WithPermission 权限门控

> 权限门控组件 / Permission gate component.

权限门控组件，根据当前用户权限决定渲染 children 或兜底内容。
Permission gate component; renders children or fallback based on current user permissions.

## 何时使用 / When to Use

需要根据用户权限控制 UI 元素显示时使用，如按钮、菜单项、操作区域。
Use when you need to control UI element visibility based on user permissions, such as buttons, menu items, or action areas.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { WithPermission } from "@/components/ui/with-permission"

export function Basic() {
  return (
    <WithPermission permissions={["order:read"]} require="order:read">
      <button>View Orders</button>
    </WithPermission>
  )
}`} lang="tsx" />

### 多权限 & Fallback / Multiple Permissions & Fallback

<CodeBlock code={`import { WithPermission } from "@/components/ui/with-permission"

export function Multiple() {
  return (
    <WithPermission
      permissions={["order:read"]}
      require={["order:approve", "order:reject"]}
      mode="any"
      fallback={<span className="text-muted-foreground">No permission</span>}
    >
      <button>Approve Order</button>
    </WithPermission>
  )
}`} lang="tsx" />

## API / Props

### WithPermission

| Prop          | Type                 | Default | 说明 / Description                                           |
| ------------- | -------------------- | ------- | ------------------------------------------------------------ |
| `permissions` | `string[]`           | `[]`    | Current user's permissions / 当前用户权限列表                |
| `require`     | `string \| string[]` | —       | Required permission(s) / 需要的权限                          |
| `mode`        | `"any" \| "all"`     | `any`   | Check mode: any=all-satisfied, all=every-required / 校验模式 |
| `fallback`    | `React.ReactNode`    | `null`  | Fallback content / 兜底内容                                  |
| `children`    | `React.ReactNode`    | —       | Guarded children / 受控子内容                                |
| `className`   | `string`             | —       | 自定义样式类名 <br/> Custom CSS class                        |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **校验模式**: `mode="any"` 任一权限满足即渲染，`mode="all"` 需全部权限满足。

# Typography 排版

> 排版组件族 / Typography component suite.

排版组件族：H1–H6、Text、Paragraph、Blockquote、InlineCode 等，统一文本层级与样式。
Typography component suite: H1–H6, Text, Paragraph, Blockquote, InlineCode, and more for consistent text hierarchy.

## 何时使用 / When to Use

当你需要「排版（Typography）」解决具体交互/展示问题时使用；优先看下方示例与 API，按业务字段接入。

Use when you need Typography for a concrete interaction; follow the examples and API below.

## 代码示例 / Examples

### 标题 / Headings

<CodeBlock code={`import { H1, H2, H3 } from "@chaos_team/chaos-ui/ui"

export function Headings() {
  return (
    <div className="space-y-2">
      <H1>Heading 1</H1>
      <H2>Heading 2</H2>
      <H3>Heading 3</H3>
    </div>
  )
}`} lang="tsx" />

### 文本变体 / Text Variants

<CodeBlock code={`import { Text } from "@chaos_team/chaos-ui/ui"

export function TextVariants() {
  return (
    <div className="flex flex-col gap-2">
      <Text variant="default">Default text</Text>
      <Text variant="muted">Muted text</Text>
      <Text variant="primary">Primary text</Text>
      <Text variant="destructive">Destructive text</Text>
      <Text variant="success">Success text</Text>
    </div>
  )
}`} lang="tsx" />

### 侧栏语义变体 / Sidebar Semantic Variants

`Text` 支持侧栏（Sidebar）语义颜色变体，适用于侧栏菜单、面包屑等布局场景：

<CodeBlock code={`import { Text } from "@chaos_team/chaos-ui/ui"

export function SidebarText() {
  return (
    <div className="bg-sidebar rounded-lg p-4 flex flex-col gap-2">
      <Text variant="sidebar-foreground" size="sm" weight="medium">
        Menu Item (active)
      </Text>
      <Text variant="sidebar-muted" size="sm">
        Menu Item (inactive, 60% opacity)
      </Text>
      <Text variant="sidebar-accent" size="sm" weight="medium">
        Accent text
      </Text>
    </div>
  )
}`} lang="tsx" />

### 段落与引用 / Paragraph & Blockquote

<CodeBlock code={`import { Paragraph, Blockquote, InlineCode } from "@chaos_team/chaos-ui/ui"

## API / Props

### Text

| Prop        | Type                                                                                                                                                         | Default   | 说明 / Description            |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------- | ----------------------------- |
| `size`      | `"xs" \| "sm" \| "base" \| "lg" \| "xl" \| "2xl"`                                                                                                            | `base`    | Text size / 文本大小          |
| `weight`    | `"normal" \| "medium" \| "semibold" \| "bold"`                                                                                                               | `normal`  | Text weight / 文本粗细        |
| `variant`   | `"default" \| "muted" \| "primary" \| "secondary" \| "destructive" \| "success" \| "warning" \| "sidebar-foreground" \| "sidebar-muted" \| "sidebar-accent"` | `default` | Text color variant / 文本颜色 |
| `italic`    | `boolean`                                                                                                                                                    | `false`   | Italic / 斜体                 |
| `underline` | `boolean`                                                                                                                                                    | `false`   | Underline / 下划线            |
| `strike`    | `boolean`                                                                                                                                                    | `false`   | Strikethrough / 删除线        |
| `truncate`  | `boolean`                                                                                                                                                    | `false`   | Truncate / 截断               |

### Typography 命名空间 / Typography Namespace

| 成员 / Member                     | 说明 / Description             |
| --------------------------------- | ------------------------------ |
| `Typography.H1` – `Typography.H6` | 标题组件 / Heading components  |
| `Typography.Text`                 | 文本组件 / Text component      |
| `Typography.Paragraph`            | 段落组件 / Paragraph component |
| `Typography.Blockquote`           | 引用块 / Blockquote            |
| `Typography.InlineCode`           | 行内代码 / Inline code         |
| `Typography.Lead`                 | 引导文本 / Lead text           |
| `Typography.Large`                | 大号文本 / Large text          |
| `Typography.Small`                | 小号文本 / Small text          |
| `Typography.Muted`                | 静默文本 / Muted text          |
| `Typography.List`                 | 列表 / List                    |

## 注意事项 / Notes

- **命名空间**: 所有排版组件可通过 `Typography.H1` 命名空间或独立命名导入 `H1` 使用。
- **可组合**: 所有组件支持 `className` 和原生 HTML 属性透传。

# SequencePreview 序列预览

> 编号预览组件 / Sequence preview component.

编号预览组件，展示编号规则生成的示例，含前缀、日期和补零序号。
Sequence preview — displays a generated sequence example with prefix, date, and zero-filled number.

## 何时使用 / When to Use

配置编号规则时预览生成结果时使用，如订单号、工单号、流水号规则配置。
Use when configuring sequence/numbering rules to preview the generated result, such as order numbers or ticket numbers.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { SequencePreview } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <SequencePreview prefix="ORD" seq={1} zeroFill={4} />
}`} lang="tsx" />

### 自定义格式 / Custom Format

<CodeBlock code={`import { SequencePreview } from "@chaos_team/chaos-ui/ui"

export function CustomFormat() {
  return (
    <div className="flex flex-col gap-2">
      <SequencePreview prefix="INV" dateFormat="YYMMDD" seq={42} zeroFill={6} />
      <SequencePreview prefix="TKT" dateFormat="none" seq={100} separator="/" />
    </div>
  )
}`} lang="tsx" />

## API / Props

### SequencePreview

| Prop         | Type                               | Default      | 说明 / Description                    |
| ------------ | ---------------------------------- | ------------ | ------------------------------------- |
| `prefix`     | `string`                           | `""`         | Prefix / 前缀                         |
| `date`       | `Date`                             | `new Date()` | Date used in sequence / 编号中的日期  |
| `dateFormat` | `"YYYYMMDD" \| "YYMMDD" \| "none"` | `YYYYMMDD`   | Date format / 日期格式                |
| `seq`        | `number`                           | `1`          | Sequence number / 序号                |
| `zeroFill`   | `number`                           | `4`          | Zero-fill width / 补零位数            |
| `separator`  | `string`                           | `"-"`        | Separator / 分隔符                    |
| `className`  | `string`                           | —            | 自定义样式类名 <br/> Custom CSS class |

## 注意事项 / Notes

- **纯展示组件**: 仅展示生成的编号，不涉及实际编号分配逻辑。

## API / Props

### AudioPlayer

| Prop         | Type                   | Default | 说明 / Description                                        |
| ------------ | ---------------------- | ------- | --------------------------------------------------------- |
| `className`  | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `src`        | `string`               | —       | Audio source URL.                                         |
| `autoPlay`   | `boolean`              | —       | Whether to auto-play on mount. Default false.             |
| `showVolume` | `boolean`              | `true`  | Whether to show volume control. Default true.             |
| `showSpeed`  | `boolean`              | —       | Whether to show playback speed selector. Default false.   |
| `onEnded`    | `() => void`           | —       | Called when playback ends.                                |
| `onPlay`     | `() => void`           | —       | Called on play/pause/timeline interaction.                |
| `onPause`    | `() => void`           | —       |                                                           |
| ...rest      | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

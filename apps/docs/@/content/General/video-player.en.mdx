# VideoPlayer

> Styled video player with play/pause, seek bar, volume, and fullscreen.

Styled video player with play/pause, seek bar, volume, and fullscreen toggle.

## When to Use

Use VideoPlayer when you need: Styled video player with play/pause, seek bar, volume, and fullscreen toggle. Use as a shared primitive across the app; prefer reuse over reinventing in feature code.

## Examples

### Basic Usage

<CodeBlock code={`import { VideoPlayer } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <VideoPlayer src="/video/sample.mp4" />
}`} lang="tsx" />

### With Poster

<CodeBlock code={`import { VideoPlayer } from "@chaos_team/chaos-ui/ui"

export function WithPoster() {
  return (
    <VideoPlayer
      src="/video/sample.mp4"
      poster="/images/poster.jpg"
    />
  )
}`} lang="tsx" />

## API / Props

### VideoPlayer

| Prop           | Type         | Default | Description                              |
| -------------- | ------------ | ------- | ---------------------------------------- |
| `src`          | `string`     | —       | Video source URL / 视频源地址 (required) |
| `poster`       | `string`     | —       | Poster image URL / 封面图地址            |
| `autoPlay`     | `boolean`    | `false` | Auto-play on mount / 自动播放            |
| `showControls` | `boolean`    | `true`  | Show controls overlay / 显示控制层       |
| `className`    | `string`     | —       | Custom CSS class                         |
| `onEnded`      | `() => void` | —       | Playback end callback / 播放结束回调     |
| `onPlay`       | `() => void` | —       | Play callback / 播放回调                 |
| `onPause`      | `() => void` | —       | Pause callback / 暂停回调                |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

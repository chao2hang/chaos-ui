## API / Props

### SplitButton

| Prop        | Type                   | Default | Description                             |
| ----------- | ---------------------- | ------- | --------------------------------------- |
| `className` | `string`               | —       | Custom CSS class                        |
| `label`     | `React.ReactNode`      | —       | Main button label                       |
| `onClick`   | `() => void`           | —       | Main button click handler               |
| `items`     | `SplitButtonItem[]`    | —       | Dropdown menu items                     |
| `onSelect`  | `() => void`           | —       | Called when a dropdown item is selected |
| `disabled`  | `boolean`              | —       | Whether disabled                        |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes       |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

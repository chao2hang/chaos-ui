# Button

> General-purpose primitives used across the app.

Buttons trigger actions or events such as submitting forms, opening dialogs, or cancelling operations.

## When to Use

Clicking triggers an action such as form submission, opening a dialog, or confirming/cancelling.

## Style Variants

**Variant**

| Variant       | Description                 | 默认 |
| ------------- | --------------------------- | ---- |
| `default`     | Default style               | ✓    |
| `outline`     | Outlined style              |      |
| `secondary`   | Secondary action style      |      |
| `ghost`       | Ghost (no background) style |      |
| `destructive` | Destructive/danger style    |      |
| `link`        | Link style                  |      |

**Size**

| Size      | Description      | 默认 |
| --------- | ---------------- | ---- |
| `default` | Default size     | ✓    |
| `xs`      | Extra small      |      |
| `sm`      | Small            |      |
| `lg`      | Large            |      |
| `icon`    | Icon-only square |      |
| `icon-xs` | Extra small icon |      |
| `icon-sm` | Small icon       |      |
| `icon-lg` | Large icon       |      |

## API / Props

### Button

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

# SplitButton 分裂按钮

> 分割按钮

带主操作区域和下拉菜单触发器的分割按钮，主按钮执行主要操作，下拉提供更多选项。

## 何时使用

主操作明确但有多个次要操作时使用，如"保存"主按钮 + "另存为/导出"下拉。

## 代码示例

### 基础用法

<CodeBlock code={`import { SplitButton } from "@/components/ui/split-button"

export function Basic() {
  return (
    <SplitButton onClick={() => console.log("save")}>
      Save
    </SplitButton>
  )
}`} lang="tsx" />

### 带下拉操作

<CodeBlock code={`import { SplitButton } from "@/components/ui/split-button"

export function WithActions() {
  return (
    <SplitButton
      onClick={() => console.log("save")}
      actions={[
        { label: "Save As", onClick: () => console.log("save-as") },
        { label: "Export PDF", onClick: () => console.log("export") },
        { label: "Delete", destructive: true, separator: true, onClick: () => console.log("delete") },
      ]}
    >
      Save
    </SplitButton>
  )
}`} lang="tsx" />

## API / Props

### SplitButton

| Prop       | Type                                                                | Default   | 说明                                       |
| ---------- | ------------------------------------------------------------------- | --------- | ------------------------------------------ |
| `children` | `React.ReactNode`                                                   | —         | Main button label / 主按钮文本 (required)  |
| `onClick`  | `() => void`                                                        | —         | Main button click handler / 主按钮点击回调 |
| `actions`  | `SplitButtonAction[]`                                               | —         | Dropdown actions / 下拉操作                |
| `variant`  | `"default" \| "outline" \| "secondary" \| "ghost" \| "destructive"` | `default` | Button variant / 按钮变体                  |
| `size`     | `"default" \| "xs" \| "sm" \| "lg"`                                 | `default` | Button size / 按钮大小                     |
| `disabled` | `boolean`                                                           | `false`   | Disable / 禁用                             |
| `icon`     | `React.ReactNode`                                                   | —         | Main button icon / 主按钮图标              |
| `loading`  | `boolean`                                                           | `false`   | Loading state / 加载状态                   |

### SplitButtonAction

| Prop          | Type              | 说明                               |
| ------------- | ----------------- | ---------------------------------- |
| `label`       | `string`          | Action label / 操作文本 (required) |
| `icon`        | `React.ReactNode` | Leading icon / 前置图标            |
| `onClick`     | `() => void`      | Click handler / 点击回调           |
| `disabled`    | `boolean`         | Disabled / 禁用                    |
| `destructive` | `boolean`         | Destructive styling / 危险样式     |
| `separator`   | `boolean`         | Show separator before / 显示分隔符 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **基于 Button**: 主按钮和下拉触发器复用 Button 组件的 variant/size 系统。

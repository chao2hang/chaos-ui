# ConfigProvider 全局配置

> 通用基础组件 / General-purpose primitives used across the app.

为组件树提供全局配置(主题、尺寸、国际化、前缀)。
Provides global configuration (theme, size, i18n, prefix) to the component tree.

## 何时使用 / When to Use

需要全局配置功能时使用本组件。

Use when you need a config provider.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { ConfigProvider, ConfigContext, Config } from "@/components/ui/config-provider"

export function Basic() {
  return (
    <div className="space-y-4">
  <ConfigProvider />
  <ConfigContext />
  <Config />
    </div>
  )
}`} lang="tsx" />

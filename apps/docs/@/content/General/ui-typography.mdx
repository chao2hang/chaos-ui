## API / Props

### Typography

| Prop        | Type                           | Default | 说明 / Description                                            |
| ----------- | ------------------------------ | ------- | ------------------------------------------------------------- |
| `className` | `string`                       | —       | 自定义样式类名 <br/> Custom CSS class                         |
| ...rest     | `React.ComponentProps<"span">` | —       | 透传 `<span>` 原生属性 <br/> Inherits all `<span>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(Title),使用组合模式按需导入。
- **原生属性**: 继承所有 `<span>` 原生 HTML 属性。

# Spin 加载中

> 通用基础组件

加载中 — Chaos UI 的通用基础组件。

## 何时使用

需要加载中功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { Spin } from "@/components/ui/spin"

export function Basic() {
  return <Spin>Default</Spin>
}`} lang="tsx" />

### 尺寸

<CodeBlock
  code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">

</div>`}
  lang="tsx"
/>

## 样式变体

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |
| `sm`      | 小号     |      |
| `lg`      | 大号     |      |

## API / Props

### Spin

| Prop        | Type                          | Default | 说明                              |
| ----------- | ----------------------------- | ------- | --------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名                    |
| `spinning`  | `boolean`                     | `true`  | Whether to show spinner           |
| `indicator` | `React.ReactNode`             | —       | Custom spinner indicator          |
| `tip`       | `React.ReactNode`             | —       | Tip text under spinner            |
| `delay`     | `number`                      | `0`     | Delay before showing spinner (ms) |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性             |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

## Style Variants

**Variant**

| Variant       | Description              | 默认 |
| ------------- | ------------------------ | ---- |
| `default`     | Default style            | ✓    |
| `primary`     | Primary style            |      |
| `success`     | Success style            |      |
| `warning`     | Warning style            |      |
| `destructive` | Destructive/danger style |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |
| `xl`      | Extra large  |      |

## API / Props

### NumberTicker

| Prop        | Type                           | Default | Description                      |
| ----------- | ------------------------------ | ------- | -------------------------------- |
| `className` | `string`                       | —       | Custom CSS class                 |
| `value`     | `number`                       | —       | Target value                     |
| `duration`  | `number`                       | `1000`  | Duration of animation in ms      |
| `decimals`  | `number`                       | `0`     | Decimal places to show           |
| `prefix`    | `string`                       | —       | Prefix text                      |
| `suffix`    | `string`                       | —       | Suffix text                      |
| `easing`    | `boolean`                      | `true`  | Use easing instead of linear     |
| ...rest     | `React.ComponentProps<"span">` | —       | Inherits all `<span>` attributes |

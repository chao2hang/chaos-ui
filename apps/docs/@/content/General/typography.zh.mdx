# Typography 排版

> 排版组件族

排版组件族：H1–H6、Text、Paragraph、Blockquote、InlineCode 等，统一文本层级与样式。

## 何时使用

需要统一文本层级、颜色、大小和样式时使用，替代手写 Tailwind 类名。

## 代码示例

### 标题

<CodeBlock code={`import { H1, H2, H3 } from "@/components/ui/typography"

export function Headings() {
  return (
    <div className="space-y-2">
      <H1>Heading 1</H1>
      <H2>Heading 2</H2>
      <H3>Heading 3</H3>
    </div>
  )
}`} lang="tsx" />

### 文本变体

<CodeBlock code={`import { Text } from "@/components/ui/typography"

export function TextVariants() {
  return (
    <div className="flex flex-col gap-2">
      <Text variant="default">Default text</Text>
      <Text variant="muted">Muted text</Text>
      <Text variant="primary">Primary text</Text>
      <Text variant="destructive">Destructive text</Text>
      <Text variant="success">Success text</Text>
    </div>
  )
}`} lang="tsx" />

### 段落与引用

<CodeBlock code={`import { Paragraph, Blockquote, InlineCode } from "@/components/ui/typography"

export function Misc() {
  return (
    <div className="space-y-3">
      <Paragraph>This is a paragraph with relaxed leading.</Paragraph>
      <Paragraph muted>This is a muted paragraph.</Paragraph>
      <Blockquote>This is a blockquote with a left border.</Blockquote>
      <p>Use <InlineCode>InlineCode</InlineCode> for code snippets.</p>
    </div>
  )
}`} lang="tsx" />

## API / Props

### Text

| Prop | Type | Default | 说明 |
| ----------- | --------------------------------------------------------------------------------------------- | --------- | ----------------------------- |
| `size`      | `"xs" \| "sm" \| "base" \| "lg" \| "xl" \| "2xl"`                                             | `base`    | Text size / 文本大小          |
| `weight`    | `"normal" \| "medium" \| "semibold" \| "bold"`                                                | `normal`  | Text weight / 文本粗细        |
| `variant`   | `"default" \| "muted" \| "primary" \| "secondary" \| "destructive" \| "success" \| "warning"` | `default` | Text color variant / 文本颜色 |
| `italic`    | `boolean`                                                                                     | `false`   | Italic / 斜体                 |
| `underline` | `boolean`                                                                                     | `false`   | Underline / 下划线            |
| `strike`    | `boolean`                                                                                     | `false`   | Strikethrough / 删除线        |
| `truncate`  | `boolean`                                                                                     | `false`   | Truncate / 截断               |

### Typography 命名空间

| 成员 | 说明 |
| --------------------------------- | ------------------------------ |
| `Typography.H1` – `Typography.H6` | 标题组件 |
| `Typography.Text` | 文本组件 |
| `Typography.Paragraph` | 段落组件 |
| `Typography.Blockquote` | 引用块 |
| `Typography.InlineCode` | 行内代码 |
| `Typography.Lead` | 引导文本 |
| `Typography.Large` | 大号文本 |
| `Typography.Small` | 小号文本 |
| `Typography.Muted` | 静默文本 |
| `Typography.List` | 列表 |

## 注意事项

- **命名空间**: 所有排版组件可通过 `Typography.H1` 命名空间或独立命名导入 `H1` 使用。
- **可组合**: 所有组件支持 `className` 和原生 HTML 属性透传。

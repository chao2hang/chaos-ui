## Style Variants

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

## API / Props

### Spin

| Prop        | Type                          | Default | Description                       |
| ----------- | ----------------------------- | ------- | --------------------------------- |
| `className` | `string`                      | —       | Custom CSS class                  |
| `spinning`  | `boolean`                     | `true`  | Whether to show spinner           |
| `indicator` | `React.ReactNode`             | —       | Custom spinner indicator          |
| `tip`       | `React.ReactNode`             | —       | Tip text under spinner            |
| `delay`     | `number`                      | `0`     | Delay before showing spinner (ms) |
| ...rest     | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes   |

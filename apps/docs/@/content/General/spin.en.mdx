# Spin

> Spin wrapper for partial loading.

Spin wrapper for partial loading; overlays a spinner on children with optional delay and tip.

## When to Use

Use when a section of the page is loading. Wraps content and shows a spinner overlay.

## Examples

### Basic Usage

<CodeBlock code={`import { Spin } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Spin spinning>Content</Spin>
}`} lang="tsx" />

### With Tip

<CodeBlock code={`import { Spin } from "@chaos_team/chaos-ui/ui"

export function WithTip() {
  return <Spin spinning tip="Loading...">Content</Spin>
}`} lang="tsx" />

### Sizes

<CodeBlock code={`import { Spin } from "@chaos_team/chaos-ui/ui"

export function Sizes() {
  return (
    <div className="flex flex-wrap items-center gap-3">
      <Spin size="sm" spinning />
      <Spin size="md" spinning />
      <Spin size="lg" spinning />
    </div>
  )
}`} lang="tsx" />

## Size Reference

| Size | Description | 默认 |
| ---- | ----------- | ---- |
| `xs` | Extra small |      |
| `sm` | Small       |      |
| `md` | Medium      | ✓    |
| `lg` | Large       |      |
| `xl` | Extra large |      |

## API / Props

### Spin

| Prop        | Type                                   | Default | Description                                  |
| ----------- | -------------------------------------- | ------- | -------------------------------------------- |
| `spinning`  | `boolean`                              | `true`  | Whether to show spinner / 是否加载中         |
| `tip`       | `React.ReactNode`                      | —       | Loading tip text / 加载提示文本              |
| `size`      | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `md`    | Spinner size / 大小                          |
| `delay`     | `number`                               | `0`     | Delay before showing spinner (ms) / 延迟显示 |
| `indicator` | `React.ReactNode`                      | —       | Custom spinner indicator / 自定义指示器      |
| `children`  | `React.ReactNode`                      | —       | Wrapped content / 包裹的内容                 |
| `className` | `string`                               | —       | Custom CSS class                             |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **无 children 时**: 当不传 children 时，仅显示加载指示器。

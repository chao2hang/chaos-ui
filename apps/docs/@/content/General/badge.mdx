# Badge 徽标

Chaos UI 徽标组件，用于标记状态、分类或数量的小型标签。
A small label for marking status, category, or count.

## 何时使用 / When to use

适合标记状态 (成功/失败/警告)、显示计数、标注分类标签。通常放在其他元素的右上角或内联使用。

Use for status tagging (success/error/warning), counts, or category labels. Typically placed at the top-right of another element or inline.

## API

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| variant | `'default' \| 'secondary' \| 'destructive' \| 'outline'` | `'default'` | 样式变体 | Style variant |
| color | `'success' \| 'warning' \| 'destructive' \| 'info' \| 'neutral'` | — | 语义颜色 | Semantic color |
| size | `'default' \| 'sm' \| 'lg'` | `'default'` | 尺寸 | Size |
| dot | `boolean` | `false` | 仅显示圆点（不显示文字） | Show dot only |
| closable | `boolean` | `false` | 显示关闭按钮 | Show close button |
| onClose | `() => void` | — | 关闭回调 | Close callback |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

## 注意事项

- **dot 模式**：`dot` 为 true 时忽略 children，仅渲染彩色圆点。适合状态指示场景。
- **closable 搭配 onClose**：提供 `closable` 时建议同时提供 `onClose` 处理关闭逻辑，否则点击关闭按钮无效果。
- **无障碍**：Badge 默认使用 `<span>` 渲染，纯展示性内容。可关闭模式下关闭按钮通过 `aria-label="Close"` 标识。
- **颜色覆盖**：`color` 和 `variant` 可同时使用，`color` 优先控制色相，`variant` 控制填充/描边风格。

## API / Props

### VideoPlayer

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `src` | `string` | — | Video source URL. |
| `poster` | `string` | — | Poster image URL. |
| `autoPlay` | `boolean` | — | Whether to auto-play on mount. Default false. |
| `showControls` | `boolean` | `true` | Whether to show controls overlay. Default true. |
| `onEnded` | `() => void` | — | Called when playback ends. |
| `onPlay` | `() => void` | — |  |
| `onPause` | `() => void` | — |  |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

# Masonry 瀑布流

> 瀑布流网格布局

基于 CSS columns 的瀑布流/砖石网格布局，支持响应式列数配置。

## 何时使用

展示不等高的卡片/图片时使用，如图片画廊、Pinterest 风格布局。

## 代码示例

### 基础用法

<CodeBlock code={`import { Masonry } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Masonry columns={3} gap={16}>
      <div className="rounded bg-primary/10 p-4" style={{ height: 120 }}>Item 1</div>
      <div className="rounded bg-primary/10 p-4" style={{ height: 80 }}>Item 2</div>
      <div className="rounded bg-primary/10 p-4" style={{ height: 160 }}>Item 3</div>
      <div className="rounded bg-primary/10 p-4" style={{ height: 100 }}>Item 4</div>
    </Masonry>
  )
}`} lang="tsx" />

### 响应式列数

<CodeBlock code={`import { Masonry } from "@chaos_team/chaos-ui/ui"

export function Responsive() {
  return (
    <Masonry columns={{ default: 1, sm: 2, md: 3, lg: 4 }} gap={12}>
      <div className="rounded border p-4">A</div>
      <div className="rounded border p-4">B</div>
      <div className="rounded border p-4">C</div>
      <div className="rounded border p-4">D</div>
    </Masonry>
  )
}`} lang="tsx" />

## API / Props

### Masonry

| Prop        | Type                                         | Default | 说明                                                 |
| ----------- | -------------------------------------------- | ------- | ---------------------------------------------------- |
| `columns`   | `number \| { default?, sm?, md?, lg?, xl? }` | `3`     | Column count or responsive config / 列数或响应式配置 |
| `gap`       | `number`                                     | `16`    | Gap between items in pixels / 项目间距（像素）       |
| `className` | `string`                                     | —       | 自定义样式类名                                       |
| ...rest     | `React.ComponentProps<"div">`                | —       | 透传 `<div>` 原生属性                                |

## 注意事项

- **CSS columns**: 基于 CSS `column-count` 实现，子元素自动 `break-inside-avoid`。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# UserBrowse 用户选择

> 用户选择对话框

用户选择对话框，支持搜索、头像展示和多选。

## 何时使用

需要选择一个或多个用户时使用，如任务分配、权限授予、审批人选择。

## 代码示例

### 基础用法

<CodeBlock code={`import { UserBrowse } from "@chaos_team/chaos-ui/ui"
import { useState } from "react"

export function Basic() {
  const [value, setValue] = useState(null)
  return <UserBrowse value={value} onChange={setValue} />
}`} lang="tsx" />

## API / Props

### UserBrowse

| Prop           | Type                              | Default | 说明                         |
| -------------- | --------------------------------- | ------- | ---------------------------- |
| `value`        | `User \| User[]`                  | —       | Selected user(s) / 已选用户  |
| `defaultValue` | `User \| User[]`                  | —       | Default selection / 默认选择 |
| `onChange`     | `(value: User \| User[]) => void` | —       | Change callback / 变化回调   |
| `multiple`     | `boolean`                         | `false` | Multi-select / 多选          |
| `placeholder`  | `string`                          | —       | Placeholder / 占位文本       |

### User

| Prop         | Type     | 说明                           |
| ------------ | -------- | ------------------------------ |
| `id`         | `string` | Unique ID / 唯一 ID (required) |
| `name`       | `string` | User name / 用户名 (required)  |
| `email`      | `string` | Email / 邮箱                   |
| `avatar`     | `string` | Avatar URL / 头像地址          |
| `department` | `string` | Department / 部门              |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **基于 Dialog**: 内部使用 Dialog 组件作为选择面板容器。

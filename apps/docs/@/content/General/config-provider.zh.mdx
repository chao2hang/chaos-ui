# ConfigProvider 全局配置

> 通用基础组件

为组件树提供全局配置(主题、尺寸、国际化、前缀)。

## 何时使用

需要全局配置功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { ConfigProvider, ConfigContext, Config } from "@/components/ui/config-provider"

export function Basic() {
  return (
    <div className="space-y-4">
  <ConfigProvider />
  <ConfigContext />
  <Config />
    </div>
  )
}`} lang="tsx" />

## API / Props

### ConfigProvider

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

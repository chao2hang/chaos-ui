## API / Props

### SequencePreview

| Prop           | Type                          | Default | Description                           |
| -------------- | ----------------------------- | ------- | ------------------------------------- |
| `className`    | `string`                      | —       | Custom CSS class                      |
| `value`        | `string`                      | —       | The sequence value to preview         |
| `mask`         | `string`                      | `•`     | Character to use for hidden chars     |
| `visibleChars` | `number`                      | —       | Show only last N chars, mask the rest |
| ...rest        | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes       |

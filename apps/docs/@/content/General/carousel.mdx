# Carousel 轮播

Chaos UI 轮播组件，支持自动播放、循环、指示点等特性。
A carousel component with autoplay, loop, and dot indicators.

## 何时使用 / When to use

用于在有限空间内循环展示一组内容：图片画廊、产品展示、Banner 轮播、用户引导等。

Use for cycling through content in limited space: image galleries, product showcases, banners, onboarding.

## API

### Carousel

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| defaultIndex | `number` | `0` | 默认展示的 slide 索引 | Default active slide index |
| loop | `boolean` | `false` | 是否循环播放 | Enable infinite loop |
| autoplay | `boolean` | `false` | 是否自动播放 | Enable autoplay |
| interval | `number` | `3000` | 自动播放间隔 (ms) | Autoplay interval in ms |
| onIndexChange | `(index: number) => void` | — | 索引变化回调 | Callback on index change |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### CarouselItem

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### CarouselPrevious / CarouselNext

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| ...rest | `ButtonProps` | — | 透传 Button 属性 | Passthrough Button props |

### CarouselDots

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| className | `string` | — | 自定义样式类名 | Custom CSS class |

## 注意事项

- **autoplay 与 loop**：自动播放建议同时开启 `loop`，否则播放到末位会停止。
- **间隔控制**：`interval` 仅在 `autoplay=true` 时生效，单位毫秒，默认 3000ms。
- **无障碍**：左右箭头使用 Button 组件，内置 `aria-label`。CarouselDots 中的每个点均可键盘聚焦。建议为 Carousel 区域添加 `aria-roledescription="carousel"` 提升屏幕阅读器体验。
- **嵌套内容**：`CarouselContent` 和 `CarouselItem` 均渲染为 `<div>`，可放置任意 React 内容。

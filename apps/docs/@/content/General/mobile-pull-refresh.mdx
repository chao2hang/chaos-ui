## API / Props

### MobilePullRefresh

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `onRefresh` | `() => void` | — | 刷新回调 / refresh handler |
| `refreshing` | `boolean` | `false` | 是否正在刷新 / refreshing state |
| `threshold` | `number` | `60` | 触发阈值(px) / pull threshold in px |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

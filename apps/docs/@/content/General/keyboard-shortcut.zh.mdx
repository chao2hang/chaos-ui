# KeyboardShortcut 键盘快捷键

> 通用基础组件

键盘快捷键 — Chaos UI 的通用基础组件。

## 何时使用

需要键盘快捷键功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { KeyboardShortcut } from "@/components/ui/keyboard-shortcut"

export function Basic() {
  return <KeyboardShortcut>Default</KeyboardShortcut>
}`} lang="tsx" />

### 尺寸

<CodeBlock
  code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <KeyboardShortcut size="sm">Sm</KeyboardShortcut>
  <KeyboardShortcut size="default">Default</KeyboardShortcut>
  <KeyboardShortcut size="lg">Lg</KeyboardShortcut>
</div>`}
  lang="tsx"
/>

## 样式变体

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |
| `sm`      | 小号     |      |
| `lg`      | 大号     |      |

## API / Props

### KeyboardShortcut

| Prop        | Type                          | Default | 说明                   |
| ----------- | ----------------------------- | ------- | ---------------------- |
| `className` | `string`                      | —       | 自定义样式类名         |
| `keys`      | `string \| string[]`          | —       | The keys to display    |
| `separator` | `string`                      | `+`     | Separator between keys |
| ...rest     | `React.ComponentProps<"kbd">` | —       | 透传 `<kbd>` 原生属性  |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<kbd>` 原生 HTML 属性。

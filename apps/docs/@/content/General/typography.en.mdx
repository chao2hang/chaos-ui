# Typography

> General-purpose primitives used across the app.

Typography — a general-purpose primitives used across the app. from Chaos UI.

## When to Use

Use when you need a typography.

## Examples

### Basic Usage

<CodeBlock code={`import { Typography, Title, Paragraph, Text, Link } from "@/components/ui/typography"

export function Basic() {
  return (
    <Typography>
      <Title>Content</Title>
    </Typography>
  )
}`} lang="tsx" />

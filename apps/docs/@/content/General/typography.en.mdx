# Typography

> Typography component suite.

Typography component suite: H1–H6, Text, Paragraph, Blockquote, InlineCode, and more for consistent text hierarchy.

## When to Use

Use when you need consistent text hierarchy, color, size, and styling — replacing ad-hoc Tailwind classes.

## Examples

### Headings

<CodeBlock code={`import { H1, H2, H3 } from "@chaos_team/chaos-ui/ui"

export function Headings() {
  return (
    <div className="space-y-2">
      <H1>Heading 1</H1>
      <H2>Heading 2</H2>
      <H3>Heading 3</H3>
    </div>
  )
}`} lang="tsx" />

### Text Variants

<CodeBlock code={`import { Text } from "@chaos_team/chaos-ui/ui"

## API / Props

### Text

| Prop | Type | Default | Description |
| ----------- | --------------------------------------------------------------------------------------------- | --------- | ----------------------------- |
| `size`      | `"xs" \| "sm" \| "base" \| "lg" \| "xl" \| "2xl"`                                             | `base`    | Text size / 文本大小          |
| `weight`    | `"normal" \| "medium" \| "semibold" \| "bold"`                                                | `normal`  | Text weight / 文本粗细        |
| `variant`   | `"default" \| "muted" \| "primary" \| "secondary" \| "destructive" \| "success" \| "warning"` | `default` | Text color variant / 文本颜色 |
| `italic`    | `boolean`                                                                                     | `false`   | Italic / 斜体                 |
| `underline` | `boolean`                                                                                     | `false`   | Underline / 下划线            |
| `strike`    | `boolean`                                                                                     | `false`   | Strikethrough / 删除线        |
| `truncate`  | `boolean`                                                                                     | `false`   | Truncate / 截断               |

### Typography Namespace

| Member | Description |
| --------------------------------- | ------------------------------ |
| `Typography.H1` – `Typography.H6` | Heading components |
| `Typography.Text` | Text component |
| `Typography.Paragraph` | Paragraph component |
| `Typography.Blockquote` | Blockquote |
| `Typography.InlineCode` | Inline code |
| `Typography.Lead` | Lead text |
| `Typography.Large` | Large text |
| `Typography.Small` | Small text |
| `Typography.Muted` | Muted text |
| `Typography.List` | List |

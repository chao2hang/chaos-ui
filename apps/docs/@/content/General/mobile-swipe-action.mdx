## API / Props

### MobileSwipeAction

| Prop           | Type                   | Default | 说明 / Description                                        |
| -------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`    | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `leftActions`  | `SwipeActionDef[]`     | —       | 左侧操作按钮 / left action buttons                        |
| `rightActions` | `SwipeActionDef[]`     | —       | 右侧操作按钮 / right action buttons                       |
| `actionWidth`  | `number`               | `80`    | 单个操作按钮宽度(px) / action button width                |
| ...rest        | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## API / Props

### Rating

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `value` | `number` | `0` | Current rating value |
| `onChange` | `() => void` | — | Called when rating changes |
| `count` | `number` | `5` | Max number of stars |
| `allowHalf` | `boolean` | `false` | Whether to allow half stars |
| `readOnly` | `boolean` | `false` | Whether the rating is read-only |
| `filledIcon` | `React.ReactNode` | — | Character/icon for filled star |
| `emptyIcon` | `React.ReactNode` | — | Character/icon for empty star |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

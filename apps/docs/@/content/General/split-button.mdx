## API / Props

### SplitButton

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `label` | `React.ReactNode` | — | Main button label |
| `onClick` | `() => void` | — | Main button click handler |
| `items` | `SplitButtonItem[]` | — | Dropdown menu items |
| `onSelect` | `() => void` | — | Called when a dropdown item is selected |
| `disabled` | `boolean` | — | 是否禁用 <br/> Whether disabled |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

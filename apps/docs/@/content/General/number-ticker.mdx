# NumberTicker 数字跳动

> 动画数字计数器 / Animated number counter for KPI/statistic displays.

用于 KPI/统计展示的动画数字计数器，支持缓动函数和前缀/后缀。
Animated number counter for KPI/statistic displays with easing and prefix/suffix support.

## 何时使用 / When to Use

展示统计数据、KPI 面板中的数字时使用，数字从 0 平滑动画到目标值。
Use for displaying statistics or KPI numbers, with smooth animation from 0 to the target value.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { NumberTicker } from "@/components/ui/number-ticker"

export function Basic() {
  return <NumberTicker value={12345} />
}`} lang="tsx" />

## API / Props

### NumberTicker

| Prop        | Type                                   | Default   | 说明 / Description                            |
| ----------- | -------------------------------------- | --------- | --------------------------------------------- |
| `value`     | `number`                               | —         | Target number / 目标数字 (required)           |
| `from`      | `number`                               | `0`       | Starting number / 起始数字                    |
| `duration`  | `number`                               | `1000`    | Animation duration in ms / 动画时长           |
| `easing`    | `"linear" \| "easeOut" \| "easeInOut"` | `easeOut` | Easing function / 缓动函数                    |
| `format`    | `(n: number) => string`                | —         | Custom formatter / 自定义格式化               |
| `decimals`  | `number`                               | `0`       | Decimal places / 小数位数                     |
| `prefix`    | `React.ReactNode`                      | —         | Prefix / 前缀                                 |
| `suffix`    | `React.ReactNode`                      | —         | Suffix / 后缀                                 |
| `reAnimate` | `boolean`                              | `true`    | Re-animate on value change / 值变化时重新动画 |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **无障碍**: 检测 `prefers-reduced-motion`，如用户偏好减少动画则直接显示最终值。

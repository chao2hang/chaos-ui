## API / Props

### Icon

| Prop        | Type                   | Default   | 说明 / Description                                         |
| ----------- | ---------------------- | --------- | ---------------------------------------------------------- |
| `className` | `string`               | —         | 自定义样式类名 <br/> Custom CSS class                      |
| `icon`      | `LucideIcon`           | —         | Lucide icon component to render / 要渲染的 Lucide 图标组件 |
| `size`      | `IconSize`             | `inherit` | Preset size / 预设尺寸 (default: "inherit")                |
| `spin`      | `boolean`              | —         | Apply spinning animation / 应用旋转动画                    |
| `pulse`     | `boolean`              | —         | Apply pulse animation / 应用脉冲动画                       |
| ...rest     | `React.HTMLAttributes` | —         | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes  |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

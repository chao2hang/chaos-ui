# KeyboardShortcut

> General-purpose primitives used across the app.

KeyboardShortcut — a general-purpose primitives used across the app. from Chaos UI.

## When to Use

Use when you need a keyboard shortcut.

<CodeBlock code={`import { KeyboardShortcut } from "@/components/ui/keyboard-shortcut"

## Style Variants

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

## API / Props

### KeyboardShortcut

| Prop        | Type                          | Default | Description                     |
| ----------- | ----------------------------- | ------- | ------------------------------- |
| `className` | `string`                      | —       | Custom CSS class                |
| `keys`      | `string \| string[]`          | —       | The keys to display             |
| `separator` | `string`                      | `+`     | Separator between keys          |
| ...rest     | `React.ComponentProps<"kbd">` | —       | Inherits all `<kbd>` attributes |

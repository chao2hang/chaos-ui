# Button 按钮

> 通用基础组件 / General-purpose primitives used across the app.

触发操作或事件,如提交表单、打开对话框、取消操作等。
Buttons trigger actions or events such as submitting forms, opening dialogs, or cancelling operations.

## 何时使用 / When to Use

点击触发操作,如提交表单、打开弹窗、确认/取消操作。

Clicking triggers an action such as form submission, opening a dialog, or confirming/cancelling.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant | 说明 / Description | 默认 |
|---|---|---|
| `default` | 默认样式 / Default style | ✓ |
| `outline` | 描边样式 / Outlined style |  |
| `secondary` | 次要样式 / Secondary action style |  |
| `ghost` | 幽灵(无背景)样式 / Ghost (no background) style |  |
| `destructive` | 危险/删除样式 / Destructive/danger style |  |
| `link` | 链接样式 / Link style |  |

**尺寸 / Size**

| Size | 说明 / Description | 默认 |
|---|---|---|
| `default` | 默认尺寸 / Default size | ✓ |
| `xs` | 超小 / Extra small |  |
| `sm` | 小号 / Small |  |
| `lg` | 大号 / Large |  |
| `icon` | 图标正方形 / Icon-only square |  |
| `icon-xs` | 超小图标 / Extra small icon |  |
| `icon-sm` | 小图标 / Small icon |  |
| `icon-lg` | 大图标 / Large icon |  |

## API / Props

### Button

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

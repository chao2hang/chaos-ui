# Icon

> Lucide icon wrapper with consistent sizing and animation.

A thin wrapper over Lucide icons providing consistent sizing, color tokens, and animation props.

## When to Use

Use when you need to display vector icons in the UI. Supports spin and pulse animations for loading states or interactive hints.

## Examples

### Basic Usage

<CodeBlock code={`import { Icon } from "@/components/ui/icon"
import { Search } from "lucide-react"

export function Basic() {
  return <Icon icon={Search} size="md" />
}`} lang="tsx" />

## Size Reference

| Size      | Description                   | 默认 |
| --------- | ----------------------------- | ---- |
| `inherit` | Inherit from parent font size | ✓    |
| `sm`      | Small                         |      |
| `md`      | Medium                        |      |
| `lg`      | Large                         |      |
| `xl`      | Extra large                   |      |

## API / Props

### Icon

| Prop        | Type                           | Default   | Description                                                           |
| ----------- | ------------------------------ | --------- | --------------------------------------------------------------------- |
| `icon`      | `LucideIcon`                   | —         | Lucide icon component to render / 要渲染的 Lucide 图标组件 (required) |
| `size`      | `IconSize`                     | `inherit` | Preset size / 预设尺寸                                                |
| `spin`      | `boolean`                      | —         | Apply spinning animation / 应用旋转动画                               |
| `pulse`     | `boolean`                      | —         | Apply pulse animation / 应用脉冲动画                                  |
| `className` | `string`                       | —         | Custom CSS class                                                      |
| ...rest     | `React.ComponentProps<"span">` | —         | Inherits standard `<span>` attributes                                 |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **必填 `icon` prop**: 必须传入一个 Lucide 图标组件,否则会报错。

# Collapsible

> Root container for a collapsible/expandable section.

Root container for a collapsible/expandable section, built on Base UI. Use with Trigger and Content sub-components.

## When to Use

Use when you need to collapse/expand content sections, such as FAQs, settings panels, or detail areas.

## Examples

### Basic Usage

<CodeBlock code={`import { Collapsible, CollapsibleTrigger, CollapsibleContent } from "@chaos_team/chaos-ui/ui"
import { Button } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Collapsible>
      <CollapsibleTrigger render={<Button variant="ghost" size="sm">Toggle</Button>} />
      <CollapsibleContent className="mt-2 text-sm">
        This content is revealed when expanded.
      </CollapsibleContent>
    </Collapsible>
  )
}`} lang="tsx" />

### Controlled

<CodeBlock code={`import { Collapsible, CollapsibleTrigger, CollapsibleContent } from "@chaos_team/chaos-ui/ui"
import { Button } from "@chaos_team/chaos-ui/ui"
import { useState } from "react"

export function Controlled() {
  const [open, setOpen] = useState(true)
  return (
    <Collapsible open={open} onOpenChange={setOpen}>
      <CollapsibleTrigger render={<Button variant="ghost" size="sm">{open ? "收起" : "展开"}</Button>} />
      <CollapsibleContent className="mt-2 text-sm">
        Controlled collapsible content.
      </CollapsibleContent>
    </Collapsible>
  )
}`} lang="tsx" />

## API / Props

### Collapsible

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ------- | ------------------------------------ |
| `open`         | `boolean`                 | —       | Controlled open state / 受控展开状态 |
| `defaultOpen`  | `boolean`                 | `false` | Default open state / 默认展开状态    |
| `onOpenChange` | `(open: boolean) => void` | —       | Open change callback / 展开变化回调  |
| `disabled`     | `boolean`                 | `false` | Disable toggle / 禁用切换            |

### CollapsibleTrigger

| Prop | Type | Default | Description |
| ---------- | -------------------- | ------- | ------------------------------------------ |
| `render`   | `React.ReactElement` | —       | Render prop for trigger element / 触发元素 |
| `disabled` | `boolean`            | `false` | Disable trigger / 禁用触发器               |

### CollapsibleContent

| Prop | Type | Default | Description |
| ---------- | ----------------- | ------- | ------------------------------ |
| `children` | `React.ReactNode` | —       | Collapsible content / 折叠内容 |

## Notes

- **Base UI**: 基于 `@base-ui/react/collapsible` 构建，无障碍(a11y)开箱即用。

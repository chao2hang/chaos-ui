# KeyboardShortcut 键盘快捷键

> 键盘快捷键展示 / Keyboard shortcut display component.

键盘快捷键展示组件，渲染格式化的快捷键提示，可附带描述文本。
Keyboard shortcut display — renders formatted key combination hints with optional description.

## 何时使用 / When to Use

在界面中展示快捷键提示时使用，如设置页面、帮助文档、命令面板。
Use when displaying keyboard shortcut hints in the UI, such as settings pages, help docs, or command palettes.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { KeyboardShortcut } from "@/components/ui/keyboard-shortcut"

export function Basic() {
  return <KeyboardShortcut keys={["Ctrl", "K"]} description="Open command palette" />
}`} lang="tsx" />

### 多个快捷键 / Multiple Shortcuts

<CodeBlock code={`import { KeyboardShortcut } from "@/components/ui/keyboard-shortcut"

export function Multiple() {
  return (
    <div className="flex flex-col gap-2">
      <KeyboardShortcut keys={["Ctrl", "N"]} description="New file" />
      <KeyboardShortcut keys={["Ctrl", "S"]} description="Save" />
      <KeyboardShortcut keys={["Ctrl", "Shift", "Z"]} description="Redo" />
    </div>
  )
}`} lang="tsx" />

## API / Props

### KeyboardShortcut

| Prop          | Type                          | Default      | 说明 / Description                                          |
| ------------- | ----------------------------- | ------------ | ----------------------------------------------------------- |
| `keys`        | `string[]`                    | —            | Key combination / 按键组合 (required)                       |
| `description` | `string`                      | —            | Description text / 描述文本                                 |
| `layout`      | `"horizontal" \| "vertical"`  | `horizontal` | Layout / 布局                                               |
| `className`   | `string`                      | —            | 自定义样式类名 <br/> Custom CSS class                       |
| ...rest       | `React.ComponentProps<"div">` | —            | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **基于 Kbd 组件**: 内部使用 `Kbd` 和 `KbdGroup` 组件渲染按键。

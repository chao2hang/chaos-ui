## API / Props

### ImportErrorTable

| Prop        | Type                   | Default | 说明                                                  |
| ----------- | ---------------------- | ------- | ----------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                        |
| `errors`    | `Array<{`              | —       | Validation errors, one per offending import row.      |
| `row`       | `number`               | —       | 1-based source spreadsheet row number.                |
| `field`     | `string`               | —       | Field/column that failed validation, if known.        |
| `value`     | `string`               | —       | The raw value that failed, if known.                  |
| `message`   | `string`               | —       | Human-readable reason the row was rejected.           |
| `onExport`  | `() => void`           | —       | Called when the user clicks the export-errors button. |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                     |

## 注意事项

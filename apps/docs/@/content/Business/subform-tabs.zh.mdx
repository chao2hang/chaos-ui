# SubformTabs Subform Tabs

> 业务组件

子表单 tab

## 何时使用

需要采集或编辑业务字段，并参与校验/提交时使用。典型用于「Subform Tabs」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

### 基础用法

## API / Props

### SubformTabs

| Prop        | Type                      | Default | 说明                                                               |
| ----------- | ------------------------- | ------- | ------------------------------------------------------------------ |
| `className` | `string`                  | —       | 自定义样式类名                                                     |
| `tabs`      | `SubformTab[]`            | —       | Tab sections to render. Defaults to a basic info / contact layout. |
| `value`     | `Record<string, unknown>` | —       | Field values keyed by name.                                        |
| `onChange`  | `() => void`              | —       | Called with the merged values whenever a field changes.            |
| ...rest     | `React.HTMLAttributes`    | —       | 透传原生 DOM 属性                                                  |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

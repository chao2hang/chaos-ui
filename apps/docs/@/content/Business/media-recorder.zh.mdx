## API / Props

### MediaRecorder

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `onStop` | `() => void` | — | Called with the recorded Blob when recording stops. |
| `onStart` | `() => void` | — | Called when recording starts. |
| `mimeType` | `string` | — | MIME type for the recording. Default audio/webm. |
| `screen` | `boolean` | — | Whether to record video (screen) instead of audio. Default false. |
| `maxDuration` | `number` | — | Maximum recording duration in seconds. 0 = unlimited. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(MediaRecorderComponent),使用组合模式按需导入。

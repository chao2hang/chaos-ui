## API / Props

### BarChart

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `data` | `Array<T>` | — | Bars to render. |
| `orientation` | `"vertical" \| "horizontal"` | `vertical` | Bar orientation. |
| `height` | `number` | `200` | Chart height in px. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

# UtmBuilder UTM 构建器

> 业务组件

UTM 构建器 — 面向 ERP/企业场景的业务组件。

## 何时使用

适用于UTM 构建器相关场景。

## 代码示例

### 基础用法

<CodeBlock code={`import { UtmBuilder } from "@chaos_team/chaos-ui/business"

## API / Props

### UtmBuilder

| Prop        | Type                          | Default | 说明                  |
| ----------- | ----------------------------- | ------- | --------------------- |
| `className` | `string`                      | —       | 自定义样式类名        |
| `baseUrl`   | `string`                      | ``      |                       |
| `onChange`  | `() => void`                  | —       | 值变化回调            |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

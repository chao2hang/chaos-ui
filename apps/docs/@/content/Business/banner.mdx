# Banner 横幅

> 业务组件 / Domain-specific business components for ERP scenarios.

横幅 — 面向 ERP/企业场景的业务组件。
Banner — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要banner时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use Banner when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant   | 说明 / Description             | 默认 |
| --------- | ------------------------------ | ---- |
| `info`    | 信息提示 / Informational style | ✓    |
| `success` | 成功状态 / Success style       |      |
| `warning` | 警告状态 / Warning style       |      |
| `error`   | 错误状态 / Error style         |      |

## API / Props

### Banner

| Prop          | Type                   | Default | 说明 / Description                                        |
| ------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`   | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `variant`     | `BannerVariant`        | `info`  | 样式变体 <br/> Style variant                              |
| `title`       | `string`               | —       | 标题 <br/> Title                                          |
| `description` | `string`               | —       | 描述文本 <br/> Description text                           |
| `action`      | `React.ReactNode`      | —       |                                                           |
| `closable`    | `boolean`              | `true`  |                                                           |
| `onClose`     | `() => void`           | —       | 关闭回调 <br/> Close handler                              |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

# Combobox 组合框

> 业务组件 / Domain-specific business components for ERP scenarios.

结合输入框和下拉列表的组合选择控件,支持自定义输入和选项匹配。
Combines input with a dropdown list; supports custom input and option matching.

## 何时使用 / When to Use

当你需要「组合框」相关能力时使用：结合输入框和下拉列表的组合选择控件,支持自定义输入和选项匹配。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use Combobox when you need: Combines input with a dropdown list; supports custom input and option matching. Use to collect or edit data with validation and submit flows.

## API / Props

### Combobox

| Prop                | Type                           | Default      | 说明 / Description                                        |
| ------------------- | ------------------------------ | ------------ | --------------------------------------------------------- |
| `className`         | `string`                       | —            | 自定义样式类名 <br/> Custom CSS class                     |
| `options`           | `ComboboxOption[]`             | —            | 可选项列表 <br/> Available options                        |
| `value`             | `string`                       | —            | 当前值(受控) <br/> Current value (controlled)             |
| `onChange`          | `() => void`                   | —            | 值变化回调 <br/> Callback when value changes              |
| `placeholder`       | `string`                       | `选择...`    | 占位文本 <br/> Placeholder text                           |
| `searchPlaceholder` | `string`                       | `搜索...`    |                                                           |
| `emptyText`         | `string`                       | `未找到结果` |                                                           |
| `disabled`          | `boolean`                      | —            | 是否禁用 <br/> Whether disabled                           |
| `align`             | `"start" \| "center" \| "end"` | `start`      | 对齐方式 <br/> Alignment                                  |
| `clearable`         | `boolean`                      | `true`       |                                                           |
| `searchable`        | `boolean`                      | `true`       |                                                           |
| `renderOption`      | `(...args) => unknown`         | —            |                                                           |
| ...rest             | `React.HTMLAttributes`         | —            | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

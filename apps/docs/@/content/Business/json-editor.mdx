# JsonEditor Json Editor

> 业务组件 / Domain-specific business components for ERP scenarios.

Json Editor — 面向 ERP/企业场景的业务组件。
JsonEditor — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

当你需要「Json Editor」相关能力时使用：Try to pretty-print a JSON string. Returns the original string on failure。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use JsonEditor when you need: JsonEditor: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### JsonEditor

| Prop          | Type                   | Default | 说明 / Description                                                           |
| ------------- | ---------------------- | ------- | ---------------------------------------------------------------------------- |
| `className`   | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                                        |
| `value`       | `string`               | —       | JSON string (controlled).                                                    |
| `onChange`    | `() => void`           | —       | Called on every change.                                                      |
| `placeholder` | `string`               | —       | Placeholder text.                                                            |
| `readOnly`    | `boolean \| undefined` | —       | Whether the editor is read-only.                                             |
| `minHeight`   | `number`               | `200`   | Minimum height.                                                              |
| `maxHeight`   | `number`               | —       | Maximum height.                                                              |
| `autoFormat`  | `boolean \| undefined` | —       | If true, auto-formats the JSON value on mount and on external value changes. |
| `onFocus`     | `() => void`           | —       | Called when the editor receives focus.                                       |
| `onBlur`      | `() => void`           | —       | Called when the editor loses focus.                                          |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes                    |

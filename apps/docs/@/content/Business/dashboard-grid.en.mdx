## API / Props

### DashboardGrid

| Prop        | Type                    | Default  | Description                       |
| ----------- | ----------------------- | -------- | --------------------------------- |
| `className` | `string`                | —        | Custom CSS class                  |
| `title`     | `string`                | `仪表盘` | 仪表盘标题                        |
| `stats`     | `Array<{`               | —        | KPI 指标数组                      |
| `label`     | `string`                | —        | Label text                        |
| `value`     | `number`                | —        | Current value (controlled)        |
| `delta`     | `number`                | —        | 环比变化，正数增长负数下降        |

| `columns`   | `1 \| 2 \| 3 \| 4 \| 6` | `4`      | 网格列数                          |
| ...rest     | `React.HTMLAttributes`  | —        | Inherits standard HTML attributes |

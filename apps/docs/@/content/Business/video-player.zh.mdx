# VideoPlayer 视频播放器

> 业务组件

视频播放器 — 面向 ERP/企业场景的业务组件。

## 何时使用

播放视频内容,支持播放/暂停、进度条、音量控制。

## 代码示例

### 基础用法

<CodeBlock code={`import { VideoPlayer } from "@chaos_team/chaos-ui/ui"

  return <VideoPlayer />
}`} lang="tsx" />

## API / Props

### VideoPlayer

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `src`       | `string`               | —       | 资源地址          |
| `poster`    | `string`               | —       |                   |
| `title`     | `string`               | —       | 标题              |
| `autoplay`  | `boolean`              | `false` |                   |
| `loop`      | `boolean`              | `false` |                   |
| `muted`     | `boolean`              | `false` |                   |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

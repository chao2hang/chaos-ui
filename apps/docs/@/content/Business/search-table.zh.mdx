# SearchTable 搜索表格

> 业务组件

搜索表格 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要搜索表格时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

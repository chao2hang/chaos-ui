# PermissionMatrix 权限矩阵

> 业务组件

权限矩阵 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要权限矩阵功能时使用本组件。

## API / Props

### PermissionMatrix

| Prop          | Type                          | Default | 说明                  |
| ------------- | ----------------------------- | ------- | --------------------- |
| `className`   | `string`                      | —       | 自定义样式类名        |
| `permissions` | `PermissionItem[]`            | —       |                       |
| `roles`       | `string[]`                    | —       |                       |
| `value`       | `Record<string, string[]>`    | —       | 当前值(受控)          |
| `onChange`    | `() => void`                  | —       | 值变化回调            |
| ...rest       | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

## API / Props

### ImMessage

| Prop          | Type                       | Default    | Description                       |
| ------------- | -------------------------- | ---------- | --------------------------------- |
| `className`   | `string`                   | —          | Custom CSS class                  |
| `direction`   | `"sent" \| "received"`     | `received` | 消息方向                          |
| `author`      | `string`                   | ``         | 发送者名称                        |
| `content`     | `string`                   | ``         | 消息正文                          |
| `timestamp`   | `string \| number \| Date` | —          | 消息时间                          |
| `attachments` | `Array<T>`                 | —          | 附件文件名列表                    |
| `avatarUrl`   | `string`                   | —          | 头像地址（可选）                  |
| ...rest       | `React.HTMLAttributes`     | —          | Inherits standard HTML attributes |

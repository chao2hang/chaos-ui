# DataTable 数据表

Chaos UI 数据表组件，提供类型安全的列定义和行数据渲染。
A type-safe data table component with column definitions and row rendering.

## 何时使用 / When to use

用于展示结构化的列表数据，提供列定义和行点击交互。适合需要类型安全和可定制列渲染的数据列表场景。

Use for structured list data with column definitions and row click interaction. Best when you need type-safe columns and customizable cell rendering.

## 代码示例 / Examples

### 基础用法 / Basic

<DataTable
  columns={[
    { key: "name", header: "姓名" },
    { key: "email", header: "邮箱" },
    { key: "role", header: "角色" },
  ]}
  data={[
    { name: "张三", email: "zhang@example.com", role: "Admin" },
    { name: "李四", email: "li@example.com", role: "Editor" },
    { name: "王五", email: "wang@example.com", role: "Viewer" },
  ]}
/>

## API

### DataTable

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| columns | `Column<T>[]` | — | **必填**，列定义数组 | **Required** column definitions |
| data | `T[]` | — | **必填**，行数据数组 | **Required** row data array |
| onRowClick | `(row: T) => void` | — | 行点击回调（客户端组件） | Row click callback (client component) |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### Column

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| key | `string` | — | **必填**，数据字段名 | **Required** data field key |
| header | `string` | — | **必填**，表头显示文字 | **Required** column header text |
| render | `(row: T) => React.ReactNode` | — | 自定义单元格渲染函数 | Custom cell render function |

## 注意事项

- **泛型支持**：`DataTable<T>` 是泛型组件，`columns` 和 `data` 共享类型参数 `T`，提供编译时类型安全。
- **render 函数**：不提供 `render` 时，默认渲染 `row[key]`。`render` 函数只能在客户端组件中使用。
- **vs Table**：`DataTable` 提供列定义和行交互 API，`Table` 是纯 HTML 表格组件。需要排序/筛选/分页时使用 `AdvancedDataTable`。
- **无障碍**：DataTable 内部使用 `<table>` 语义元素，表头使用 `<th>`。空状态时显示 `colSpan` 合并单元格提示。

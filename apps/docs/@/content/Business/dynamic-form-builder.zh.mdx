## API / Props

### DynamicFormBuilder

| Prop        | Type                      | Default | 说明              |
| ----------- | ------------------------- | ------- | ----------------- |
| `className` | `string`                  | —       | 自定义样式类名    |
| `schema`    | `Array<{`                 | —       |                   |
| `name`      | `string`                  | —       | 表单字段名        |
| `label`     | `string`                  | —       | 标签文本          |
| `type`      | `string`                  | —       | 类型              |
| `required`  | `boolean`                 | —       | 是否必填          |
| `options`   | `unknown[]`               | —       | 可选项列表        |
| `value`     | `Record<string, unknown>` | —       | 当前值(受控)      |
| `onChange`  | `() => void`              | —       | 值变化回调        |
| ...rest     | `React.HTMLAttributes`    | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

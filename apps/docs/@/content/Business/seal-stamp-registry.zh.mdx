## API / Props

### SealStampRegistry

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `seals` | `SealRecord[]` | — | Registered seals. |
| `usageLogs` | `SealUsageLog[]` | — | Usage log entries. |
| `onSealClick` | `() => void` | — | Seal click handler. |
| `onApplyUsage` | `() => void` | — | Apply for usage handler. |
| `readOnly` | `boolean` | `false` | Read-only mode. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

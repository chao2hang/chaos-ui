# AreaChart Area Chart

> 业务组件 / Domain-specific business components for ERP scenarios.

AreaChart — area chart component.

## 何时使用 / When to Use

当你需要「Area Chart」相关能力时使用：面积图(趋势+填充)。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use AreaChart when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### ProductBrowse

| Prop                | Type                   | Default | 说明                                                                           |
| ------------------- | ---------------------- | ------- | ------------------------------------------------------------------------------ |
| `className`         | `string`               | —       | 自定义样式类名                                                                 |
| `open`              | `boolean`              | —       | 是否打开(受控)                                                                 |
| `onOpenChange`      | `() => void`           | —       | 打开状态变化回调                                                               |
| `onSelect`          | `() => void`           | —       | Single-select emits the chosen item; multi-select emits the last toggled item. |
| `multiple`          | `boolean`              | `false` | Enable multi-select (default false). / 多选                                    |
| `items`             | `ProductBrowseItem[]`  | —       | 选项/数据项列表                                                                |
| `title`             | `string`               | —       | 标题                                                                           |
| `searchPlaceholder` | `string`               | —       |                                                                                |
| `emptyText`         | `string`               | —       |                                                                                |
| ...rest             | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                                              |

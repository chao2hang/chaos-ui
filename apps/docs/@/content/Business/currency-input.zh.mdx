# CurrencyInput 货币输入

> 业务组件

货币输入 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要currency-input时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## API / Props

### CurrencyInput

| Prop          | Type                   | Default | 说明              |
| ------------- | ---------------------- | ------- | ----------------- |
| `className`   | `string`               | —       | 自定义样式类名    |
| `value`       | `number \| undefined`  | —       | 当前值(受控)      |
| `onChange`    | `() => void`           | —       | 值变化回调        |
| `currency`    | `string`               | `USD`   |                   |
| `locale`      | `string`               | `en-US` |                   |
| `placeholder` | `string`               | —       | 占位文本          |
| `disabled`    | `boolean`              | —       | 是否禁用          |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

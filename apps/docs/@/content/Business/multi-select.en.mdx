# MultiSelect

> Domain-specific business components for ERP scenarios.

Select multiple options from a panel; selections shown as tags.

## When to Use

Select multiple values from a predefined list.


## Examples

### Basic Usage

<CodeBlock code={`import { MultiSelect } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <MultiSelect />
}`} lang="tsx" />

### States

<CodeBlock code={`<MultiSelect>Normal</MultiSelect>
<MultiSelect disabled>Disabled</MultiSelect>`} lang="tsx" />


## API / Props

### MultiSelect

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `options` | `MultiSelectOption[]` | — | Available options |
| `value` | `string[]` | — | Current value (controlled) |
| `onChange` | `() => void` | — | Callback when value changes |
| `placeholder` | `string` | `Select...` | Placeholder text |
| `searchPlaceholder` | `string` | `Search...` |  |
| `emptyText` | `string` | `No results found` |  |
| `disabled` | `boolean` | — | Whether disabled |
| `maxCount` | `number` | `3` |  |
| `maxSelected` | `number` | — |  |
| `clearable` | `boolean` | `true` |  |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

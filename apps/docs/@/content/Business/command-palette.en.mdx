## API / Props

### CommandPalette

| Prop           | Type                   | Default             | Description                       |
| -------------- | ---------------------- | ------------------- | --------------------------------- |
| `className`    | `string`               | —                   | Custom CSS class                  |
| `open`         | `boolean`              | —                   | Open state (controlled)           |
| `onOpenChange` | `() => void`           | —                   | Callback when open state changes  |
| `groups`       | `CommandGroup[]`       | —                   |                                   |
| `placeholder`  | `string`               | `输入命令或搜索...` | Placeholder text                  |
| `emptyText`    | `string`               | `未找到结果`        |                                   |
| `trigger`      | `React.ReactNode`      | —                   |                                   |
| `shortcut`     | `string`               | `mod+K`             |                                   |
| `showShortcut` | `boolean`              | `true`              |                                   |
| ...rest        | `React.HTMLAttributes` | —                   | Inherits standard HTML attributes |

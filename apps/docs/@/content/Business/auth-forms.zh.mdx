# AuthForms 认证表单

> 业务组件

认证表单 — 面向 ERP/企业场景的业务组件。

## 何时使用

登录、注册、找回密码等认证相关表单。

## API / Props

### SignInForm

| Prop           | Type                   | Default | 说明              |
| -------------- | ---------------------- | ------- | ----------------- |
| `className`    | `string`               | —       | 自定义样式类名    |
| `onSubmit`     | `() => void`           | —       | 提交回调          |
| `ssoProviders` | `Array<T>`             | —       |                   |
| `signupHref`   | `string`               | —       |                   |
| `forgotHref`   | `string`               | —       |                   |
| ...rest        | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

# MapView

> Domain-specific business components for ERP scenarios.

MapView — a business component for ERP / enterprise scenarios.

## When to Use

Embed a map showing markers, tracks, and location information.

## Examples

### Basic Usage

<CodeBlock code={`import { MapView } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <MapView />
}`} lang="tsx" />

## API / Props

### MapView

| Prop         | Type                          | Default | Description                                                   |
| ------------ | ----------------------------- | ------- | ------------------------------------------------------------- |
| `className`  | `string`                      | —       | Custom CSS class                                              |
| `center`     | `[number, number]`            | —       | Map center [lng, lat]                                         |
| `zoom`       | `number`                      | `11`    | Zoom level                                                    |
| `markers`    | `MapMarker[]`                 | —       | Array of map markers                                          |
| `height`     | `number \| string`            | `400`   | Component height                                              |
| `apiKey`     | `string`                      | —       | 高德地图 JS API key。生产环境必须通过 props 传入,不要硬编码。 |
| `apiVersion` | `string`                      | `2.0`   | 高德 JS API 版本,默认 "2.0"                                   |
| ...rest      | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes                               |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

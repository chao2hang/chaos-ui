# Chat 聊天

> 业务组件 / Domain-specific business components for ERP scenarios.

聊天 — 面向 ERP/企业场景的业务组件。
Chat — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要chat时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use Chat when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

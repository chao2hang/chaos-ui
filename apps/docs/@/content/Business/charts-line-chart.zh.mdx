# LineChart 折线图

> 业务组件

折线图 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要charts-line-chart时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## API / Props

### LineChart

| Prop          | Type                         | Default | 说明                            |
| ------------- | ---------------------------- | ------- | ------------------------------- |
| `series`      | `{ name; values; color? }[]` | 内置    | 多系列                          |
| `labels`      | `string[]`                   | 内置    | X 轴标签                        |
| `height`      | `number`                     | `180`   | SVG 高度 px                     |
| `showTooltip` | `boolean`                    | `true`  | 悬停 tooltip + crosshair（#22） |
| `className`   | `string`                     | —       | 根节点类名                      |

## 注意事项

- **Client Component**: 使用 `"use client"`，依赖 React hooks/状态，需在客户端渲染。
- **默认可读数**: 与 pure SVG `AreaChart` 同一交互契约（按 x 索引对齐多系列值）。
- 重交互/大数据请用 Recharts `chart.tsx` 系列。

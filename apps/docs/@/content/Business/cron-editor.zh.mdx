## API / Props

### CronEditor

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `value` | `string` | `* * * * *` | 5 段 Cron 表达式 |
| `onChange` | `() => void` | — | 表达式变化回调 |
| `showDescription` | `boolean` | `true` | 是否显示人类可读描述 |
| `readOnly` | `boolean` | `false` | 只读模式 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

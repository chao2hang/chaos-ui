## API / Props

### ArApAgingTable

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `entries` | `AgingEntry[]` | — | Entries to display. |
| `asOfDate` | `string` | — | Reference date for aging calculation. Defaults to today. |
| `type` | `"ar" \| "ap"` | `ar` | "ar" for receivable, "ap" for payable. |
| `buckets` | `AgingBucket[]` | — | Custom bucket definitions. |
| `currencySymbol` | `string` | `¥` | Currency symbol for display. |
| `onRowClick` | `() => void` | — | Row click handler. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

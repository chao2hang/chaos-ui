# IotSensorGrid Iot Sensor Grid

> 业务组件

Iot Sensor Grid — 面向 ERP/企业场景的业务组件。

## 何时使用

在需要「Iot Sensor Grid」时使用。用于 ERP/业务域流程（单据、选择、审批、结算等）。

## 代码示例

### 基础用法

<CodeBlock code={`import { IotSensorGrid } from "@chaos_team/chaos-ui/business"

  return <IotSensorGrid />
}`} lang="tsx" />

## API / Props

### IotSensorGrid

| Prop              | Type                   | Default           | 说明                                        |
| ----------------- | ---------------------- | ----------------- | ------------------------------------------- |
| `className`       | `string`               | —                 | 自定义样式类名                              |
| `sensors`         | `Sensor[]`             | —                 | Sensors to display.                         |
| `title`           | `string`               | `IoT Sensor Grid` | Dashboard title.                            |
| `refreshInterval` | `number`               | `0`               | Auto-refresh interval in ms (0 = disabled). |
| `onSensorClick`   | `() => void`           | —                 | Sensor click handler.                       |
| ...rest           | `React.HTMLAttributes` | —                 | 透传原生 DOM 属性                           |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

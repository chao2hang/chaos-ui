# EmptyState 空状态

> 业务组件 / Domain-specific business components for ERP scenarios.

当前没有数据时的占位展示,可搭配引导操作。
Placeholder for empty data states, optionally with a call-to-action.

## 何时使用 / When to Use

需要空状态功能时使用本组件。

Use when you need an empty state.

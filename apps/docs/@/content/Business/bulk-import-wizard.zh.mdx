# BulkImportWizard 批量导入向导

> 业务组件

批量导入向导 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要批量导入向导功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { BulkImportWizard } from "@/components/business/bulk-import-wizard"

export function Basic() {
  return <BulkImportWizard onImport={"example"} onCancel={() => {}} />
}`} lang="tsx" />
}`} lang="tsx" />

## API / Props

### BulkImportWizard

| Prop        | Type                          | Default | 说明                  |
| ----------- | ----------------------------- | ------- | --------------------- |
| `className` | `string`                      | —       | 自定义样式类名        |
| `onImport`  | `() => void`                  | —       |                       |
| `onCancel`  | `() => void`                  | —       |                       |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

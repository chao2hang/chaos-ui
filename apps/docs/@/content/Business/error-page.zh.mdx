# ErrorPage 错误页

> 业务组件

错误页 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要错误页时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { ErrorPage } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <ErrorPage>Default</ErrorPage>
}`} lang="tsx" />

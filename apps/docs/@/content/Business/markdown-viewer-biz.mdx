# MarkdownViewerBiz 业务Markdown

> 业务组件 / Domain-specific business components for ERP scenarios.

业务Markdown — 业务 Markdown 查看器，支持附件列表与下载链接。
MarkdownViewerBiz — Business markdown viewer with attachment support and download links.

## 何时使用 / When to Use

当你需要「业务Markdown」相关能力时使用：业务 Markdown 查看器，支持附件列表与下载链接。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use MarkdownViewerBiz when you need: Business markdown viewer with attachment support and download links. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

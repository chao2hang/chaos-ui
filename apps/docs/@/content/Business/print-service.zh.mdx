## API / Props

### PrintService

| Prop        | Type                   | Default | 说明                                            |
| ----------- | ---------------------- | ------- | ----------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                  |
| `jobs`      | `PrintServiceJob[]`    | —       | Queued print jobs.                              |
| `onPrint`   | `() => void`           | —       | Invoked when a job's print action is triggered. |
| `onRefresh` | `() => void`           | —       | Invoked when the user requests a queue refresh. |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                               |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

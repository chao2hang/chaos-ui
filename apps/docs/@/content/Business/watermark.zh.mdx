# Watermark 水印

> 业务组件

在内容上叠加指定文字/图片水印,用于版权标识。

## 何时使用

需要水印功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { Watermark } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Watermark />
}`} lang="tsx" />

## API / Props

### Watermark

| Prop        | Type                   | Default    | 说明              |
| ----------- | ---------------------- | ---------- | ----------------- |
| `className` | `string`               | —          | 自定义样式类名    |
| `text`      | `string`               | `内部资料` |                   |
| `image`     | `string`               | —          |                   |
| `rotate`    | `number`               | `-22`      |                   |
| `gap`       | `[number, number]`     | —          | 间距              |
| `fontSize`  | `number`               | `14`       |                   |
| `fullPage`  | `boolean`              | `true`     |                   |
| `zIndex`    | `number`               | `1000`     |                   |
| `opacity`   | `number`               | `0.08`     |                   |
| `color`     | `string`               | `#000`     | 颜色主题          |
| ...rest     | `React.HTMLAttributes` | —          | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

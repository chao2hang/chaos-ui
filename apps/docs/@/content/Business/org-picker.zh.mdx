# OrgPicker 组织选择器

> 业务组件

组织选择器 — 组织选择器对话框，支持部门树与成员多选。

## 何时使用

当你需要「组织选择器」相关能力时使用：组织选择器对话框，支持部门树与成员多选。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

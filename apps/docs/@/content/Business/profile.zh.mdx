# Profile 个人资料

> 业务组件

个人资料 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要profile时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { Avatar, AvatarFallback, Button } from "@chaos_team/chaos-ui/ui"

  return <ProfileHeader>Default</ProfileHeader>
}`} lang="tsx" />

## API / Props

### ProfileHeader

| Prop             | Type                          | Default            | 说明                        |
| ---------------- | ----------------------------- | ------------------ | --------------------------- |
| `className`      | `string`                      | —                  | 自定义样式类名              |
| `user`           | `ProfileHeaderUser`           | —                  |                             |
| `stats`          | `Array<T>`                    | —                  |                             |
| `actions`        | `React.ReactNode`             | —                  |                             |
| `onAvatarUpload` | `() => void`                  | —                  |                             |
| `coverColor`     | `string`                      | `var(--brand-200)` | @default "var(--brand-200)" |
| ...rest          | `React.ComponentProps<"div">` | —                  | 透传 `<div>` 原生属性       |

## 注意事项

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

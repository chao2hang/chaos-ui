GaugeChart — gauge chart component.

## 何时使用 / When to Use

需要Gauge Chart时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use GaugeChart when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### GaugeChart

| Prop        | Type                   | Default   | 说明 / Description                                        |
| ----------- | ---------------------- | --------- | --------------------------------------------------------- |
| `className` | `string`               | —         | 自定义样式类名 <br/> Custom CSS class                     |
| `value`     | `number`               | `64`      | Current value.                                            |
| `min`       | `number`               | `0`       | Range floor.                                              |
| `max`       | `number`               | `100`     | Range ceiling.                                            |
| `label`     | `string`               | —         | Caption below the gauge.                                  |
| `color`     | `string`               | `#3b82f6` | Arc color.                                                |
| ...rest     | `React.HTMLAttributes` | —         | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

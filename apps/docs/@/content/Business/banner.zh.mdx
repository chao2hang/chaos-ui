横幅 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要横幅功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { Banner } from "@/components/business/banner"

## 样式变体

**变体**

| Variant | 说明 | 默认 |
| --------- | ------------------------------ | ---- |
| `info` | 信息提示 | ✓ |
| `success` | 成功状态 |  |
| `warning` | 警告状态 |  |
| `error` | 错误状态 |  |

## API / Props

### Banner

| Prop | Type | Default | 说明 |
| ------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string` | — | 自定义样式类名 |
| `variant` | `BannerVariant` | `info` | 样式变体 |
| `title` | `string` | — | 标题 |
| `description` | `string` | — | 描述文本 |
| `action`      | `React.ReactNode`      | —       |                                                           |
| `closable`    | `boolean`              | `true`  |                                                           |
| `onClose` | `() => void` | — | 关闭回调 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

# BudgetPacingCard 预算节奏卡片

> 业务组件

预算节奏卡片 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要预算节奏卡片功能时使用本组件。

## API / Props

### BudgetPacingCard

| Prop        | Type                   | Default    | 说明              |
| ----------- | ---------------------- | ---------- | ----------------- |
| `className` | `string`               | —          | 自定义样式类名    |
| `budget`    | `number`               | —          |                   |
| `spent`     | `number`               | —          |                   |
| `period`    | `string`               | `本月`     |                   |
| `label`     | `string`               | `预算消耗` | 标签文本          |
| `currency`  | `string`               | `¥`        |                   |
| ...rest     | `React.HTMLAttributes` | —          | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

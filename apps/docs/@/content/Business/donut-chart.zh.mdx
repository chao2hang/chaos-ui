## API / Props

### DonutChart

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `data` | `Array<T>` | — | Segments to render. |
| `size` | `number` | `128` | Donut pixel size. |
| `thickness` | `number` | `12` | Stroke thickness. |
| `centerLabel` | `string` | — | Center hole label. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

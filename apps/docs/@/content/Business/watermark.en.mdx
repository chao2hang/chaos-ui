# Watermark

> Domain-specific business components for ERP scenarios.

Overlays specified text/image watermark on content for copyright.

## When to Use

Use when you need a watermark.

## Examples

### Basic Usage

<CodeBlock code={`import { Watermark } from "@/components/ui/watermark"

export function Basic() {
  return <Watermark />
}`} lang="tsx" />

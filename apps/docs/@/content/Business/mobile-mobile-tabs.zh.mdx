# MobileTabs Mobile Mobile Tabs

> 业务组件

Mobile Mobile Tabs — 面向 ERP/企业场景的业务组件。

## 何时使用

需要统一页面框架或区域编排时使用。典型用于「MobileTabs」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## 代码示例

### 基础用法

<CodeBlock code={`import { MobileTabs } from "@chaos_team/chaos-ui/mobile"

  return <MobileTabs />
}`} lang="tsx" />

## API / Props

### MobileTabs

| Prop           | Type                   | Default | 说明              |
| -------------- | ---------------------- | ------- | ----------------- |
| `className`    | `string`               | —       | 自定义样式类名    |
| `tabs`         | `MobileTabItem[]`      | —       |                   |
| `defaultValue` | `string`               | —       | 默认值(非受控)    |
| ...rest        | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

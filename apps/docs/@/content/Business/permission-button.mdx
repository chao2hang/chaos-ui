## API / Props

### PermissionButton

| Prop          | Type                                       | Default   | 说明 / Description                                        |
| ------------- | ------------------------------------------ | --------- | --------------------------------------------------------- |
| `className`   | `string`                                   | —         | 自定义样式类名 <br/> Custom CSS class                     |
| `permission`  | `string`                                   | —         | 所需权限码                                                |
| `permissions` | `string[]`                                 | —         | 当前用户拥有的权限码集合                                  |
| `mode`        | `"disable" \| "hide"`                      | `disable` | 无权限时的处理方式                                        |
| `onClick`     | `() => void`                               | —         | 点击回调                                                  |
| `variant`     | `"default" \| "outline" \| "secondary"...` | `default` | 按钮样式                                                  |
| `disabled`    | `boolean`                                  | `false`   | 禁用状态（外部控制，与权限叠加）                          |
| ...rest       | `React.HTMLAttributes`                     | —         | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

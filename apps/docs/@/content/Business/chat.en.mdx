# Chat

> Domain-specific business components for ERP scenarios.

Chat — a business component for ERP / enterprise scenarios.

## When to Use

Use Chat when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## Examples

### Basic Usage

<CodeBlock code={`import { ChatShell, ChatMessageBubble } from "@chaos_team/chaos-ui/business";

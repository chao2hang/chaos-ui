# FeatureGate 功能开关

> 业务组件

功能开关 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要feature-gate时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { PermissionButton } from "@chaos_team/chaos-ui/business";

export function Basic() {
  // docs-only FeatureGate was removed; PermissionButton illustrates gated UI
  return <PermissionButton permission="order:create">创建订单</PermissionButton>;
}`} lang="tsx" />

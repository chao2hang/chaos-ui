# Banner

> Domain-specific business components for ERP scenarios.

Banner — a business component for ERP / enterprise scenarios.

## When to Use

Use Banner when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## Style Variants

**Variant**

| Variant   | Description         | 默认 |
| --------- | ------------------- | ---- |
| `info`    | Informational style | ✓    |
| `success` | Success style       |      |
| `warning` | Warning style       |      |
| `error`   | Error style         |      |

## API / Props

### RedPacket

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `label` | `string` | — | Amount or greeting text. |
| `subtitle` | `string` | — | Subtitle. |
| `opened` | `boolean` | — | Whether the red packet has been opened. |
| `onOpen` | `() => void` | — | Called when user clicks to open. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# CopyButton 复制按钮

> 业务组件

复制按钮 — 面向 ERP/企业场景的业务组件。

## 何时使用

当你需要「复制按钮」相关能力时使用：一键复制按钮，带成功反馈。作为页面中的通用基础能力使用；优先复用而非在业务里再造一套。

## API / Props

### CopyButton

| Prop          | Type                                | Default  | 说明              |
| ------------- | ----------------------------------- | -------- | ----------------- |
| `className`   | `string`                            | —        | 自定义样式类名    |
| `text`        | `string`                            | —        |                   |
| `label`       | `string`                            | `复制`   | 标签文本          |
| `copiedLabel` | `string`                            | `已复制` |                   |
| `variant`     | `"default" \| "ghost" \| "outline"` | `ghost`  | 样式变体          |
| ...rest       | `React.HTMLAttributes`              | —        | 透传原生 DOM 属性 |

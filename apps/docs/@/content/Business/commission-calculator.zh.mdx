# CommissionCalculator Commission Calculator

> 业务组件

Commission Calculator — 面向 ERP/企业场景的业务组件。

## 何时使用

在需要Commission Calculator时使用：Commission Calculator — 组件库业务组件，详见 Storybook 与源码。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## API / Props

### CommissionCalculator

| Prop             | Type                   | Default | 说明                                              |
| ---------------- | ---------------------- | ------- | ------------------------------------------------- |
| `className`      | `string`               | —       | 自定义样式类名                                    |
| `tiers`          | `CommissionTier[]`     | —       | Commission tier structure.                        |
| `deals`          | `CommissionDeal[]`     | —       | Sales deals.                                      |
| `quota`          | `number`               | —       | Quota for multiplier. If met, bonus rate applies. |
| `quotaBonusRate` | `number`               | `0`     | Bonus rate when quota is met.                     |
| `currencySymbol` | `string`               | `¥`     | Currency symbol.                                  |
| `onDealsChange`  | `() => void`           | —       | On deals change.                                  |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

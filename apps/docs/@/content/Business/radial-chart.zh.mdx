## API / Props

### RadialChart

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `value` | `number` | `66` | Current value. |
| `max` | `number` | `100` | Ceiling. |
| `label` | `string` | `达成率` | Caption below the ring. |
| `color` | `string` | `#3b82f6` | Ring color. |
| `size` | `number` | `120` | Pixel size. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

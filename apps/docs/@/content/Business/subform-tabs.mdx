# SubformTabs Subform Tabs

> 业务组件 / Domain-specific business components for ERP scenarios.

子表单 tab
SubformTabs — subform tabs component.

## 何时使用 / When to Use

需要Subform Tabs时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use SubformTabs when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### SubformTabs

| Prop        | Type                      | Default | 说明 / Description                                                 |
| ----------- | ------------------------- | ------- | ------------------------------------------------------------------ |
| `className` | `string`                  | —       | 自定义样式类名 <br/> Custom CSS class                              |
| `tabs`      | `SubformTab[]`            | —       | Tab sections to render. Defaults to a basic info / contact layout. |
| `value`     | `Record<string, unknown>` | —       | Field values keyed by name.                                        |
| `onChange`  | `() => void`              | —       | Called with the merged values whenever a field changes.            |
| ...rest     | `React.HTMLAttributes`    | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes          |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

## API / Props

### SubformTabs

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `tabs` | `SubformTab[]` | — | Tab sections to render. Defaults to a basic info / contact layout. |
| `value` | `Record<string, unknown>` | — | Field values keyed by name. |
| `onChange` | `() => void` | — | Called with the merged values whenever a field changes. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

# BomTreeEditor Bom Tree Editor

> 业务组件

Bom Tree Editor — 面向 ERP/企业场景的业务组件。

## 何时使用

需要采集或编辑业务字段，并进入校验/提交链路时使用。（组件：Bom Tree Editor）

## 代码示例

### 基础用法

<CodeBlock code={`import { BOMTreeEditor } from "@chaos_team/chaos-ui/business"

  return <BOMTreeEditor />
}`} lang="tsx" />

## API / Props

### BOMTreeEditor

| Prop             | Type                   | Default | 说明                           |
| ---------------- | ---------------------- | ------- | ------------------------------ |
| `className`      | `string`               | —       | 自定义样式类名                 |
| `items`          | `BOMItem[]`            | —       | BOM items (tree structure)     |
| `onChange`       | `() => void`           | —       | Change callback                |
| `readOnly`       | `boolean`              | `false` | Read-only mode                 |
| `showCostRollup` | `boolean`              | `true`  | Show cost rollup column        |
| `currencySymbol` | `string`               | `¥`     | Currency symbol (default: "¥") |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性              |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

## API / Props

### PrintService

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `jobs`      | `PrintServiceJob[]`    | —       | Queued print jobs.                                        |
| `onPrint`   | `() => void`           | —       | Invoked when a job's print action is triggered.           |
| `onRefresh` | `() => void`           | —       | Invoked when the user requests a queue refresh.           |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

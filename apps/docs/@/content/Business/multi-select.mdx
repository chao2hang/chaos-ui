# MultiSelect 多选

> 业务组件 / Domain-specific business components for ERP scenarios.

在弹出面板中选择多个选项,支持标签形式展示已选项。
Select multiple options from a panel; selections shown as tags.

## 何时使用 / When to Use

从预设选项中选择多个值。

Select multiple values from a predefined list.

## API / Props

### MultiSelect

| Prop                | Type                   | Default            | 说明 / Description                                        |
| ------------------- | ---------------------- | ------------------ | --------------------------------------------------------- |
| `className`         | `string`               | —                  | 自定义样式类名 <br/> Custom CSS class                     |
| `options`           | `MultiSelectOption[]`  | —                  | 可选项列表 <br/> Available options                        |
| `value`             | `string[]`             | —                  | 当前值(受控) <br/> Current value (controlled)             |
| `onChange`          | `() => void`           | —                  | 值变化回调 <br/> Callback when value changes              |
| `placeholder`       | `string`               | `Select...`        | 占位文本 <br/> Placeholder text                           |
| `searchPlaceholder` | `string`               | `Search...`        |                                                           |
| `emptyText`         | `string`               | `No results found` |                                                           |
| `disabled`          | `boolean`              | —                  | 是否禁用 <br/> Whether disabled                           |
| `maxCount`          | `number`               | `3`                |                                                           |
| `maxSelected`       | `number`               | —                  |                                                           |
| `clearable`         | `boolean`              | `true`             |                                                           |
| ...rest             | `React.HTMLAttributes` | —                  | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

## API / Props

### InlineEdit

| Prop          | Type                           | Default    | 说明 / Description                                            |
| ------------- | ------------------------------ | ---------- | ------------------------------------------------------------- |
| `className`   | `string`                       | —          | 自定义样式类名 <br/> Custom CSS class                         |
| `value`       | `string`                       | —          | 当前值(受控) <br/> Current value (controlled)                 |
| `onChange`    | `() => void`                   | —          | 值变化回调 <br/> Callback when value changes                  |
| `onSave`      | `() => void`                   | —          |                                                               |
| `placeholder` | `string`                       | `点击编辑` | 占位文本 <br/> Placeholder text                               |
| ...rest       | `React.ComponentProps<"span">` | —          | 透传 `<span>` 原生属性 <br/> Inherits all `<span>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<span>` 原生 HTML 属性。

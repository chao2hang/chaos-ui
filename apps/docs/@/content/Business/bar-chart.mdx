# BarChart Bar Chart

> 业务组件 / Domain-specific business components for ERP scenarios.

柱状图(分组/堆叠/横向)
BarChart — bar chart component.

## 何时使用 / When to Use

当你需要「Bar Chart」相关能力时使用：柱状图(分组/堆叠/横向)。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use BarChart when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### BarChart

| Prop          | Type                         | Default    | 说明 / Description                                        |
| ------------- | ---------------------------- | ---------- | --------------------------------------------------------- |
| `className`   | `string`                     | —          | 自定义样式类名 <br/> Custom CSS class                     |

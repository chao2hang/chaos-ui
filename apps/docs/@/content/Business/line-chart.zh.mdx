## API / Props

### LineChart

| Prop        | Type                   | Default | 说明                                               |
| ----------- | ---------------------- | ------- | -------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                     |
| `series`    | `Array<T>`             | —       | One or more named series.                          |
| `labels`    | `string[]`             | —       | X-axis labels aligned 1:1 with each series values. |
| `height`    | `number`               | —       | SVG height in px.                                  |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                  |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

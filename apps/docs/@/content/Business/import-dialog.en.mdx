## API / Props

### ImportDialog

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `open` | `boolean` | — | Open state (controlled) |
| `onOpenChange` | `() => void` | — | Callback when open state changes |
| `onImport` | `() => void` | — |  |
| `title` | `string` | `导入数据` | Title |
| `description` | `string` | `上传文件并预览后再确认导入` | Description text |
| `accept` | `string` | `.xlsx,.csv` |  |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

## Notes

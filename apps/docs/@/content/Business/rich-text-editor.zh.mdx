# RichTextEditor 富文本编辑器

> 业务组件

富文本编辑器 — 面向 ERP/企业场景的业务组件。

## 何时使用

提供带格式的富文本编辑能力(加粗、列表、链接等)。

## 代码示例

### 基础用法

<CodeBlock code={`import { RichTextEditor } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <RichTextEditor />
}`} lang="tsx" />

## API / Props

### RichTextEditor

| Prop          | Type                   | Default       | 说明              |
| ------------- | ---------------------- | ------------- | ----------------- |
| `className`   | `string`               | —             | 自定义样式类名    |
| `value`       | `string`               | ``            | 当前值(受控)      |
| `onChange`    | `() => void`           | —             | 值变化回调        |
| `placeholder` | `string`               | `开始输入...` | 占位文本          |
| `editable`    | `boolean`              | `true`        |                   |
| `showToolbar` | `boolean`              | `true`        |                   |
| `minHeight`   | `number`               | `200`         |                   |
| `onUpload`    | `(...args) => unknown` | —             |                   |
| ...rest       | `React.HTMLAttributes` | —             | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

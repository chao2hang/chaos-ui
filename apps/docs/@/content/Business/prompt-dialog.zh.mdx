# PromptDialog 提示对话框

> 业务组件

提示对话框 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要提示对话框时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { PromptDialog } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <PromptDialog />
}`} lang="tsx" />

## API / Props

### PromptDialog

| Prop           | Type                                       | Default  | 说明              |
| -------------- | ------------------------------------------ | -------- | ----------------- |
| `className`    | `string`                                   | —        | 自定义样式类名    |
| `open`         | `boolean`                                  | —        | 是否打开(受控)    |
| `onOpenChange` | `() => void`                               | —        | 打开状态变化回调  |
| `title`        | `string`                                   | `请输入` | 标题              |
| `description`  | `React.ReactNode`                          | —        | 描述文本          |
| `label`        | `string`                                   | —        | 标签文本          |
| `placeholder`  | `string`                                   | —        | 占位文本          |
| `defaultValue` | `string`                                   | ``       | 默认值(非受控)    |
| `confirmText`  | `string`                                   | `确认`   |                   |
| `cancelText`   | `string`                                   | `取消`   |                   |
| `onConfirm`    | `() => void`                               | —        |                   |
| `required`     | `boolean`                                  | `true`   | 是否必填          |
| `inputType`    | `"text" \| "email" \| "password" \| "n...` | `text`   |                   |
| ...rest        | `React.HTMLAttributes`                     | —        | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

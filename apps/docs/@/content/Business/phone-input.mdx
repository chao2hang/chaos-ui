# PhoneInput 手机号输入

> 业务组件 / Domain-specific business components for ERP scenarios.

手机号输入 — 面向 ERP/企业场景的业务组件。
PhoneInput — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

当你需要「电话输入」相关能力时使用：国际电话号码输入，支持区号选择。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use PhoneInput when you need: International phone number input with country code selector. Use to collect or edit data with validation and submit flows.

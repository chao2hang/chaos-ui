# Workflow 工作流

> 业务组件 / Domain-specific business components for ERP scenarios.

工作流 — 面向 ERP/企业场景的业务组件。
Workflow — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要workflow时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use Workflow when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

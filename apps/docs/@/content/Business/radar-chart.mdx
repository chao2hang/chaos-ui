## API / Props

### RadarChart

| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `axes`      | `string[]`             | —       | Dimension labels (vertices).                              |
| `series`    | `Array<T>`             | —       | Named series; each `values` array aligns 1:1 with `axes`. |
| `max`       | `number`               | —       | Scale ceiling.                                            |

# EmptyState 空状态

> 业务组件

当前没有数据时的占位展示,可搭配引导操作。

## 何时使用

当你需要「空状态」相关能力时使用：当前没有数据时的占位展示,可搭配引导操作。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

## 代码示例

### 基础用法

<CodeBlock code={`import { EmptyState, Config } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <EmptyState />
  <Config />
    </div>
  )
}`} lang="tsx" />

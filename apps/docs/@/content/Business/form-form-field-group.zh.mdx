# FormFieldGroup 表单字段分组

> 业务组件

表单字段分组 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要表单字段分组功能时使用本组件。

## API / Props

### FormFieldGroup

| Prop | Type | Default | 说明 |
| ------------- | ---------------------------------- | ------- | --------------------------------------------------------------------- |
| `className` | `string` | — | 自定义样式类名 |
| `legend`      | `string`                           | —       |                                                                       |
| `description` | `string` | — | 描述文本 |
| `required` | `boolean` | — | 是否必填 |
| `columns` | `1 \| 2 \| 3` | `1` | 列配置 |
| ...rest | `React.ComponentProps<"fieldset">` | — | 透传 `<fieldset>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<fieldset>` 原生 HTML 属性。

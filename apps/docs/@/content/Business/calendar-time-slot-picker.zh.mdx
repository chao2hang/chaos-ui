# TimeSlotPicker 时间段选择器

> 业务组件

时间段选择器 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要calendar-time-slot-picker时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## API / Props

### TimeSlotPicker

| Prop          | Type                   | Default | 说明              |
| ------------- | ---------------------- | ------- | ----------------- |
| `className`   | `string`               | —       | 自定义样式类名    |
| `date`        | `Date`                 | —       |                   |
| `slots`       | `TimeSlot[]`           | —       |                   |
| `selected`    | `Date`                 | —       |                   |
| `onSelect`    | `() => void`           | —       | 选中回调          |
| `startHour`   | `number`               | `9`     |                   |
| `endHour`     | `number`               | `18`    |                   |
| `stepMinutes` | `number`               | `30`    |                   |
| `locale`      | `string`               | `zh-CN` |                   |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

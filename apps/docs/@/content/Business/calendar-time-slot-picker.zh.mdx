| `date`        | `Date`                 | —       |                                                           |
| `slots`       | `TimeSlot[]`           | —       |                                                           |
| `selected`    | `Date`                 | —       |                                                           |
| `onSelect` | `() => void` | — | 选中回调 |
| `startHour`   | `number`               | `9`     |                                                           |

## API / Props

### GanttChartPro

| Prop                 | Type                   | Default | 说明 / Description                                        |
| -------------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`          | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `tasks`              | `GanttTask[]`          | —       | Task list                                                 |
| `dependencies`       | `GanttDependency[]`    | —       | Dependency links between tasks                            |
| `zoom`               | `ZoomLevel`            | `day`   | Timeline zoom level (default: "day")                      |
| `showToday`          | `boolean`              | `true`  | Show today marker (default: true)                         |
| `showWeekSeparators` | `boolean`              | `true`  | Show week column separators (default: true)               |
| `showTaskList`       | `boolean`              | `true`  | Show task list sidebar (default: true)                    |
| `rowHeight`          | `number`               | `36`    | Row height in px (default: 36)                            |
| `onTaskClick`        | `() => void`           | —       | Allow task bar click                                      |
| ...rest              | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

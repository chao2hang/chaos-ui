# CodeEditor 代码编辑器

> 业务组件

代码编辑器 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要代码编辑器功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { CodeEditor } from "@/components/business/code-editor"

  return <CodeEditor />
}`} lang="tsx" />

## API / Props

### CodeEditor

| Prop              | Type                   | Default      | 说明              |
| ----------------- | ---------------------- | ------------ | ----------------- |
| `className`       | `string`               | —            | 自定义样式类名    |
| `value`           | `string`               | ``           | 当前值(受控)      |
| `onChange`        | `() => void`           | —            | 值变化回调        |
| `language`        | `SupportedLang`        | `typescript` |                   |
| `readonly`        | `boolean`              | `false`      |                   |
| `showLineNumbers` | `boolean`              | `true`       |                   |
| `showCopy`        | `boolean`              | `true`       |                   |
| `height`          | `string \| number`     | `320`        | 组件高度          |
| `theme`           | `"light" \| "dark"`    | `light`      | 主题              |
| `filename`        | `string`               | —            |                   |
| ...rest           | `React.HTMLAttributes` | —            | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

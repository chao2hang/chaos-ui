## API / Props

### DensitySwitcher

| Prop        | Type                                      | Default   | 说明              |
| ----------- | ----------------------------------------- | --------- | ----------------- |
| `className` | `string`                                  | —         | 自定义样式类名    |
| `open`      | `boolean`                                 | —         | 是否打开(受控)    |
| `density`   | `"compact" \| "default" \| "comfortable"` | `default` |                   |
| `onChange`  | `() => void`                              | —         | 值变化回调        |
| `size`      | `"sm" \| "default" \| "lg"`               | `default` | 组件尺寸          |
| ...rest     | `React.HTMLAttributes`                    | —         | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

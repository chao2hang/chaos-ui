# FieldMask 字段掩码

> 业务组件

字段掩码 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要字段掩码功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { FieldMask } from "@chaos_team/chaos-ui/business"

## API / Props

### FieldMask

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `mask` | `MaskType` | `phone` |  |
| `customPattern` | `string` | — |  |
| `value` | `string` | — | 当前值(受控) |
| `onChange` | `() => void` | — | 值变化回调 |
| ...rest | `React.ComponentProps<"input">` | — | 透传 `<input>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<input>` 原生 HTML 属性。

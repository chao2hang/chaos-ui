## API / Props

### RadarChart

| Prop        | Type                   | Default | 说明                                                      |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                            |
| `axes`      | `string[]`             | —       | Dimension labels (vertices).                              |
| `series`    | `Array<T>`             | —       | Named series; each `values` array aligns 1:1 with `axes`. |
| `max`       | `number`               | —       | Scale ceiling.                                            |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                         |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

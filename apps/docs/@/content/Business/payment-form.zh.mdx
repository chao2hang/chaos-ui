# PaymentForm 支付表单

> 业务组件

支付表单 — 面向 ERP/企业场景的业务组件。

## 何时使用

订单支付信息填写表单。

## 代码示例

### 基础用法

<CodeBlock code={`import { PaymentForm } from "@/components/business/payment-form"

export function Basic() {
  return (
    <PaymentForm
      onSubmit={(data) => console.log(data)}
    />
  )
}`} lang="tsx" />

## API / Props

### PaymentForm

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `onSubmit`  | `() => void`           | —       | 提交回调          |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **表单校验**: 集成 `zod` + `react-hook-form`,支持类型安全的表单校验与提交。

## API / Props

### SocialShare

| Prop        | Type                                       | Default | 说明                                                 |
| ----------- | ------------------------------------------ | ------- | ---------------------------------------------------- |
| `className` | `string`                                   | —       | 自定义样式类名                                       |
| `url`       | `string`                                   | —       | URL to share.                                        |
| `title`     | `string`                                   | —       | Title for the shared content.                        |
| `platforms` | `("wechat" \| "weibo" \| "qq" \| "link...` | —       | Platforms to show. Default: wechat, weibo, qq, link. |
| ...rest     | `React.HTMLAttributes`                     | —       | 透传原生 DOM 属性                                    |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

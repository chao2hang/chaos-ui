# MobileSkeleton Mobile Mobile Skeleton

> 业务组件

Mobile Mobile Skeleton — 面向 ERP/企业场景的业务组件。

## 何时使用

需要结构化展示多行业务数据，并支持浏览/对比时使用。典型用于「MobileListItemSkeleton」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## 代码示例

### 基础用法

<CodeBlock code={`import { MobileListItemSkeleton } from "@chaos_team/chaos-ui/mobile"

export function Basic() {
  return <MobileListItemSkeleton>Default</MobileListItemSkeleton>
}`} lang="tsx" />

# MobileTabs 移动端标签页

> 业务组件

移动端标签页 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要移动端标签页功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { MobileTabs } from "@/components/mobile/mobile-tabs"

  return <MobileTabs />
}`} lang="tsx" />

## API / Props

### MobileTabs

| Prop           | Type                   | Default | 说明              |
| -------------- | ---------------------- | ------- | ----------------- |
| `className`    | `string`               | —       | 自定义样式类名    |
| `tabs`         | `MobileTabItem[]`      | —       |                   |
| `defaultValue` | `string`               | —       | 默认值(非受控)    |
| ...rest        | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# MapView 地图视图

> 业务组件

地图视图 — 面向 ERP/企业场景的业务组件。

## 何时使用

在页面中嵌入地图,展示标记点、轨迹和位置信息。

## 代码示例

### 基础用法

<CodeBlock code={`import { MapView } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <MapView />
}`} lang="tsx" />

## API / Props

### MapView

| Prop         | Type                          | Default | 说明                                                          |
| ------------ | ----------------------------- | ------- | ------------------------------------------------------------- |
| `className`  | `string`                      | —       | 自定义样式类名                                                |
| `center`     | `[number, number]`            | —       | 地图中心点坐标 [经度, 纬度]                                   |
| `zoom`       | `number`                      | `11`    | 缩放级别                                                      |
| `markers`    | `MapMarker[]`                 | —       | 标记点数组                                                    |
| `height`     | `number \| string`            | `400`   | 组件高度                                                      |
| `apiKey`     | `string`                      | —       | 高德地图 JS API key。生产环境必须通过 props 传入,不要硬编码。 |
| `apiVersion` | `string`                      | `2.0`   | 高德 JS API 版本,默认 "2.0"                                   |
| ...rest      | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性                                         |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

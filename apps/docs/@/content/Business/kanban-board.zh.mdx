# KanbanBoard 看板

> 业务组件

看板 — 面向 ERP/企业场景的业务组件。

## 何时使用

看板视图,以列和卡片形式展示任务/工单。

## 代码示例

### 基础用法

<CodeBlock code={`import { KanbanBoard } from "@/components/business/kanban-board"

export function Basic() {
  return <KanbanBoard />
}`} lang="tsx" />

## API / Props

### KanbanBoard

| Prop              | Type                   | Default | 说明              |
| ----------------- | ---------------------- | ------- | ----------------- |
| `className`       | `string`               | —       | 自定义样式类名    |
| `columns`         | `KanbanColumnData[]`   | —       | 列配置            |
| `onColumnsChange` | `() => void`           | —       |                   |
| `renderCard`      | `(...args) => unknown` | —       |                   |
| ...rest           | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# ColorBoard Color Board

> 业务组件 / Domain-specific business components for ERP scenarios.

Color Board — 面向 ERP/企业场景的业务组件。
ColorBoard — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要Color Board时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use ColorBoard when you need: ColorBoard: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### ColorBoard

| Prop               | Type                   | Default   | 说明 / Description                                        |
| ------------------ | ---------------------- | --------- | --------------------------------------------------------- |
| `className`        | `string`               | —         | 自定义样式类名 <br/> Custom CSS class                     |
| `value`            | `string`               | `#3b82f6` | 当前颜色值(HEX 格式,如 #3b82f6)                           |
| `onChange`         | `() => void`           | —         | 颜色变化回调                                              |
| `format`           | `ColorFormat`          | `hex`     | 显示格式标签                                              |
| `presets`          | `string[]`             | —         | 预设颜色列表                                              |
| `history`          | `boolean`              | `true`    | 是否启用历史记录(localStorage 持久化)                     |
| `enableEyeDropper` | `boolean`              | `true`    | 是否启用吸管按钮(需浏览器支持 EyeDropper API)             |
| `enableAlpha`      | `boolean`              | `false`   | 是否启用透明度滑块                                        |
| `showSwatches`     | `boolean`              | `true`    | 是否显示预设色板                                          |
| ...rest            | `React.HTMLAttributes` | —         | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# MobileShareSheet Mobile Mobile Share Sheet

> 业务组件

Mobile Mobile Share Sheet — 面向 ERP/企业场景的业务组件。

## 何时使用

需要在当前页之上完成确认、短表单或辅助流程时使用。典型用于「MobileShareSheet」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## API / Props

### MobileShareSheet

| Prop        | Type                   | Default | 说明                                |
| ----------- | ---------------------- | ------- | ----------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                      |
| `open`      | `boolean`              | —       | Whether the sheet is open.          |
| `onClose`   | `() => void`           | —       | Called when the sheet is dismissed. |
| `title`     | `string`               | —       | Share title.                        |
| `platforms` | `() => void`           | —       | Share platforms to show.            |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                   |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

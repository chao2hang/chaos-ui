## API / Props

### TreeCrudPage

| Prop              | Type                   | Default | 说明 / Description                                        |
| ----------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`       | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `tree`            | `TreeNodeData[]`       | —       | 左侧分类树                                                |
| `selected`        | `string`               | —       | 当前选中节点 id（受控）                                   |
| `defaultSelected` | `string`               | —       | 默认选中节点 id（非受控）                                 |
| `onSelect`        | `() => void`           | —       | 选中节点回调                                              |
| `onCreate`        | `() => void`           | —       | 点击“新增分类”回调；不传则隐藏按钮                        |
| `onRefresh`       | `() => void`           | —       | 点击“刷新”回调；不传则隐藏按钮                            |
| `query`           | `string`               | —       | 搜索关键字（受控）                                        |
| `onQueryChange`   | `() => void`           | —       | 搜索关键字变化回调                                        |
| ...rest           | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

# MobilePullToRefresh 移动端下拉刷新

> 业务组件

移动端下拉刷新 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要移动端下拉刷新功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { PullToRefresh } from "@/components/business/mobile-pull-to-refresh"

export function Basic() {
  return <PullToRefresh />
}`} lang="tsx" />

## API / Props

### PullToRefresh

| Prop             | Type                          | Default       | 说明                  |
| ---------------- | ----------------------------- | ------------- | --------------------- |
| `className`      | `string`                      | —             | 自定义样式类名        |
| `onRefresh`      | `() => void`                  | —             |                       |
| `threshold`      | `number`                      | `60`          |                       |
| `disabled`       | `boolean`                     | —             | 是否禁用              |
| `refreshingText` | `string`                      | `正在刷新...` |                       |
| `pullText`       | `string`                      | `下拉刷新`    |                       |
| `releaseText`    | `string`                      | `释放刷新`    |                       |
| ...rest          | `React.ComponentProps<"div">` | —             | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

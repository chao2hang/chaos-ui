# MobilePageHeader Mobile Mobile Page Header

> 业务组件

Mobile Mobile Page Header — 面向 ERP/企业场景的业务组件。

## 何时使用

需要统一页面框架或区域编排时使用。典型用于「MobilePageHeader」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

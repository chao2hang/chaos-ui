# Transfer 穿梭框

> 业务组件

穿梭框 — 面向 ERP/企业场景的业务组件。

## 何时使用

穿梭框,在左右两栏之间移动候选项进行选择。

## 代码示例

### 基础用法

<CodeBlock code={`import { Transfer } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Transfer />
}`} lang="tsx" />

## API / Props

### Transfer

| Prop           | Type                   | Default | 说明              |
| -------------- | ---------------------- | ------- | ----------------- |
| `className`    | `string`               | —       | 自定义样式类名    |
| `dataSource`   | `TransferItem[]`       | —       |                   |
| `targetKeys`   | `string[]`             | —       |                   |
| `onChange`     | `() => void`           | —       | 值变化回调        |
| `titles`       | `[string, string]`     | —       |                   |
| `searchable`   | `boolean`              | `true`  |                   |
| `filterOption` | `(...args) => unknown` | —       |                   |
| `disabled`     | `boolean`              | —       | 是否禁用          |
| `oneWay`       | `boolean`              | `false` |                   |
| ...rest        | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# Anchor 锚点

> 业务组件

跳转到页面指定位置的锚点导航,常用于目录。

## 何时使用

需要锚点功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { Anchor } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Anchor />
}`} lang="tsx" />

## API / Props

### Anchor

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `sections`  | `AnchorSection[]`      | —       |                   |
| `offset`    | `number`               | `80`    | 偏移量            |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

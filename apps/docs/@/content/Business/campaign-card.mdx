# CampaignCard 活动卡片

> 业务组件 / Domain-specific business components for ERP scenarios.

活动卡片 — 面向 ERP/企业场景的业务组件。
CampaignCard — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要活动卡片时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use CampaignCard when you need: CampaignCard: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

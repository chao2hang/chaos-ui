# CoachMark 引导标记

> 业务组件

引导标记 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要coach-mark时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

### 基础用法

## API / Props

### CoachMark

| Prop           | Type                                     | Default  | 说明              |
| -------------- | ---------------------------------------- | -------- | ----------------- |
| `className`    | `string`                                 | —        | 自定义样式类名    |
| `open`         | `boolean`                                | —        | 是否打开(受控)    |
| `onOpenChange` | `() => void`                             | —        | 打开状态变化回调  |
| `target`       | `HTMLElement \| null`                    | —        | 链接打开目标      |
| `title`        | `string`                                 | —        | 标题              |
| `description`  | `React.ReactNode`                        | —        | 描述文本          |
| `placement`    | `"top" \| "bottom" \| "left" \| "right"` | `bottom` | 弹出位置          |
| `onNext`       | `() => void`                             | —        |                   |
| `onSkip`       | `() => void`                             | —        |                   |
| `nextLabel`    | `string`                                 | `下一步` |                   |
| `skipLabel`    | `string`                                 | `跳过`   |                   |
| `showSkip`     | `boolean`                                | `true`   |                   |
| ...rest        | `React.HTMLAttributes`                   | —        | 透传原生 DOM 属性 |

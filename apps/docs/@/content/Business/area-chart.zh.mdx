## API / Props

### AreaChart

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `data` | `number[]` | — | Numeric series values, left-to-right. |
| `labels` | `string[]` | — | X-axis labels aligned 1:1 with `data`. |
| `height` | `number` | `180` | SVG height in px. |
| `color` | `string` | `currentColor` | Stroke + fill color. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

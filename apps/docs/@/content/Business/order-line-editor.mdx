# OrderLineEditor 订单行编辑器

> 业务组件 / Domain-specific business components for ERP scenarios.

**轻预设**：内置 SKU / 单价 / 数量 / 小计 / 备注，适合演示或极简单据明细。  
**不是**完整下单引擎——赠送、双单位数量/金额换算、政策价等请在消费方用 `LineEditor` 编排。

Thin preset for SKU · price · qty · subtotal. Not a full order engine — gifts, UOM conversion, and domain pricing belong in the consumer app with `LineEditor`.

## 何时使用 / When to Use

| 场景                        | 推荐                           |
| --------------------------- | ------------------------------ |
| 演示、简单明细、无换算      | `OrderLineEditor`              |
| 真实下单、多单位、价税/政策 | `LineEditor` + 域列/换算       |
| 搭赠 / 赠品行               | `GiftLineEditor`（与主表分离） |

## 代码示例 / Examples

### 基础用法（轻预设） / Basic (thin preset)

<CodeBlock code={`import { OrderLineEditor } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return (
    <OrderLineEditor
      data={[{ id: "1", sku: "", qty: 1, price: 0 }]}
      onChange={() => {}}
      skuOptions={[{ value: "sku-1", label: "示例商品", price: 10 }]}
      currency="CNY"
    />
  )
}`} lang="tsx" />

### 复杂订单：自组 LineEditor（推荐） / Complex order: compose LineEditor

下单页请在业务侧定义列与换算（示意）：

<CodeBlock code={`import { LineEditor } from "@chaos_team/chaos-ui/business"
import { GiftLineEditor } from "@chaos_team/chaos-ui/business"

// columns: SKU picker, sales UOM, qty, baseQty (readonly from domain convert),
// price per UOM, subtotal (domain compute), remark …
// gifts: separate GiftLineEditor bound to promotion results

export function OrderLinesPanel({ lines, gifts, onLines, onGifts }) {
  return (
    <>
      <LineEditor columns={/* app columns */} data={lines} onChange={onLines} />
      <GiftLineEditor value={gifts} onChange={onGifts} />
    </>
  )
}`} lang="tsx" />

## API / Props

### OrderLineEditor

| Prop                  | Type                          | Default | 说明 / Description     |
| --------------------- | ----------------------------- | ------- | ---------------------- |
| `data`                | `OrderLine[]`                 | —       | 明细行 / Line rows     |
| `onChange`            | `(data: OrderLine[]) => void` | —       | 行变更 / Row change    |
| `skuOptions`          | `SkuOption[]`                 | `[]`    | SKU 选项 / SKU options |
| `onSkuSearch`         | `(keyword: string) => void`   | —       | SKU 搜索（若接入）     |
| `readOnly`            | `boolean`                     | `false` | 只读                   |
| `minRows` / `maxRows` | `number`                      | `1` / — | 行数限制               |
| `currency`            | `string`                      | `"¥"`   | 货币符号               |
| `className`           | `string`                      | —       | 样式类名               |
| ...rest               | `div` props                   | —       | 透传容器               |

## 注意事项 / Notes

- **Client Component**（`"use client"`）。
- 列文案目前为英文预设字符串；复杂中文 ERP 页面请用 `LineEditor` 自带中文列标题，而不是在本组件上堆 `labels` 业务 API（产品边界 #42）。
- 单位数量/金额换算、赠送策略属于**域逻辑**，应放在消费方（或业务 monorepo），不要期望本组件扩展全链路。

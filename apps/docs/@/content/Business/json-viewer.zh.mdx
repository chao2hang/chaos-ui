# JsonViewer JSON 查看器

> 业务组件

JSON 查看器 — 面向 ERP/企业场景的业务组件。

## 何时使用

在需要「JSON 查看器」时使用。用于 ERP/业务域流程（单据、选择、审批、结算等）。

## 代码示例

### 基础用法

<CodeBlock code={`import { JsonViewer } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <JsonViewer />
}`} lang="tsx" />

## API / Props

### JsonViewer

| Prop                    | Type                   | Default | 说明              |
| ----------------------- | ---------------------- | ------- | ----------------- |
| `className`             | `string`               | —       | 自定义样式类名    |
| `data`                  | `unknown`              | —       | 数据源            |
| `defaultCollapsedDepth` | `number`               | `3`     |                   |
| `rootClassName`         | `string`               | —       |                   |
| `showCopy`              | `boolean`              | `true`  |                   |
| `indentSize`            | `number`               | `2`     |                   |
| ...rest                 | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

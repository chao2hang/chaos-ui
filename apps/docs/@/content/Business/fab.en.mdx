# Fab

> Domain-specific business components for ERP scenarios.

Circular floating action button (bottom-right) for the primary create/add action.

## When to Use

Use when you need a fab.


## Examples

## API / Props

### Fab

| Prop | Type | Default | Description |
| ----------- | ------------------------------------------ | -------------- | --------------------------------------------------------- |
| `className` | `string` | — | Custom CSS class |
| `icon` | `React.ReactNode` | — | Icon element |
| `label` | `string` | — | Label text |
| `position`  | `"bottom-right" \| "bottom-left" \| "b...` | `bottom-right` |                                                           |
| `offset` | `number` | `4` | Offset |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(FabSpeedDial),使用组合模式按需导入。

# ImportMappingWizard Import Mapping Wizard

> 业务组件 / Domain-specific business components for ERP scenarios.

Import Mapping Wizard — 面向 ERP/企业场景的业务组件。
ImportMappingWizard — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要Import Mapping Wizard时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use ImportMappingWizard when you need: ImportMappingWizard: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### ImportMappingWizard

| Prop               | Type                   | Default | 说明 / Description                                        |
| ------------------ | ---------------------- | ------- | --------------------------------------------------------- |
| `className`        | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `sourceColumns`    | `SourceColumn[]`       | —       | Source columns from the file                              |
| `targetFields`     | `TargetField[]`        | —       | Target fields in the system                               |
| `mappings`         | `ColumnMapping[]`      | —       | Current mappings                                          |
| `onMappingsChange` | `() => void`           | —       | Mapping change callback                                   |
| `showSamples`      | `boolean`              | `true`  | Show sample data preview                                  |
| `onBack`           | `() => void`           | —       | Back button handler                                       |
| `onContinue`       | `() => void`           | —       | Continue/next button handler                              |
| ...rest            | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

# Workflow 工作流

> 业务组件

工作流 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要workflow时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { WorkflowDesigner } from "@chaos_team/chaos-ui/business";

export function Basic() {
  return (
    <div className="h-80 w-full">
      <WorkflowDesigner />
    </div>
  );
}`} lang="tsx" />

## API / Props

### WorkflowViewer

| Prop           | Type                          | Default | 说明                  |
| -------------- | ----------------------------- | ------- | --------------------- |
| `className`    | `string`                      | —       | 自定义样式类名        |
| `nodes`        | `Node<WorkflowNodeData>[]`    | —       |                       |
| `edges`        | `Edge[]`                      | —       |                       |
| `onNodeClick`  | `() => void`                  | —       |                       |
| `showControls` | `boolean`                     | `true`  |                       |
| `showMinimap`  | `boolean`                     | `false` |                       |
| `height`       | `number`                      | `480`   | 组件高度              |
| ...rest        | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

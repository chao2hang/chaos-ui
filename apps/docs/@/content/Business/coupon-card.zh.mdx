## API / Props

### CouponCard

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `value` | `string` | — | Coupon value display (e.g. "¥50" or "20% OFF"). |
| `description` | `string` | — | Coupon description or conditions. |
| `expiresAt` | `string` | — | Expiration date string. |
| `used` | `boolean` | — | Whether the coupon has been used. |
| `onUse` | `() => void` | — | Called when user clicks "Use" / "Claim". |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

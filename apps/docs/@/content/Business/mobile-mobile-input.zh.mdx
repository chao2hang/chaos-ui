# MobileInput Mobile Mobile Input

> 业务组件

Mobile Mobile Input — 面向 ERP/企业场景的业务组件。

## 何时使用

需要采集或编辑业务字段，并参与校验/提交时使用。典型用于「MobileInput」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

# DateRangePicker 日期范围选择器

> 业务组件

日期范围选择器 — 面向 ERP/企业场景的业务组件。

## 何时使用

选择日期范围。

## 代码示例

### 基础用法

<CodeBlock code={`import { DateRangePicker } from "@chaos_team/chaos-ui/business"

  return <DateRangePicker />

## API / Props

### DateRangePicker

| Prop             | Type                           | Default        | 说明              |
| ---------------- | ------------------------------ | -------------- | ----------------- |
| `className`      | `string`                       | —              | 自定义样式类名    |
| `value`          | `DateRange`                    | —              | 当前值(受控)      |
| `onChange`       | `() => void`                   | —              | 值变化回调        |
| `placeholder`    | `string`                       | `选择日期范围` | 占位文本          |
| `numberOfMonths` | `number`                       | `2`            |                   |
| `disabled`       | `boolean`                      | —              | 是否禁用          |
| `align`          | `"start" \| "center" \| "end"` | `start`        | 对齐方式          |
| `presets`        | `boolean`                      | `true`         |                   |
| ...rest          | `React.HTMLAttributes`         | —              | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

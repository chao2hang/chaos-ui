# BarChart 柱状图

> 业务组件

柱状图 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要charts-bar-chart时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## API / Props

### BarChart

| Prop             | Type                         | Default      | 说明                      |
| ---------------- | ---------------------------- | ------------ | ------------------------- |
| `data`           | `{ label; value; color? }[]` | 内置         | 柱数据                    |
| `orientation`    | `"vertical" \| "horizontal"` | `"vertical"` | 方向                      |
| `height`         | `number`                     | `200`        | 高度 px                   |
| `xLabelRotate`   | `number`                     | `0`          | 纵向标签旋转角            |
| `maxLabelLength` | `number`                     | —            | 标签截断长度              |
| `showTooltip`    | `boolean`                    | `true`       | 纵向柱悬停 tooltip（#22） |
| `className`      | `string`                     | —            | 根节点类名                |

## 注意事项

- **Client Component**: 使用 `"use client"`，依赖 React hooks/状态，需在客户端渲染。
- 纵向柱默认 hover 读数；横向柱已内联显示数值。
- 重交互/大数据请用 Recharts `chart.tsx` 系列。

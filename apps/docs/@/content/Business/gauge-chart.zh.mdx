## API / Props

### GaugeChart

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `value` | `number` | `64` | Current value. |
| `min` | `number` | `0` | Range floor. |
| `max` | `number` | `100` | Range ceiling. |
| `label` | `string` | — | Caption below the gauge. |
| `color` | `string` | `#3b82f6` | Arc color. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

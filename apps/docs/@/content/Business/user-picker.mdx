# UserPicker 用户选择器

> 业务组件 / Domain-specific business components for ERP scenarios.

用户选择器 — 用户选择器对话框，支持搜索与已选标签栏。
UserPicker — User selector dialog with search and selected chips bar.

## 何时使用 / When to Use

当你需要「用户选择器」相关能力时使用：用户选择器对话框，支持搜索与已选标签栏。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use UserPicker when you need: User selector dialog with search and selected chips bar. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

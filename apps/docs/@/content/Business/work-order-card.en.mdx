## API / Props

### WorkOrderCard

| Prop           | Type                                      | Default  | Description                       |
| -------------- | ----------------------------------------- | -------- | --------------------------------- |
| `className`    | `string`                                  | —        | Custom CSS class                  |
| `orderNo`      | `string`                                  | —        | Work order number.                |
| `product`      | `string`                                  | —        | Product being manufactured.       |
| `productCode`  | `string`                                  | —        | Product code.                     |
| `status`       | `WorkOrderStatus`                         | —        | Work order status.                |
| `plannedStart` | `string`                                  | —        | Planned start date.               |
| `plannedEnd`   | `string`                                  | —        | Planned end date.                 |
| `plannedQty`   | `number`                                  | —        | Total planned quantity.           |
| `completedQty` | `number`                                  | —        | Completed quantity.               |
| `unit`         | `string`                                  | `pcs`    | Unit.                             |
| `priority`     | `"low" \| "medium" \| "high" \| "urgent"` | `medium` | Priority.                         |
| `workCenter`   | `string`                                  | —        | Assigned work center.             |
| `operations`   | `WorkOrderOperation[]`                    | —        | Operations / routing steps.       |
| `supervisor`   | `string`                                  | —        | Responsible person.               |
| `onClick`      | `() => void`                              | —        | Click handler for the card.       |
| ...rest        | `React.HTMLAttributes`                    | —        | Inherits standard HTML attributes |

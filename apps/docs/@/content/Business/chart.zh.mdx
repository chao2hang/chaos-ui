# Chart 图表

> 业务组件

图表 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要图表功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { LineChart, BarChart, AreaChart } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return (
    <div className="space-y-4">
  <LineChart />
  <BarChart />
  <AreaChart />
    </div>
  )
}`} lang="tsx" />

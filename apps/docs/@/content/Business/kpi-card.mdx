# KPICard KPI 卡片

Chaos UI KPI 指标卡片组件，用于展示关键业务指标及趋势变化。
A KPI indicator card for displaying key business metrics with trend changes.

## 何时使用 / When to use

用于仪表盘和报表中展示核心指标：营收、用户数、转化率、订单量等。支持显示同比/环比变化、趋势折线图、目标达成率。

Use for dashboards and reports: revenue, user count, conversion rate, order volume. Supports YoY/MoM changes, sparkline trends, and target tracking.

## API

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| title | `string` | — | **必填**，指标标题 | **Required** indicator title |
| value | `string` | — | **必填**，指标数值（预格式化字符串） | **Required** indicator value (pre-formatted) |
| change | `string` | — | 变化值（如 "+12.5%") | Change value (e.g., "+12.5%") |
| changeLabel | `string` | — | 变化标签（如 "vs 上月"） | Change label (e.g., "vs last month") |
| changeType | `'positive' \| 'negative' \| 'neutral'` | `'neutral'` | 变化方向 | Change direction |
| sparkline | `{ value: number }[]` | — | 趋势折线数据 | Sparkline data points |
| target | `number` | — | 目标值 (0–100) | Target value (0–100) |
| targetLabel | `string` | — | 目标标签 | Target label |
| icon | `React.ElementType` | — | 自定义图标组件 | Custom icon component |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

## 注意事项

- **数值格式化**：`value` 应预格式化为字符串（如 `"¥ 847,290"`、`"24,821"`、`"3.24%"`），KPICard 不负责数值格式化。
- **change 方向**：`changeType` 控制箭头方向和颜色——`positive` 绿色向上，`negative` 红色向下，`neutral` 灰色横线。
- **sparkline 格式**：`{ value: number }[]` 数组，value 为相对数值（0–100 范围更佳），会自动缩放到卡片宽度。
- **target 范围**：`target` 接受 0–100 的百分比数值，渲染为进度条。
- **无障碍**：KPICard 为纯展示组件，无交互。建议配合语义化 `<section>` 或 `<article>` 包装，为标题添加适当 heading 层级。

# PaymentResult Payment Result

> 业务组件

Payment Result — 面向 ERP/企业场景的业务组件。

## 何时使用

在需要Payment Result时使用：Payment Result — 组件库业务组件，详见 Storybook 与源码。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

### 基础用法

## API / Props

### PaymentResult

| Prop             | Type                   | Default | 说明                              |
| ---------------- | ---------------------- | ------- | --------------------------------- |
| `className`      | `string`               | —       | 自定义样式类名                    |
| `status`         | `"success" \| "fail"`  | —       | Result status.                    |
| `orderId`        | `string`               | —       | Order or transaction ID.          |
| `amount`         | `string`               | —       | Amount paid.                      |
| `primaryLabel`   | `string`               | —       | Primary action button label.      |
| `onPrimary`      | `() => void`           | —       | Called on primary action click.   |
| `secondaryLabel` | `string`               | —       | Secondary action button label.    |
| `onSecondary`    | `() => void`           | —       | Called on secondary action click. |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                 |

## 注意事项

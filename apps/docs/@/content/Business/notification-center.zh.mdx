# NotificationCenter 通知中心

> 业务组件

通知中心 — 面向 ERP/企业场景的业务组件。

## 何时使用

站内消息/通知聚合列表,展示未读/已读状态。


## 代码示例

### 基础用法

<CodeBlock code={`import { NotificationCenter } from "@chaos_team/chaos-ui/business"

  return <NotificationCenter />
}`} lang="tsx" />

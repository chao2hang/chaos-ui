Use when you need a mobile pull to refresh.

## API / Props

### PullToRefresh

| Prop             | Type                          | Default       | Description                     |
| ---------------- | ----------------------------- | ------------- | ------------------------------- |
| `className`      | `string`                      | —             | Custom CSS class                |
| `onRefresh`      | `() => void`                  | —             |                                 |
| `threshold`      | `number`                      | `60`          |                                 |
| `disabled`       | `boolean`                     | —             | Whether disabled                |
| `refreshingText` | `string`                      | `正在刷新...` |                                 |
| `pullText`       | `string`                      | `下拉刷新`    |                                 |
| `releaseText`    | `string`                      | `释放刷新`    |                                 |
| ...rest          | `React.ComponentProps<"div">` | —             | Inherits all `<div>` attributes |

## Notes

# OeeDashboard Oee Dashboard

> 业务组件

Oee Dashboard — 面向 ERP/企业场景的业务组件。

## 何时使用

在需要「Oee Dashboard」时使用。用于 ERP/业务域流程（单据、选择、审批、结算等）。

## 代码示例

### 基础用法

<CodeBlock code={`import { OeeDashboard } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <OeeDashboard />
}`} lang="tsx" />

## API / Props

### OeeDashboard

| Prop        | Type                   | Default         | 说明                                  |
| ----------- | ---------------------- | --------------- | ------------------------------------- |
| `className` | `string`               | —               | 自定义样式类名                        |
| `equipment` | `OeeEquipment`         | —               | Equipment data.                       |
| `losses`    | `OeeLoss[]`            | —               | Loss breakdown items.                 |
| `title`     | `string`               | `OEE Dashboard` | Dashboard title.                      |
| `targetOee` | `number`               | `85`            | Target OEE percentage for comparison. |
| ...rest     | `React.HTMLAttributes` | —               | 透传原生 DOM 属性                     |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

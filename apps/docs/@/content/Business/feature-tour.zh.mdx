# FeatureTour Feature Tour

> 业务组件

## 何时使用

当你需要「Feature Tour（FeatureTour）」解决具体交互/展示问题时使用；优先看下方示例与 API，按业务字段接入。

## 代码示例

### 基础用法

<CodeBlock code={`import { FeatureTour } from "@chaos_team/chaos-ui/business"

  return <FeatureTour />
}`} lang="tsx" />

## API / Props

### FeatureTour

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `steps`     | `Array<T>`             | —       |                   |
| `open`      | `boolean`              | —       | 是否打开(受控)    |
| `onClose`   | `() => void`           | —       | 关闭回调          |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

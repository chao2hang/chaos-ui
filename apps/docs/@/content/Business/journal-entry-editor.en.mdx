## API / Props

### JournalEntryEditor

| Prop                | Type                   | Default | Description                        |
| ------------------- | ---------------------- | ------- | ---------------------------------- |
| `className`         | `string`               | —       | Custom CSS class                   |
| `lines`             | `JournalEntryLine[]`   | —       | Journal entry lines                |
| `onChange`          | `() => void`           | —       | Change callback with updated lines |
| `accounts`          | `AccountOption[]`      | —       | Available accounts for selection   |
| `readOnly`          | `boolean`              | `false` | Read-only mode                     |
| `currencySymbol`    | `string`               | `¥`     | Currency symbol (default: "¥")     |
| `entryDate`         | `string`               | —       | Entry date                         |
| `voucherNo`         | `string`               | —       | Entry description / voucher number |
| `showBalanceStatus` | `boolean`              | `true`  | Show balance status badge          |
| ...rest             | `React.HTMLAttributes` | —       | Inherits standard HTML attributes  |

## API / Props

### MultiCurrencyInput

| Prop                | Type                     | Default | Description                                                                     |
| ------------------- | ------------------------ | ------- | ------------------------------------------------------------------------------- |
| `className`         | `string`                 | —       | Custom CSS class                                                                |
| `value`             | `number`                 | `0`     | Amount value in the selected (source) currency                                  |
| `onChange`          | `() => void`             | —       | Change callback with amount and computed base amount                            |
| `currencies`        | `CurrencyOption[]`       | —       | Available currencies (default: common set)                                      |
| `currency`          | `string`                 | `USD`   | Selected currency code                                                          |
| `onCurrencyChange`  | `() => void`             | —       | Currency change callback                                                        |
| `baseCurrency`      | `string`                 | `CNY`   | Base currency code (default: "CNY")                                             |
| `rate`              | `number`                 | —       | Exchange rate (1 source = rate base). If not provided, looked up from rates map |
| `onRateChange`      | `() => void`             | —       | Rate override callback                                                          |
| `rates`             | `Record<string, number>` | —       | Exchange rate map (e.g. USD: 7.25, EUR: 7.85)                                   |
| `readOnly`          | `boolean`                | —       | Read-only mode                                                                  |
| `allowRateOverride` | `boolean`                | —       | Allow manual rate override (default: true)                                      |
| `showConversion`    | `boolean`                | —       | Show conversion preview (default: true)                                         |
| `placeholder`       | `string`                 | —       | Placeholder text                                                                |
| ...rest             | `React.HTMLAttributes`   | —       | Inherits standard HTML attributes                                               |

## API / Props

### TemplateDownload

| Prop         | Type                     | Default        | 说明                                                           |
| ------------ | ------------------------ | -------------- | -------------------------------------------------------------- |
| `className`  | `string`                 | —              | 自定义样式类名                                                 |
| `templates`  | `TemplateDownloadItem[]` | —              | Templates available for download.                              |
| `onDownload` | `() => void`             | —              | Called with the selected template when a link is clicked.      |
| `emptyLabel` | `string`                 | `暂无可用模板` | Text shown when the list is empty. Defaults to "暂无可用模板". |
| ...rest      | `React.HTMLAttributes`   | —              | 透传原生 DOM 属性                                              |

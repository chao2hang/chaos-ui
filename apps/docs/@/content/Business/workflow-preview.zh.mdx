# WorkflowPreview Workflow Preview

> 业务组件

<WorkflowPreview
  nodes={[
    { id: "n1", name: "提交申请", status: "done" },
    { id: "n2", name: "主管审批", status: "active" },
  ]}
  edges={[{ from: "n1", to: "n2" }]}
/>

## 何时使用

在审批、流转、审计轨迹相关流程中使用。典型用于「Workflow Preview」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

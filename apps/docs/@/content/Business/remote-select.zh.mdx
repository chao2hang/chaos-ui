# RemoteSelect 远程选择器

> 业务组件

远程选择器 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要远程选择器功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { RemoteSelect } from "@/components/business/remote-select"

  return <RemoteSelect />

## API / Props

### RemoteSelect

| Prop          | Type                   | Default         | 说明              |
| ------------- | ---------------------- | --------------- | ----------------- |
| `className`   | `string`               | —               | 自定义样式类名    |
| `value`       | `string`               | —               | 当前值(受控)      |
| `onChange`    | `() => void`           | —               | 值变化回调        |
| `onSearch`    | `(...args) => unknown` | —               |                   |
| `placeholder` | `string`               | `搜索并选择...` | 占位文本          |
| `debounceMs`  | `number`               | `300`           |                   |
| ...rest       | `React.HTMLAttributes` | —               | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

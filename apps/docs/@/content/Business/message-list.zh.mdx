## API / Props

### MessageList

| Prop            | Type                                       | Default | 说明              |
| --------------- | ------------------------------------------ | ------- | ----------------- |
| `className`     | `string`                                   | —       | 自定义样式类名    |
| `messages`      | `Array<{`                                  | —       | 消息数组          |
| `id`            | `string`                                   | —       | DOM 元素 id       |
| `title`         | `string`                                   | —       | 标题              |
| `content`       | `string`                                   | —       | 内容              |
| `type`          | `"info" \| "warning" \| "success" \| "...` | —       | 类型              |
| `time`          | `string \| number \| Date`                 | —       |                   |
| `read`          | `boolean`                                  | —       |                   |
| `onSelect`      | `() => void`                               | —       | 选中消息回调      |
| `onMarkAllRead` | `() => void`                               | —       | 全部标记已读回调  |
| ...rest         | `React.HTMLAttributes`                     | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

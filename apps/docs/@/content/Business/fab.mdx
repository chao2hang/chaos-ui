# Fab 悬浮按钮

> 业务组件 / Domain-specific business components for ERP scenarios.

悬浮在页面右下角的圆形操作按钮,用于最主要的创建/新增操作。
Circular floating action button (bottom-right) for the primary create/add action.

## 何时使用 / When to Use

需要悬浮按钮功能时使用本组件。

Use when you need a fab.

## 代码示例 / Examples

## API / Props

### Fab

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `icon` | `React.ReactNode` | — | 图标元素 <br/> Icon element |
| `label` | `string` | — | 标签文本 <br/> Label text |
| `position` | `"bottom-right" \| "bottom-left" \| "b...` | `bottom-right` |  |
| `offset` | `number` | `4` | 偏移量 <br/> Offset |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(FabSpeedDial),使用组合模式按需导入。

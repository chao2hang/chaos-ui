当你需要「悬浮按钮」相关能力时使用：悬浮在页面右下角的圆形操作按钮,用于最主要的创建/新增操作。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use Fab when you need: Circular floating action button (bottom-right) for the primary create/add action. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

## API / Props

### Fab

| Prop        | Type                                       | Default        | 说明 / Description                                        |
| ----------- | ------------------------------------------ | -------------- | --------------------------------------------------------- |
| `className` | `string`                                   | —              | 自定义样式类名 <br/> Custom CSS class                     |
| `icon`      | `React.ReactNode`                          | —              | 图标元素 <br/> Icon element                               |
| `label`     | `string`                                   | —              | 标签文本 <br/> Label text                                 |
| `position`  | `"bottom-right" \| "bottom-left" \| "b...` | `bottom-right` |                                                           |
| `offset`    | `number`                                   | `4`            | 偏移量 <br/> Offset                                       |
| ...rest     | `React.HTMLAttributes`                     | —              | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(FabSpeedDial),使用组合模式按需导入。

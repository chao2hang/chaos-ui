# GlobalLoading 全局加载

> 业务组件

全局加载 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要全局加载时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { GlobalLoading } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <GlobalLoading />
}`} lang="tsx" />

## API / Props

### GlobalLoading

| Prop         | Type                   | Default     | 说明              |
| ------------ | ---------------------- | ----------- | ----------------- |
| `className`  | `string`               | —           | 自定义样式类名    |
| `loading`    | `boolean`              | `true`      | 是否加载中        |
| `text`       | `string`               | `加载中...` |                   |
| `fullscreen` | `boolean`              | `true`      |                   |
| ...rest      | `React.HTMLAttributes` | —           | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# ImMessage Im Message

> 业务组件

Im Message — 面向 ERP/企业场景的业务组件。

## 何时使用

需要上传、预览或管理附件与媒体时使用。（组件：Im Message）

## API / Props

### ImMessage

| Prop          | Type                       | Default    | 说明              |
| ------------- | -------------------------- | ---------- | ----------------- |
| `className`   | `string`                   | —          | 自定义样式类名    |
| `direction`   | `"sent" \| "received"`     | `received` | 消息方向          |
| `author`      | `string`                   | ``         | 发送者名称        |
| `content`     | `string`                   | ``         | 消息正文          |
| `timestamp`   | `string \| number \| Date` | —          | 消息时间          |
| `attachments` | `Array<T>`                 | —          | 附件文件名列表    |
| `avatarUrl`   | `string`                   | —          | 头像地址（可选）  |
| ...rest       | `React.HTMLAttributes`     | —          | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

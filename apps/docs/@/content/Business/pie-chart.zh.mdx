# PieChart Pie Chart

> 业务组件

Pie Chart — 面向 ERP/企业场景的业务组件。

## 何时使用

需要把经营指标或趋势做成图表供扫读时使用。（组件：Pie Chart）

## 代码示例

### 基础用法

<CodeBlock code={`import { PieChart } from "@chaos_team/chaos-ui/business"

  return <PieChart />
}`} lang="tsx" />

## API / Props

### PieChart

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `data`      | `Array<T>`             | —       | Slices to render. |
| `size`      | `number`               | —       | Pie pixel size.   |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

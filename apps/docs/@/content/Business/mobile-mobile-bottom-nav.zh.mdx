# MobileBottomNav Mobile Mobile Bottom Nav

> 业务组件

Mobile Mobile Bottom Nav — 面向 ERP/企业场景的业务组件。

## 何时使用

在手机/窄屏流程中使用，优先触控与单手操作。典型用于「MobileBottomNav」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## API / Props

### SocialShare

| Prop        | Type                                       | Default | 说明 / Description                                        |
| ----------- | ------------------------------------------ | ------- | --------------------------------------------------------- |
| `className` | `string`                                   | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `url`       | `string`                                   | —       | URL to share.                                             |
| `title`     | `string`                                   | —       | Title for the shared content.                             |
| `platforms` | `("wechat" \| "weibo" \| "qq" \| "link...` | —       | Platforms to show. Default: wechat, weibo, qq, link.      |
| ...rest     | `React.HTMLAttributes`                     | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

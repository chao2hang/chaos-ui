# MobileListItem Mobile List Item

> 业务组件

## 何时使用

需要结构化展示多行业务数据，并支持浏览/对比时使用。典型用于「Mobile List Item」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## 代码示例

### 基础用法

  return <MobileListItem />
}`} lang="tsx" />

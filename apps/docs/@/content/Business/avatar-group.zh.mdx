# AvatarGroup 头像组

> 业务组件

多个头像堆叠展示,显示剩余数量 +N。

## 何时使用

当你需要「头像组」相关能力时使用：多个头像堆叠展示,显示剩余数量 +N。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## API / Props

### AvatarGroup

| Prop              | Type                                | Default   | 说明                  |
| ----------------- | ----------------------------------- | --------- | --------------------- |
| `className`       | `string`                            | —         | 自定义样式类名        |
| `users`           | `AvatarUser[]`                      | —         |                       |
| `max`             | `number`                            | `4`       | 最大值                |
| `size`            | `"sm" \| "default" \| "lg" \| "xl"` | `default` | 组件尺寸              |
| `showOverflow`    | `boolean`                           | `true`    |                       |
| `onOverflowClick` | `() => void`                        | —         |                       |
| ...rest           | `React.ComponentProps<"div">`       | —         | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

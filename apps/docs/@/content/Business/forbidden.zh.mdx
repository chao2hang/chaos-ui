# Forbidden 403 禁止访问

> 业务组件

403 禁止访问 — 面向 ERP/企业场景的业务组件。

## 何时使用

适用于403 禁止访问相关场景。


## 代码示例

### 基础用法

<CodeBlock code={`import { Forbidden } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <Forbidden />
}`} lang="tsx" />


## API / Props

### Forbidden

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `title` | `string` | `403 无权限` | 标题 |
| `description` | `string` | `您没有访问此页面的权限，请联系管理员。` | 描述文本 |
| `action` | `React.ReactNode` | — |  |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

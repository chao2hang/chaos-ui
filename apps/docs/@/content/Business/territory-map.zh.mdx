## API / Props

### TerritoryMap

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `regions` | `TerritoryRegion[]` | — | Territory regions. |
| `title` | `string` | `Sales Territory Map` | Map title. |
| `currencySymbol` | `string` | `¥` | Currency symbol. |
| `gridCols` | `number` | `4` | Grid columns for layout. |
| `onRegionClick` | `() => void` | — | Region click handler. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

## API / Props

### AuditTrailDiff

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `entries` | `AuditChangeEntry[]` | — | Audit change entries |
| `showFilter` | `boolean` | `true` | Show search/filter bar (default: true) |
| `showEntity` | `boolean` | `true` | Show entity column (default: true) |
| `showUser` | `boolean` | `true` | Show user column (default: true) |
| `showTimestamp` | `boolean` | `true` | Show timestamp column (default: true) |
| `showReason` | `boolean` | `false` | Show reason column (default: false) |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

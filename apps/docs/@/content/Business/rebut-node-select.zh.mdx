# RebutNodeSelect Rebut Node Select

> 业务组件

## 何时使用

需要从主数据/档案中挑选一条或多条记录时使用（支持搜索、确认回填）。典型用于「Rebut Node Select」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

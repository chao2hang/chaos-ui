## API / Props

### CheckoutBar

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `total` | `string \| number` | — | Total amount to display. |
| `currency` | `string` | `¥` | Currency symbol or code. |
| `payLabel` | `string` | `Pay Now` | Checkout button label. Default "Pay Now". |
| `isLoading` | `boolean` | — | Whether the checkout is in a loading/processing state. |
| `onPay` | `() => void` | — | Called when the user clicks the pay button. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# AuthGuard 认证守卫

> 业务组件

认证守卫 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要认证守卫功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { AuthGuard, AuthGuardProvider, Permissions } from "@/components/business/auth-guard"

export function Basic() {
  return (
    <AuthGuard>
      <AuthGuardProvider>Content</AuthGuardProvider>
    </AuthGuard>
  )
}`} lang="tsx" />


## API / Props

### AuthGuard

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `permission` | `string \| string[]` | — |  |
| `permissions` | `string[]` | — |  |
| `mode` | `"all" \| "any"` | `any` |  |
| `fallback` | `React.ReactNode` | — |  |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(AuthGuardProvider),使用组合模式按需导入。

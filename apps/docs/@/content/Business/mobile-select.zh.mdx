需要移动端选择器功能时使用本组件。

## API / Props

### MobileSelect

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `options` | `MobileSelectOption[]` | — | 可选项列表 |
| `value` | `string` | — | 当前值(受控) |
| `onValueChange` | `() => void` | — |  |
| `placeholder` | `string` | — | 占位文本 |
| `disabled` | `boolean` | — | 是否禁用 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

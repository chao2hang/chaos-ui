## API / Props

### Chip

| Prop        | Type                           | Default   | 说明 / Description                                            |
| ----------- | ------------------------------ | --------- | ------------------------------------------------------------- |
| `className` | `string`                       | —         | 自定义样式类名 <br/> Custom CSS class                         |
| `variant`   | `ChipVariant`                  | `default` | 样式变体 <br/> Style variant                                  |
| `size`      | `ChipSize`                     | `default` | 组件尺寸 <br/> Component size                                 |
| `removable` | `boolean`                      | —         |                                                               |
| `onRemove`  | `() => void`                   | —         |                                                               |
| `icon`      | `React.ReactNode`              | —         | 图标元素 <br/> Icon element                                   |
| ...rest     | `React.ComponentProps<"span">` | —         | 透传 `<span>` 原生属性 <br/> Inherits all `<span>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<span>` 原生 HTML 属性。

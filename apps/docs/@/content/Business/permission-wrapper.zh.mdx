# PermissionWrapper 权限包装器

> 业务组件

权限包装器 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要权限包装器时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { PermissionWrapper } from "@chaos_team/chaos-ui/business"

  return <PermissionWrapper />
}`} lang="tsx" />

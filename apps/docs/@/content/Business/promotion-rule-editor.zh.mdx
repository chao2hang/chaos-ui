# PromotionRuleEditor Promotion Rule Editor

> 业务组件

促销规则编辑器

## 何时使用

需要采集或编辑业务字段，并参与校验/提交时使用。典型用于「Promotion Rule Editor」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## 代码示例

### 基础用法

## API / Props

### PromotionRuleEditor

| Prop        | Type                   | Default | 说明                                                    |
| ----------- | ---------------------- | ------- | ------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                          |
| `rules`     | `PromotionRule[]`      | —       | Initial rules. Defaults to a single empty starter rule. |
| `onChange`  | `() => void`           | —       | Called with the current rules whenever they change.     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                       |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# ScreenCapture 屏幕截图

> 业务组件 / Domain-specific business components for ERP scenarios.

屏幕截图 — 屏幕截图与标注组件，基于 getDisplayMedia API。
ScreenCapture — Screen capture and annotation component with getDisplayMedia API.

## 何时使用 / When to Use

当你需要「屏幕截图」相关能力时使用：屏幕截图与标注组件，基于 getDisplayMedia API。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use ScreenCapture when you need: Screen capture and annotation component with getDisplayMedia API. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

## API / Props

### DashboardGrid

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `title` | `string` | `仪表盘` | 仪表盘标题 |
| `stats` | `Array<{` | — | KPI 指标数组 |
| `label` | `string` | — | 标签文本 <br/> Label text |
| `value` | `number` | — | 当前值(受控) <br/> Current value (controlled) |
| `delta` | `number` | — | 环比变化，正数增长负数下降 |
| `trend` | `"up" \| "down"` | — | 显式趋势方向，省略时按 delta 推断 |
| `columns` | `1 \| 2 \| 3 \| 4 \| 6` | `4` | 网格列数 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

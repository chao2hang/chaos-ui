# DashboardGrid Dashboard Grid

> 业务组件 / Domain-specific business components for ERP scenarios.

Dashboard Grid — 面向 ERP/企业场景的业务组件。
DashboardGrid — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要Dashboard Grid时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use DashboardGrid when you need: DashboardGrid: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

### DashboardGrid

| Prop        | Type                    | Default  | 说明 / Description                                        |

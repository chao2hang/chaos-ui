# MobileButton Mobile Mobile Button

> 业务组件

Mobile Mobile Button — 面向 ERP/企业场景的业务组件。

## 何时使用

在手机/窄屏流程中使用，优先触控与单手操作。典型用于「MobileButton」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

## 代码示例

### 基础用法

## API / Props

### MobileButton

| Prop        | Type                   | Default | 说明               |
| ----------- | ---------------------- | ------- | ------------------ |
| `className` | `string`               | —       | 自定义样式类名     |
| `fullWidth` | `boolean`              | `true`  | 是否占满父容器宽度 |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性  |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# Combobox 组合框

> 业务组件

结合输入框和下拉列表的组合选择控件,支持自定义输入和选项匹配。

## 何时使用

当你需要「组合框」相关能力时使用：结合输入框和下拉列表的组合选择控件,支持自定义输入和选项匹配。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

## API / Props

### Combobox

| Prop                | Type                           | Default      | 说明              |
| ------------------- | ------------------------------ | ------------ | ----------------- |
| `className`         | `string`                       | —            | 自定义样式类名    |
| `options`           | `ComboboxOption[]`             | —            | 可选项列表        |
| `value`             | `string`                       | —            | 当前值(受控)      |
| `onChange`          | `() => void`                   | —            | 值变化回调        |
| `placeholder`       | `string`                       | `选择...`    | 占位文本          |
| `searchPlaceholder` | `string`                       | `搜索...`    |                   |
| `emptyText`         | `string`                       | `未找到结果` |                   |
| `disabled`          | `boolean`                      | —            | 是否禁用          |
| `align`             | `"start" \| "center" \| "end"` | `start`      | 对齐方式          |
| `clearable`         | `boolean`                      | `true`       |                   |
| `searchable`        | `boolean`                      | `true`       |                   |
| `renderOption`      | `(...args) => unknown`         | —            |                   |
| ...rest             | `React.HTMLAttributes`         | —            | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

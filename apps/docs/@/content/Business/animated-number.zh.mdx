# AnimatedNumber 动画数字

> 业务组件

动画数字 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要animated-number时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { NumberTicker as AnimatedNumber } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <AnimatedNumber />
}`} lang="tsx" />

## API / Props

### AnimatedNumber

| Prop        | Type                   | Default | 说明                    |
| ----------- | ---------------------- | ------- | ----------------------- |
| `className` | `string`               | —       | 自定义样式类名          |
| `value`     | `number`               | —       | 当前值(受控)            |
| `duration`  | `number`               | `1000`  | 持续时间/展示时长(毫秒) |
| `decimals`  | `number`               | `0`     |                         |
| `prefix`    | `string`               | ``      | 前缀内容                |
| `suffix`    | `string`               | ``      | 后缀内容                |
| `format`    | `(...args) => unknown` | —       |                         |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性       |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

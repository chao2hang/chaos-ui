# Dock 停靠栏

> 业务组件

停靠栏 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要dock时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock
  code={`export function Basic() {
  return <Dock />
}`}
  lang="tsx"
/>

## API / Props

### Dock

| Prop            | Type                          | Default      | 说明                  |
| --------------- | ----------------------------- | ------------ | --------------------- |
| `className`     | `string`                      | —            | 自定义样式类名        |
| `items`         | `DockItem[]`                  | —            | 选项/数据项列表       |
| `orientation`   | `"horizontal" \| "vertical"`  | `horizontal` | 排列方向              |
| `magnification` | `boolean`                     | `true`       |                       |
| ...rest         | `React.ComponentProps<"div">` | —            | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

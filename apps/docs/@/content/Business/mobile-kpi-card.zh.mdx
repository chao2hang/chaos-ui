# MobileKpiCard 移动端 KPI 卡片

> 业务组件

移动端 KPI 卡片 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要移动端 KPI 卡片功能时使用本组件。

### 基础用法

## API / Props

### MobileKPICard

| Prop         | Type                                    | Default   | 说明              |
| ------------ | --------------------------------------- | --------- | ----------------- |
| `className`  | `string`                                | —         | 自定义样式类名    |
| `title`      | `string`                                | —         | 标题              |
| `value`      | `string`                                | —         | 当前值(受控)      |
| `change`     | `string`                                | —         |                   |
| `changeType` | `"positive" \| "negative" \| "neutral"` | `neutral` |                   |
| `icon`       | `React.ElementType`                     | —         | 图标元素          |
| ...rest      | `React.HTMLAttributes`                  | —         | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

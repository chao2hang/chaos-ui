## API / Props

### PromotionRuleCard

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `name` | `string` | `未命名促销` | 促销规则名称 |
| `type` | `"满减" \| "折扣" \| "买赠" \| "直降"` | `折扣` | 促销类型，如 满减 |
| `discount` | `number` | `0` | 折扣力度：满减/直降为减免金额，折扣为 0~1 的比例 |
| `threshold` | `number` | — | 满减门槛金额 |
| `startDate` | `string` | — | 生效开始日期（ISO 字符串） |
| `endDate` | `string` | — | 生效结束日期（ISO 字符串） |
| `scope` | `string` | `全部商品` | 适用范围描述 |
| `active` | `boolean` | `false` | 是否生效中 |
| `onEdit` | `() => void` | — | 点击编辑回调 |
| `onDuplicate` | `() => void` | — | 复制规则回调 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

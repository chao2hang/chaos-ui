# CodeBlock 代码块

> 业务组件 / Domain-specific business components for ERP scenarios.

代码块 — 面向 ERP/企业场景的业务组件。
CodeBlock — a business component for ERP / enterprise scenarios.

## 何时使用 / When to Use

需要代码块时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use CodeBlock when you need: CodeBlock: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

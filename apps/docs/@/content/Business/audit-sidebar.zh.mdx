# AuditSidebar 审计侧栏

> 业务组件

审计侧栏 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要审计侧栏功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { AuditSidebar } from "@chaos_team/chaos-ui/business"

  return <AuditSidebar />
}`} lang="tsx" />

## API / Props

### AuditSidebar

| Prop           | Type                          | Default    | 说明                  |
| -------------- | ----------------------------- | ---------- | --------------------- |
| `className`    | `string`                      | —          | 自定义样式类名        |
| `entries`      | `AuditSidebarEntry[]`         | —          |                       |
| `open`         | `boolean`                     | —          | 是否打开(受控)        |
| `onOpenChange` | `() => void`                  | —          | 打开状态变化回调      |
| `title`        | `string`                      | `操作记录` | 标题                  |
| ...rest        | `React.ComponentProps<"div">` | —          | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

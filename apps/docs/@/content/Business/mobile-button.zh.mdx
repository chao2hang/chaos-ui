# MobileButton 移动端按钮

> 业务组件

移动端按钮 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要移动端按钮功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { MobileButton } from "@chaos_team/chaos-ui/mobile"

export function Basic() {
  return <MobileButton />
}`} lang="tsx" />

## API / Props

### MobileButton

| Prop        | Type                   | Default | 说明               |
| ----------- | ---------------------- | ------- | ------------------ |
| `className` | `string`               | —       | 自定义样式类名     |
| `fullWidth` | `boolean`              | `true`  | 是否占满父容器宽度 |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性  |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

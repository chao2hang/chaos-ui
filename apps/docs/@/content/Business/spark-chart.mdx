## 何时使用 / When to Use

需要Spark Chart时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use SparkChart when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

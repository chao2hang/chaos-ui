# SegmentedControl 分段控制器

> 业务组件

分段控制器 — 面向 ERP/企业场景的业务组件。

## 何时使用

当你需要「分段控制器」相关能力时使用：分段控制器 — Chaos UI 的数据展示。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

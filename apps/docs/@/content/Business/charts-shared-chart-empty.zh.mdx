# ChartEmpty 图表空状态

> 业务组件

图表空状态 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要charts-shared-chart-empty时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

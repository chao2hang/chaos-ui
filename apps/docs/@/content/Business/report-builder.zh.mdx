# ReportBuilder 报表构建器

> 业务组件

报表构建器 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要报表构建器功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { ReportBuilder } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <ReportBuilder />
}`} lang="tsx" />

## API / Props

### ReportBuilder

| Prop              | Type                          | Default | 说明                  |
| ----------------- | ----------------------------- | ------- | --------------------- |
| `className`       | `string`                      | —       | 自定义样式类名        |
| `fields`          | `ReportField[]`               | —       |                       |
| `filters`         | `ReportFilter[]`              | —       |                       |
| `onFieldsChange`  | `() => void`                  | —       |                       |
| `onFiltersChange` | `() => void`                  | —       |                       |
| `onPreview`       | `() => void`                  | —       |                       |
| `onExport`        | `() => void`                  | —       |                       |
| `availableFields` | `string[]`                    | —       |                       |
| ...rest           | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

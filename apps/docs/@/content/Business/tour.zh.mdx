# Tour 新手引导

> 业务组件

新手引导 — 面向 ERP/企业场景的业务组件。

## 何时使用

当你需要「新手引导」相关能力时使用：逐步引导式产品导览覆盖层，带有聚焦高亮、导航和 localStorage 持久化。作为页面中的通用基础能力使用；优先复用而非在业务里再造一套。

## API / Props

### Tour

| Prop           | Type                   | Default | 说明              |
| -------------- | ---------------------- | ------- | ----------------- |
| `className`    | `string`               | —       | 自定义样式类名    |
| `steps`        | `TourStep[]`           | —       |                   |

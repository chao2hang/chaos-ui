# ChartCard Chart Card

> 业务组件 / Domain-specific business components for ERP scenarios.

ChartCard — a card wrapper for chart visualizations with title and description.
ChartCard — chart card component.

## 何时使用 / When to Use

需要Chart Card时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use ChartCard when this capability is required. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

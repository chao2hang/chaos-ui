## API / Props

### ConfirmDialog

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `open` | `boolean` | — | 是否打开(受控) <br/> Open state (controlled) |
| `onOpenChange` | `() => void` | — | 打开状态变化回调 <br/> Callback when open state changes |
| `title` | `string` | `确认操作` | 标题 <br/> Title |
| `description` | `React.ReactNode` | `此操作不可撤销，是否继续？` | 描述文本 <br/> Description text |
| `confirmText` | `string` | `确认` |  |
| `cancelText` | `string` | `取消` |  |
| `variant` | `"default" \| "destructive"` | `default` | 样式变体 <br/> Style variant |
| `onConfirm` | `() => void` | — |  |
| `loading` | `boolean` | — | 是否加载中 <br/> Loading state |
| `icon` | `React.ReactNode` | — | 图标元素 <br/> Icon element |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(ConfirmDialogContainer),使用组合模式按需导入。

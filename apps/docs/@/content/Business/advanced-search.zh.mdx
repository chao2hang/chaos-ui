# AdvancedSearch 高级搜索

> 业务组件

高级搜索 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要高级搜索时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

  return <AdvancedSearch />

## API / Props

### AdvancedSearch

| Prop        | Type                          | Default | 说明                  |
| ----------- | ----------------------------- | ------- | --------------------- |
| `className` | `string`                      | —       | 自定义样式类名        |
| `fields`    | `SearchField[]`               | —       |                       |
| `onSearch`  | `() => void`                  | —       |                       |
| `onReset`   | `() => void`                  | —       |                       |
| `loading`   | `boolean`                     | `false` | 是否加载中            |
| `compact`   | `boolean`                     | `false` |                       |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

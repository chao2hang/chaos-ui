# PrintButton 打印按钮

> 业务组件

打印按钮 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要打印按钮功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { PrintButton } from "@chaos_team/chaos-ui/business"

  return <PrintButton />

## API / Props

### PrintButton

| Prop        | Type                             | Default | 说明                     |
| ----------- | -------------------------------- | ------- | ------------------------ |
| `className` | `string`                         | —       | 自定义样式类名           |
| `onClick`   | `() => void`                     | —       | 点击回调                 |
| `label`     | `string`                         | `打印`  | 标签文本                 |
| ...rest     | `React.ComponentProps<"button">` | —       | 透传 `<button>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<button>` 原生 HTML 属性。

# ErrorBoundary 错误边界

> 业务组件

错误边界 — 面向 ERP/企业场景的业务组件。

## 何时使用

需要错误边界时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

## 代码示例

### 基础用法

<CodeBlock code={`import { ErrorBoundary } from "@chaos_team/chaos-ui/business"

export function Basic() {
  return <ErrorBoundary />
}`} lang="tsx" />

## API / Props

### ErrorBoundary

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `fallback`  | `(...args) => unknown` | —       |                   |
| `onError`   | `() => void`           | —       |                   |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

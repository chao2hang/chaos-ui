## API / Props

### LabelDesigner

| Prop             | Type                   | Default | 说明                    |
| ---------------- | ---------------------- | ------- | ----------------------- |
| `className`      | `string`               | —       | 自定义样式类名          |
| `fields`         | `LabelField[]`         | —       | Label fields.           |
| `onFieldsChange` | `() => void`           | —       | On fields change.       |
| `labelSize`      | `LabelSize`            | —       | Label size preset.      |
| `sizePresets`    | `LabelSize[]`          | —       | Available size presets. |
| `onSizeChange`   | `() => void`           | —       | On size change.         |
| `onPrint`        | `() => void`           | —       | Print handler.          |
| `readOnly`       | `boolean`              | `false` | Read-only mode.         |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性       |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

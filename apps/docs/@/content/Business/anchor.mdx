# Anchor 锚点

> 业务组件 / Domain-specific business components for ERP scenarios.

跳转到页面指定位置的锚点导航,常用于目录。
Jumps to specified positions on a page; commonly used for table of contents.

## 何时使用 / When to Use

需要锚点功能时使用本组件。

Use when you need an anchor.

## API / Props

### TodoListTable

| Prop        | Type                   | Default | 说明                                                                     |
| ----------- | ---------------------- | ------- | ------------------------------------------------------------------------ |
| `className` | `string`               | —       | 自定义样式类名                                                           |
| `items`     | `TodoItem[]`           | —       | Todo items. Defaults to a small sample list so the table is never empty. |
| `onToggle`  | `() => void`           | —       | Called with the toggled item id whenever a checkbox is clicked.          |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                                        |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# Combobox

> Domain-specific business components for ERP scenarios.

Combines input with a dropdown list; supports custom input and option matching.

## When to Use

Use Combobox when you need: Combines input with a dropdown list; supports custom input and option matching. Use to collect or edit data with validation and submit flows.

## API / Props

### Combobox

| Prop                | Type                           | Default      | Description                       |
| ------------------- | ------------------------------ | ------------ | --------------------------------- |
| `className`         | `string`                       | —            | Custom CSS class                  |
| `options`           | `ComboboxOption[]`             | —            | Available options                 |
| `value`             | `string`                       | —            | Current value (controlled)        |
| `onChange`          | `() => void`                   | —            | Callback when value changes       |
| `placeholder`       | `string`                       | `选择...`    | Placeholder text                  |
| `searchPlaceholder` | `string`                       | `搜索...`    |                                   |
| `emptyText`         | `string`                       | `未找到结果` |                                   |
| `disabled`          | `boolean`                      | —            | Whether disabled                  |
| `align`             | `"start" \| "center" \| "end"` | `start`      | Alignment                         |
| `clearable`         | `boolean`                      | `true`       |                                   |
| `searchable`        | `boolean`                      | `true`       |                                   |
| `renderOption`      | `(...args) => unknown`         | —            |                                   |
| ...rest             | `React.HTMLAttributes`         | —            | Inherits standard HTML attributes |

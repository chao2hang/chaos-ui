## API / Props

### MobileCheckout

| Prop        | Type                   | Default   | 说明                            |
| ----------- | ---------------------- | --------- | ------------------------------- |
| `className` | `string`               | —         | 自定义样式类名                  |
| `total`     | `string`               | —         | Total amount string.            |
| `itemCount` | `number`               | —         | Item count.                     |
| `payLabel`  | `string`               | `Pay Now` | Pay button label.               |
| `isLoading` | `boolean`              | —         | Whether payment is in progress. |
| `onPay`     | `() => void`           | —         | Called when user taps pay.      |
| `onBack`    | `() => void`           | —         | Called when user taps back.     |
| ...rest     | `React.HTMLAttributes` | —         | 透传原生 DOM 属性               |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# ConfirmDialog

> Domain-specific business components for ERP scenarios.

ConfirmDialog — a business component for ERP / enterprise scenarios.

## When to Use

Use when you need a confirm dialog.

## Examples

### Basic Usage

<CodeBlock code={`import { ConfirmDialog, ConfirmDialogContainer } from "@/components/business/confirm-dialog"

export function Basic() {
  return (
    <ConfirmDialog>
      <ConfirmDialogContainer>Content</ConfirmDialogContainer>
    </ConfirmDialog>
  )
}`} lang="tsx" />

## API / Props

### ConfirmDialog

| Prop           | Type                         | Default                      | Description                       |
| -------------- | ---------------------------- | ---------------------------- | --------------------------------- |
| `className`    | `string`                     | —                            | Custom CSS class                  |
| `open`         | `boolean`                    | —                            | Open state (controlled)           |
| `onOpenChange` | `() => void`                 | —                            | Callback when open state changes  |
| `title`        | `string`                     | `确认操作`                   | Title                             |
| `description`  | `React.ReactNode`            | `此操作不可撤销，是否继续？` | Description text                  |
| `confirmText`  | `string`                     | `确认`                       |                                   |
| `cancelText`   | `string`                     | `取消`                       |                                   |
| `variant`      | `"default" \| "destructive"` | `default`                    | Style variant                     |
| `onConfirm`    | `() => void`                 | —                            |                                   |
| `loading`      | `boolean`                    | —                            | Loading state                     |
| `icon`         | `React.ReactNode`            | —                            | Icon element                      |
| ...rest        | `React.HTMLAttributes`       | —                            | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(ConfirmDialogContainer),使用组合模式按需导入。

## API / Props

### LeadPipelineBoard

| Prop             | Type                   | Default | 说明                                             |
| ---------------- | ---------------------- | ------- | ------------------------------------------------ |
| `className`      | `string`               | —       | 自定义样式类名                                   |
| `stages`         | `PipelineStage[]`      | —       | Stage definitions in funnel order.               |
| `deals`          | `Deal[]`               | —       | All deals.                                       |
| `onDealMove`     | `() => void`           | —       | Callback when a deal moves to a different stage. |
| `onDealClick`    | `() => void`           | —       | Deal click handler.                              |
| `currencySymbol` | `string`               | `¥`     | Currency symbol for display.                     |
| `readOnly`       | `boolean`              | `false` | Read-only mode disables drag.                    |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

## API / Props

### ChartExportButton

| Prop | Type | Default | 说明 |
| ----------- | -------------------------------------- | ------- | --------------------------------------------------------- |
| `className` | `string` | — | 自定义样式类名 |
| `target` | `React.RefObject<HTMLElement \| null>` | — | 链接打开目标 |
| `filename`  | `string`                               | `chart` |                                                           |
| `format`    | `"png" \| "svg"`                       | `png`   |                                                           |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

# FormDesignerRuntime Form Designer Runtime

> 业务组件

<FormDesignerRuntime
  schema={[{ id: "name", type: "text", label: "姓名", required: true }]}
  value={{ name: "张三" }}
/>

## 何时使用

需要采集或编辑业务字段，并参与校验/提交时使用。典型用于「Form Designer Runtime」场景。面向 ERP/业务域（单据、选择器、审批、结算等），按业务语义接入。

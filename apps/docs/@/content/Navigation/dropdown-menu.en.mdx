# DropdownMenu

> Navigation components for moving through the app.

Menu list that pops downward, holding actions or navigation options.

## When to Use

A dropdown listing action commands triggered by a click.


## Examples

### Basic Usage

<CodeBlock code={`import { DropdownMenu, DropdownMenuPortal, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuGroup, DropdownMenuLabel } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <DropdownMenu>
      <DropdownMenuContent>Content</DropdownMenuContent>
    </DropdownMenu>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 14 个子组件(DropdownMenuPortal, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuGroup, DropdownMenuLabel…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

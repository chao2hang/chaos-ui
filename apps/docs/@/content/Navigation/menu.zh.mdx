# Menu 导航菜单

> 导航组件

页面层级导航菜单,支持横向/纵向、子菜单、图标。

## 何时使用

当你需要「导航菜单」相关能力时使用：页面层级导航菜单,支持横向/纵向、子菜单、图标。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

## 样式变体

**方向**

| direction    | 说明 | 默认 |
| ------------ | ---- | ---- |
| `vertical`   |      | ✓    |
| `horizontal` |      |      |

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |
| `sm`      | 小号     |      |
| `lg`      | 大号     |      |

**active**

| active  | 说明 | 默认 |
| ------- | ---- | ---- |
| `true`  |      | ✓    |
| `false` |      |      |

**变体**

| Variant       | 说明          | 默认 |
| ------------- | ------------- | ---- |
| `default`     | 默认样式      | ✓    |
| `destructive` | 危险/删除样式 |      |

## API / Props

### Menu

| Prop           | Type                          | Default  | 说明                                                                             |
| -------------- | ----------------------------- | -------- | -------------------------------------------------------------------------------- |
| `className`    | `string`                      | —        | 自定义样式类名                                                                   |
| `items`        | `MenuItem[]`                  | —        | 选项/数据项列表                                                                  |
| `onSelect`     | `() => void`                  | —        | 选中回调                                                                         |
| `selectedKeys` | `string[]`                    | —        |                                                                                  |
| `mode`         | `"inline" \| "popup"`         | `inline` | Mode: "inline" shows children indented, "popup" shows them in a submenu on hover |
| ...rest        | `React.ComponentProps<"div">` | —        | 透传 `<div>` 原生属性                                                            |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

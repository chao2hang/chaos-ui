# Command 命令面板

> 导航组件 / Navigation components for moving through the app.

类似 VS Code Ctrl+K 的命令面板,键盘驱动搜索/执行。
VS Code-style Ctrl+K command palette driven by keyboard search.

## 何时使用 / When to Use

命令面板/快捷键搜索,通过输入快速定位命令或内容。

A command palette for quick search and navigation via keyboard input.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Command, CommandDialog, CommandInput, CommandList, CommandEmpty, CommandGroup } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Command>
      <CommandDialog>Content</CommandDialog>
    </Command>
  )
}`} lang="tsx" />

## API / Props

### Command

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 8 个子组件(CommandDialog, CommandInput, CommandList, CommandEmpty, CommandGroup…),使用组合模式按需导入。

# DropdownMenu 下拉菜单

> 导航组件

向下弹出的菜单列表,承载操作或导航选项。

## 何时使用

点击触发弹出的下拉菜单,收纳一组操作命令。


## 代码示例

### 基础用法

<CodeBlock code={`import { DropdownMenu, DropdownMenuPortal, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuGroup, DropdownMenuLabel } from "@/components/ui/dropdown-menu"

export function Basic() {
  return (
    <DropdownMenu>
      <DropdownMenuContent>Content</DropdownMenuContent>
    </DropdownMenu>
  )
}`} lang="tsx" />

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 14 个子组件(DropdownMenuPortal, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuGroup, DropdownMenuLabel…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

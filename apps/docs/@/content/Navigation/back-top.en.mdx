# BackTop

> Navigation components for moving through the app.

Button pinned to bottom-right that scrolls to the top.

## When to Use

Use BackTop when you need: Button pinned to bottom-right that scrolls to the top. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

## Style Variants

**Variant**

| Variant   | Description                 | 默认 |
| --------- | --------------------------- | ---- |
| `default` | Default style               | ✓    |
| `outline` | Outlined style              |      |
| `ghost`   | Ghost (no background) style |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

## API / Props

### BackTop

| Prop               | Type                             | Default | Description                        |
| ------------------ | -------------------------------- | ------- | ---------------------------------- |
| `className`        | `string`                         | —       | Custom CSS class                   |
| `target`           | `(...args) => unknown`           | —       | Scroll container target            |
| `visibilityHeight` | `number`                         | `400`   | Threshold in px before showing     |
| ...rest            | `React.ComponentProps<"button">` | —       | Inherits all `<button>` attributes |

## Notes

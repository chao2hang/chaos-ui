# Dot 状态点

> 导航组件 / Navigation components for moving through the app.

小圆点状态指示器,常用于在线状态、消息提醒、列表项前。
Small dot status indicator used for online status, notification dots, or list items.

## 何时使用 / When to Use

当你需要「状态点」相关能力时使用：小圆点状态指示器,常用于在线状态、消息提醒、列表项前。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use Dot when you need: Small dot status indicator used for online status, notification dots, or list items. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

## 样式变体 / Style Variants

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `sm`      | 小号 / Small            |      |
| `default` | 默认尺寸 / Default size | ✓    |
| `lg`      | 大号 / Large            |      |

**变体 / Variant**

| Variant       | 说明 / Description                       | 默认 |
| ------------- | ---------------------------------------- | ---- |
| `default`     | 默认样式 / Default style                 | ✓    |
| `primary`     | 主要样式 / Primary style                 |      |
| `success`     | 成功状态 / Success style                 |      |
| `warning`     | 警告状态 / Warning style                 |      |
| `destructive` | 危险/删除样式 / Destructive/danger style |      |
| `info`        | 信息提示 / Informational style           |      |

**pulse / pulse**

| pulse   | 说明 / Description | 默认 |
| ------- | ------------------ | ---- |
| `true`  |                    | ✓    |
| `false` |                    |      |

# Anchor 锚点

> 导航组件 / Navigation components for moving through the app.

跳转到页面指定位置的锚点导航,常用于目录。
Jumps to specified positions on a page; commonly used for table of contents.

## 何时使用 / When to Use

需要锚点功能时使用本组件。

Use when you need an anchor.

## API / Props

### Anchor

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `links` | `AnchorLink[]` | — |  |
| `offset` | `number` | `0` | Offset for active link detection |
| `replace` | `boolean` | — | Replace history or push |
| ...rest | `React.ComponentProps<"nav">` | — | 透传 `<nav>` 原生属性 <br/> Inherits all `<nav>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<nav>` 原生 HTML 属性。

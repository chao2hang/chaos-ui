# Menubar 菜单栏

> 导航组件 / Navigation components for moving through the app.

水平菜单栏,通常提供多组下拉菜单。
Horizontal menu bar providing grouped dropdown menus.

## 何时使用 / When to Use

当你需要「菜单栏」相关能力时使用：水平菜单栏,通常提供多组下拉菜单。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use Menubar when you need: Horizontal menu bar providing grouped dropdown menus. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

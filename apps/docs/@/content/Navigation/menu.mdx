# Menu 导航菜单

> 导航组件 / Navigation components for moving through the app.

页面层级导航菜单,支持横向/纵向、子菜单、图标。
Hierarchical navigation menu supporting horizontal/vertical orientations, submenus, and icons.

## 何时使用 / When to Use

需要导航菜单功能时使用本组件。

Use when you need a menu.

## API / Props

### Menu

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `items` | `MenuItem[]` | — | 选项/数据项列表 <br/> Items / data list |
| `onSelect` | `() => void` | — | 选中回调 <br/> Selection handler |
| `selectedKeys` | `string[]` | — |  |
| `mode` | `"inline" \| "popup"` | `inline` | Mode: "inline" shows children indented, "popup" shows them in a submenu on hover |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

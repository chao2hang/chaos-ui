# NavigationMenu 导航菜单

> 导航组件 / Navigation components for moving through the app.

导航菜单 — Chaos UI 的导航组件。
NavigationMenu — a navigation components for moving through the app. from Chaos UI.

## 何时使用 / When to Use

需要导航菜单时使用本组件。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use NavigationMenu when you need: NavigationMenu — a navigation components for moving through the app. from Chaos UI. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

## 注意事项 / Notes

- **复合组件**: 包含 7 个子组件(NavigationMenuContent, NavigationMenuIndicator, NavigationMenuItem, NavigationMenuLink, NavigationMenuList…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

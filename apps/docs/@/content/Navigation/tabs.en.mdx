# Tabs

> Navigation components for moving through the app.

Switch between sibling content views on the same page.

## When to Use

Switch between multiple sibling content panels while preserving context.

## Examples

### Basic Usage

<CodeBlock code={`import { Tabs, TabsList, TabsTrigger, TabsContent } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Tabs defaultValue="tab1">
      <TabsList>
        <TabsTrigger value="tab1">Tab 1</TabsTrigger>
        <TabsTrigger value="tab2">Tab 2</TabsTrigger>
      </TabsList>
      <TabsContent value="tab1">Content 1</TabsContent>
      <TabsContent value="tab2">Content 2</TabsContent>
    </Tabs>
  )
}`} lang="tsx" />

## Style Variants

**Variant**

| Variant   | Description   | 默认 |
| --------- | ------------- | ---- |
| `default` | Default style | ✓    |
| `line`    |               |      |

## API / Props

### Tabs

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 3 个子组件(TabsList, TabsTrigger, TabsContent),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

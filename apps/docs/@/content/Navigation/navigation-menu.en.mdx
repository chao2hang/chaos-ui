# NavigationMenu

> Navigation components for moving through the app.

NavigationMenu — a navigation components for moving through the app. from Chaos UI.

## When to Use

Use when you need a navigation menu.


## Examples

### Basic Usage

<CodeBlock code={`import { NavigationMenu, NavigationMenuContent, NavigationMenuIndicator, NavigationMenuItem, NavigationMenuLink, NavigationMenuList } from "@/components/ui/navigation-menu"

export function Basic() {
  return (
    <NavigationMenu>
      <NavigationMenuContent>Content</NavigationMenuContent>
    </NavigationMenu>
  )
}`} lang="tsx" />

## Notes

- **复合组件**: 包含 7 个子组件(NavigationMenuContent, NavigationMenuIndicator, NavigationMenuItem, NavigationMenuLink, NavigationMenuList…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 8 个子组件(CommandDialog, CommandInput, CommandList, CommandEmpty, CommandGroup…),使用组合模式按需导入。

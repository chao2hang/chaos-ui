# Pagination Ui Pagination

> 导航组件 / Navigation components for moving through the app.

Ui Pagination — Chaos UI 的导航组件。
Pagination — a navigation components for moving through the app. from Chaos UI.

## 何时使用 / When to Use

适用于Ui Pagination相关场景。

Use when you need a pagination.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Pagination, PaginationContent, PaginationEllipsis, PaginationItem, PaginationLink, PaginationNext } from "@/components/ui/pagination"

export function Basic() {
  return (
    <Pagination>
      <PaginationContent>Content</PaginationContent>
    </Pagination>
  )
}`} lang="tsx" />

## API / Props

### Pagination

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **复合组件**: 包含 6 个子组件(PaginationContent, PaginationEllipsis, PaginationItem, PaginationLink, PaginationNext…),使用组合模式按需导入。

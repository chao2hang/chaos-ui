# Anchor

> Navigation components for moving through the app.

Jumps to specified positions on a page; commonly used for table of contents.

## When to Use

Use Anchor when you need: Jumps to specified positions on a page; commonly used for table of contents. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

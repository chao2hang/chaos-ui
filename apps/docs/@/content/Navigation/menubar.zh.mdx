# Menubar 菜单栏

> 导航组件

水平菜单栏,通常提供多组下拉菜单。

## 何时使用

当你需要「菜单栏」相关能力时使用：水平菜单栏,通常提供多组下拉菜单。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

## 代码示例

### 基础用法

<CodeBlock code={`import { Menubar } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Menubar />
}`} lang="tsx" />

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

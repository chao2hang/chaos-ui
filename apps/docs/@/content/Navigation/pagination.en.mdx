# Pagination

> Navigation components for moving through the app.

Splits long data sets across pages when rendering all at once would be too heavy.

## When to Use

Split large datasets into pages to avoid rendering too much at once.


## Examples

### Basic Usage

<CodeBlock code={`import { Pagination, PaginationContent, PaginationEllipsis, PaginationItem, PaginationLink, PaginationNext } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Pagination>
      <PaginationContent>Content</PaginationContent>
    </Pagination>
  )
}`} lang="tsx" />


## API / Props

### Pagination

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

## Notes

- **复合组件**: 包含 6 个子组件(PaginationContent, PaginationEllipsis, PaginationItem, PaginationLink, PaginationNext…),使用组合模式按需导入。

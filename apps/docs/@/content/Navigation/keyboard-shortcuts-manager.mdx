# KeyboardShortcutsManager 快捷键管理器

> 导航组件 / Navigation components for page routing and orientation.

快捷键管理器 — 全局快捷键管理器，支持注册与冲突检测。
KeyboardShortcutsManager — Global keyboard shortcuts manager with registration and conflict detection.

## 何时使用 / When to Use

当你需要「快捷键管理器」相关能力时使用：全局快捷键管理器，支持注册与冲突检测。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use KeyboardShortcutsManager when you need: Global keyboard shortcuts manager with registration and conflict detection. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

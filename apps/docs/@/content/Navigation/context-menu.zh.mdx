# ContextMenu 右键菜单

> 导航组件

右键触发的上下文操作菜单。

## 何时使用

需要右键菜单功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { ContextMenu, ContextMenuCheckboxItem, ContextMenuContent, ContextMenuGroup, ContextMenuItem, ContextMenuLabel } from "@/components/ui/context-menu"

export function Basic() {
  return (
    <ContextMenu>
      <ContextMenuContent>Content</ContextMenuContent>
    </ContextMenu>
  )
}`} lang="tsx" />

## API / Props

### ContextMenu

| Prop        | Type                   | Default | 说明              |

| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 13 个子组件(ContextMenuCheckboxItem, ContextMenuContent, ContextMenuGroup, ContextMenuItem, ContextMenuLabel…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# Stepper 步骤条

> 导航组件 / Navigation components for moving through the app.

水平步骤条,引导用户完成多步骤流程。
Horizontal stepper guiding users through a multi-step workflow.

## 何时使用 / When to Use

当你需要「步骤条」相关能力时使用：水平步骤条,引导用户完成多步骤流程。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use Stepper when you need: Horizontal stepper guiding users through a multi-step workflow. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

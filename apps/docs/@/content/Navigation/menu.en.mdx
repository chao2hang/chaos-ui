# Menu

> Navigation components for moving through the app.

Hierarchical navigation menu supporting horizontal/vertical orientations, submenus, and icons.

## When to Use

Use when you need a menu.

## Examples

## Style Variants

**direction**

| direction    | Description | 默认 |
| ------------ | ----------- | ---- |
| `vertical`   |             | ✓    |
| `horizontal` |             |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

**active**

| active  | Description | 默认 |
| ------- | ----------- | ---- |
| `true`  |             | ✓    |
| `false` |             |      |

**Variant**

| Variant       | Description              | 默认 |
| ------------- | ------------------------ | ---- |
| `default`     | Default style            | ✓    |
| `destructive` | Destructive/danger style |      |

## API / Props

### Menu

| Prop           | Type                          | Default  | Description                                                                      |
| -------------- | ----------------------------- | -------- | -------------------------------------------------------------------------------- |
| `className`    | `string`                      | —        | Custom CSS class                                                                 |
| `items`        | `MenuItem[]`                  | —        | Items / data list                                                                |
| `onSelect`     | `() => void`                  | —        | Selection handler                                                                |
| `selectedKeys` | `string[]`                    | —        |                                                                                  |
| `mode`         | `"inline" \| "popup"`         | `inline` | Mode: "inline" shows children indented, "popup" shows them in a submenu on hover |
| ...rest        | `React.ComponentProps<"div">` | —        | Inherits all `<div>` attributes                                                  |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

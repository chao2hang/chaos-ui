# BackTop 回到顶部

> 导航组件 / Navigation components for moving through the app.

页面右下角按钮,一键返回页面顶部。
Button pinned to bottom-right that scrolls to the top.

## 何时使用 / When to Use

当你需要「回到顶部」相关能力时使用：页面右下角按钮,一键返回页面顶部。用于页面/模块间跳转与定位；保持信息层级清晰，避免同页过多导航入口。

Use BackTop when you need: Button pinned to bottom-right that scrolls to the top. Use for moving between pages/sections; keep hierarchy clear and avoid navigation clutter.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant   | 说明 / Description                             | 默认 |
| --------- | ---------------------------------------------- | ---- |
| `default` | 默认样式 / Default style                       | ✓    |
| `outline` | 描边样式 / Outlined style                      |      |
| `ghost`   | 幽灵(无背景)样式 / Ghost (no background) style |      |

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |

## API / Props

### BackTop

| Prop               | Type                             | Default | 说明 / Description                                                |
| ------------------ | -------------------------------- | ------- | ----------------------------------------------------------------- |
| `className`        | `string`                         | —       | 自定义样式类名 <br/> Custom CSS class                             |
| `target`           | `(...args) => unknown`           | —       | Scroll container target                                           |
| `visibilityHeight` | `number`                         | `400`   | Threshold in px before showing                                    |
| ...rest            | `React.ComponentProps<"button">` | —       | 透传 `<button>` 原生属性 <br/> Inherits all `<button>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<button>` 原生 HTML 属性。

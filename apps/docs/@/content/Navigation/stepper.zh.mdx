# Stepper 步骤条

> 导航组件

水平步骤条,引导用户完成多步骤流程。

## 何时使用

需要步骤条功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { Stepper } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Stepper>Default</Stepper>
}`} lang="tsx" />

## API / Props

### Stepper

| Prop          | Type                         | Default      | 说明              |
| ------------- | ---------------------------- | ------------ | ----------------- |
| `className`   | `string`                     | —            | 自定义样式类名    |
| `activeStep`  | `number`                     | `0`          |                   |
| `orientation` | `"horizontal" \| "vertical"` | `horizontal` | 排列方向          |
| ...rest       | `React.HTMLAttributes`       | —            | 透传原生 DOM 属性 |

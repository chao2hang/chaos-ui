# Fab

> Navigation components for moving through the app.

Circular floating action button (bottom-right) for the primary create/add action.

## When to Use

Use when you need a fab.


## Examples

### Basic Usage

<CodeBlock code={`import { Fab } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Fab />
}`} lang="tsx" />

## Style Variants

**Variant**

| Variant | Description | 默认 |
|---|---|---|
| `default` | Default style | ✓ |
| `secondary` | Secondary action style |  |
| `destructive` | Destructive/danger style |  |
| `outline` | Outlined style |  |
| `ghost` | Ghost (no background) style |  |

**Size**

| Size | Description | 默认 |
|---|---|---|
| `default` | Default size | ✓ |
| `sm` | Small |  |
| `lg` | Large |  |

**position**

| position | Description | 默认 |
|---|---|---|
| `bottom-right` |  | ✓ |
| `bottom-left` |  |  |
| `top-right` |  |  |
| `top-left` |  |  |
| `none` |  |  |

## API / Props

### Fab

| Prop | Type | Default | Description |

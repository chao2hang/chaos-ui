# Fab 悬浮按钮

> 导航组件 / Navigation components for moving through the app.

悬浮在页面右下角的圆形操作按钮,用于最主要的创建/新增操作。
Circular floating action button (bottom-right) for the primary create/add action.

## 何时使用 / When to Use

需要悬浮按钮功能时使用本组件。

Use when you need a fab.

## API / Props

### Fab

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| ...rest | `React.ComponentProps<"button">` | — | 透传 `<button>` 原生属性 <br/> Inherits all `<button>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<button>` 原生 HTML 属性。

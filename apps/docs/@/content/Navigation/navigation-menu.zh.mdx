# NavigationMenu 导航菜单

> 导航组件

导航菜单 — Chaos UI 的导航组件。

## 何时使用

需要导航菜单功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { NavigationMenu, NavigationMenuContent, NavigationMenuIndicator, NavigationMenuItem, NavigationMenuLink, NavigationMenuList } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <NavigationMenu>
      <NavigationMenuContent>Content</NavigationMenuContent>
    </NavigationMenu>
  )
}`} lang="tsx" />

## API / Props

### NavigationMenu

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **复合组件**: 包含 7 个子组件(NavigationMenuContent, NavigationMenuIndicator, NavigationMenuItem, NavigationMenuLink, NavigationMenuList…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# Dot

> Navigation components for moving through the app.

Small dot status indicator used for online status, notification dots, or list items.

## When to Use

Use when you need a dot.


## Examples

### Basic Usage

<CodeBlock code={`import { Dot } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Dot>Default</Dot>
}`} lang="tsx" />

### Variants

<CodeBlock code={`{/* 展示 variant 变体 */}
<div className="flex flex-wrap items-center gap-3">
  <Dot variant="default">Default</Dot>
  <Dot variant="primary">Primary</Dot>
  <Dot variant="success">Success</Dot>
  <Dot variant="warning">Warning</Dot>
  <Dot variant="destructive">Destructive</Dot>
</div>`} lang="tsx" />

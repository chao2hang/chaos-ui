## API / Props

### Anchor

| Prop        | Type                          | Default | 说明 / Description                                          |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `links`     | `AnchorLink[]`                | —       |                                                             |
| `offset`    | `number`                      | `0`     | Offset for active link detection                            |
| `replace`   | `boolean`                     | —       | Replace history or push                                     |
| ...rest     | `React.ComponentProps<"nav">` | —       | 透传 `<nav>` 原生属性 <br/> Inherits all `<nav>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<nav>` 原生 HTML 属性。

# Breadcrumb 面包屑

Chaos UI 面包屑导航组件，帮助用户了解当前位置并提供返回上层路径的快捷方式。
Breadcrumb navigation to show current location and provide quick back-navigation links.

## 何时使用 / When to use

当页面层级较深（3 层以上）时，使用面包屑帮助用户定位当前位置并快速返回上级页面。与 Next.js 路由配合良好。

Use when the page hierarchy is deep (3+ levels) to help users orient and quickly navigate upward. Works well with Next.js routing.

## 注意事项

- **渲染语义**：Breadcrumb 渲染为 `<nav aria-label="Breadcrumb">`，BreadcrumbList 渲染为 `<ol>`，符合面包屑导航的语义规范。
- **asChild 模式**：`BreadcrumbLink` 支持 `asChild`，可配合 Next.js `<Link>` 或任意自定义 `<a>` 元素使用，避免 `<a>` 嵌套问题。
- **BreadcrumbPage**：当前页面项使用 `BreadcrumbPage`（`<span>`），不是链接，且视觉上高亮显示。
- **无障碍**：`aria-current="page"` 自动应用于当前页面项。分隔符使用 `aria-hidden="true"` 避免屏幕阅读器重复朗读。

# Result 结果页

> 反馈组件 / Feedback and status communication to the user.

向用户反馈操作结果(成功/失败/警告/404/403),常为整页。
Feedback for an operation result (success/fail/warning/404/403), typically full-page.

## 何时使用 / When to Use

需要结果页功能时使用本组件。

Use when you need a result.

## API / Props

### Result

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `status` | `"success" \| "error" \| "info" \| "wa...` | `info` | Status determines default icon and color |
| `title` | `React.ReactNode` | — | Title text |
| `subtitle` | `React.ReactNode` | — | Subtitle / description |
| `icon` | `React.ReactNode` | — | Custom icon override |
| `extra` | `React.ReactNode` | — | Extra content (e.g., action buttons) |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

## 样式变体 / Style Variants

**变体 / Variant**

| Variant   | 说明 / Description       | 默认 |
| --------- | ------------------------ | ---- |
| `default` | 默认样式 / Default style | ✓    |
| `muted`   |                          |      |
| `white`   |                          |      |

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |
| `xl`      | 超大 / Extra large      |      |

## API / Props

### Spinner

| Prop        | Type                          | Default      | 说明 / Description                                          |
| ----------- | ----------------------------- | ------------ | ----------------------------------------------------------- |
| `className` | `string`                      | —            | 自定义样式类名 <br/> Custom CSS class                       |
| `label`     | `string`                      | `Loading...` | Accessible label                                            |
| ...rest     | `React.ComponentProps<"svg">` | —            | 透传 `<svg>` 原生属性 <br/> Inherits all `<svg>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<svg>` 原生 HTML 属性。

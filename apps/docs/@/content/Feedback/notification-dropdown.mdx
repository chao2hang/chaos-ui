# NotificationDropdown 通知下拉

> 反馈组件 / Feedback components for user interaction and status.

通知下拉 — Header 通知下拉面板，支持已读/未读与全部清除。
NotificationDropdown — Header notification dropdown panel with read/unread and clear all.

## 何时使用 / When to Use

当你需要「通知下拉」相关能力时使用：Header 通知下拉面板，支持已读/未读与全部清除。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use NotificationDropdown when you need: Header notification dropdown panel with read/unread and clear all. Use for loading, confirmation, warnings, and notifications—never fail silently.

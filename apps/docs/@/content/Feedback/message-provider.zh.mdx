# MessageProvider 消息提供者

> 反馈组件

消息提供者 — Chaos UI 的反馈组件。

## 何时使用

需要消息提供者功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { MessageProvider, Message, MessageContext } from "@/components/ui/message-provider"

export function Basic() {
  return (
    <div className="space-y-4">
  <MessageProvider />
  <Message />
  <MessageContext />
    </div>
  )
}`} lang="tsx" />

## API / Props

### MessageProvider

| Prop        | Type                          | Default | 说明                         |
| ----------- | ----------------------------- | ------- | ---------------------------- |
| `className` | `string`                      | —       | 自定义样式类名               |
| `max`       | `number`                      | `5`     | Max messages visible at once |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性        |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# AlertDialog

> Feedback and status communication to the user.

Blocking confirmation dialog for destructive actions, styled for emphasis.

## When to Use

Use AlertDialog when you need: Blocking confirmation dialog for destructive actions, styled for emphasis. Use for loading, confirmation, warnings, and notifications—never fail silently.

## Examples

### Basic Usage

<CodeBlock code={`import { AlertDialog, AlertDialogTrigger, AlertDialogContent, AlertDialogHeader, AlertDialogTitle, AlertDialogDescription } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <AlertDialog>
      <AlertDialogTrigger>Open</AlertDialogTrigger>
      <AlertDialogContent>
      <AlertDialogHeader>
        <AlertDialogTitle>Title</AlertDialogTitle>
        <AlertDialogDescription>Description.</AlertDialogDescription>
      </AlertDialogHeader>
      </AlertDialogContent>
    </AlertDialog>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 8 个子组件(AlertDialogTrigger, AlertDialogContent, AlertDialogHeader, AlertDialogTitle, AlertDialogDescription…),使用组合模式按需导入。

# MessageProvider 消息提供者

> 反馈组件 / Feedback and status communication to the user.

消息提供者 — Chaos UI 的反馈组件。
MessageProvider — a feedback and status communication to the user. from Chaos UI.

## 何时使用 / When to Use

需要消息提供者时使用本组件。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use MessageProvider when you need: MessageProvider — a feedback and status communication to the user. from Chaos UI. Use for loading, confirmation, warnings, and notifications—never fail silently.

## API / Props

### MessageProvider

| Prop        | Type                          | Default | 说明 / Description                                          |

| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

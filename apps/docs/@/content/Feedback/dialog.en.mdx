# Dialog

> Feedback and status communication to the user.

Modal popup for critical user interactions; does not support stacking.

## When to Use

Use a modal when the user must handle a critical task without leaving the current workflow.

## Examples

### Basic Usage

<CodeBlock code={`import { Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Dialog>
      <DialogTrigger>Open</DialogTrigger>
      <DialogContent>
      <DialogHeader>
        <DialogTitle>Title</DialogTitle>
        <DialogDescription>Description.</DialogDescription>
      </DialogHeader>
      </DialogContent>
    </Dialog>
  )
}`} lang="tsx" />

## API / Props

### Dialog

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 9 个子组件(DialogClose, DialogContent, DialogDescription, DialogFooter, DialogHeader…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# Dialog Ui Dialog

> 反馈组件 / Feedback and status communication to the user.

Ui Dialog — Chaos UI 的反馈组件。
Dialog — a feedback and status communication to the user. from Chaos UI.

## 何时使用 / When to Use

适用于Ui Dialog相关场景。

Use when you need a dialog.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription } from "@/components/ui/dialog"

export function Basic() {
  return (
    <Dialog>
      <DialogTrigger>Open</DialogTrigger>
      <DialogContent>
      <DialogHeader>
        <DialogTitle>Title</DialogTitle>
        <DialogDescription>Description.</DialogDescription>
      </DialogHeader>
      </DialogContent>
    </Dialog>
  )
}`} lang="tsx" />

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 9 个子组件(DialogClose, DialogContent, DialogDescription, DialogFooter, DialogHeader…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

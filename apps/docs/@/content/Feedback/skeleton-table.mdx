当你需要「表格骨架屏」相关能力时使用：表格骨架屏占位，按列精确占位。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use SkeletonTable when you need: Table skeleton placeholder with column-aware shimmer. Use for loading, confirmation, warnings, and notifications—never fail silently.

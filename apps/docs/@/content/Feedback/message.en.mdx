# Message

> Feedback and status communication to the user.

Centered top feedback overlay that auto-dismisses.

## When to Use

Use when you need a message.

## Examples

### Basic Usage

<CodeBlock code={`import { Message } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Message />
}`} lang="tsx" />

## API / Props

### Message

| Prop        | Type                          | Default | Description                                           |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------- |
| `className` | `string`                      | —       | Custom CSS class                                      |
| `variant`   | `MessageVariant`              | `info`  | Message variant / 消息变体                            |
| `icon`      | `React.ReactNode`             | —       | Custom icon (overrides variant default) / 自定义图标  |
| `action`    | `React.ReactNode`             | —       | Optional action element (button, link) / 可选操作元素 |
| ...rest     | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes                       |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

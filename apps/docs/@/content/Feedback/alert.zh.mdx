# Alert 警告提示

> 反馈组件

静态展示需要关注的信息,非浮层。支持 default/info/success/warning/destructive 变体。

## 何时使用

向用户显示重要提示、警告、错误或成功反馈信息。

## 代码示例

### 基础用法

<CodeBlock code={`import { Alert, AlertTitle, AlertDescription } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Alert>
      <AlertTitle>Title</AlertTitle>
      <AlertDescription>Alert description goes here.</AlertDescription>
    </Alert>
  )
}`} lang="tsx" />

### 变体

<CodeBlock
  code={`{/* 展示 variant 变体 */}
<div className="flex flex-wrap items-center gap-3">
  <Alert variant="default">Default</Alert>
  <Alert variant="info">Info</Alert>
  <Alert variant="success">Success</Alert>
  <Alert variant="warning">Warning</Alert>
  <Alert variant="destructive">Destructive</Alert>
</div>`}
  lang="tsx"
/>

## 样式变体

**变体**

| Variant       | 说明          | 默认 |
| ------------- | ------------- | ---- |
| `default`     | 默认样式      | ✓    |
| `info`        | 信息提示      |      |
| `success`     | 成功状态      |      |
| `warning`     | 警告状态      |      |
| `destructive` | 危险/删除样式 |      |

## API / Props

### Alert

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `variant`   | `AlertVariant`         | —       | 样式变体          |
| `icon`      | `React.ReactNode`      | —       | 图标元素          |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 2 个子组件(AlertTitle, AlertDescription),使用组合模式按需导入。

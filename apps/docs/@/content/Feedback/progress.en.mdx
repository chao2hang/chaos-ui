# Progress

> Feedback and status communication to the user.

Shows current progress of an operation.

## When to Use

Show current completion progress of an operation.


## Examples

### Basic Usage

<CodeBlock code={`import { Progress, ProgressTrack, ProgressIndicator, ProgressLabel, ProgressValue } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Progress>
      <ProgressTrack>Content</ProgressTrack>
    </Progress>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 4 个子组件(ProgressTrack, ProgressIndicator, ProgressLabel, ProgressValue),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

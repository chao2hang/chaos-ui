## API / Props

### EmptyState

| Prop          | Type                          | Default | 说明 / Description                                          |
| ------------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className`   | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `icon`        | `React.ReactNode`             | —       | Icon element                                                |
| `title`       | `React.ReactNode`             | —       | Title text                                                  |
| `description` | `React.ReactNode`             | —       | Description text                                            |
| `action`      | `React.ReactNode`             | —       | Action element (button, link, etc.)                         |
| ...rest       | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

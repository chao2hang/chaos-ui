# ErrorBoundaryUI 错误边界

> 反馈组件 / Feedback components for user interaction and status.

错误边界 — React 错误边界，支持降级渲染与重置。
ErrorBoundaryUI — React error boundary UI with fallback render and reset.

## 何时使用 / When to Use

当你需要「错误边界」相关能力时使用：React 错误边界，支持降级渲染与重置。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use ErrorBoundaryUI when you need: React error boundary UI with fallback render and reset. Use for loading, confirmation, warnings, and notifications—never fail silently.

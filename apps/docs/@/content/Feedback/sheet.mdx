# Sheet 面板

> 反馈组件 / Feedback and status communication to the user.

抽屉式底部/侧边面板,移动端常用。
Bottom/side panel (common on mobile).

## 何时使用 / When to Use

抽屉式面板的别名,常用于移动端临时内容层。

Alias for drawer-style panels; often used for temporary layers on mobile.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Sheet, SheetTrigger, SheetContent, SheetHeader, SheetTitle, SheetDescription } from "@/components/ui/sheet"

export function Basic() {
  return (
    <Sheet>
      <SheetTrigger>Open</SheetTrigger>
      <SheetContent>
      <SheetHeader>
        <SheetTitle>Title</SheetTitle>
        <SheetDescription>Description.</SheetDescription>
      </SheetHeader>
      </SheetContent>
    </Sheet>
  )
}`} lang="tsx" />

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 7 个子组件(SheetTrigger, SheetClose, SheetContent, SheetHeader, SheetFooter…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

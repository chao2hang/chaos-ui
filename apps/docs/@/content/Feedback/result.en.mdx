# Result

> Feedback and status communication to the user.

Feedback for an operation result (success/fail/warning/404/403), typically full-page.

## When to Use

Use when you need a result.


## Examples

### Basic Usage

<CodeBlock code={`import { Result } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Result>Default</Result>
}`} lang="tsx" />

### Sizes

<CodeBlock code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <Result size="sm">Sm</Result>
  <Result size="default">Default</Result>
  <Result size="lg">Lg</Result>
</div>`} lang="tsx" />

## API / Props

### Result

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `status` | `"success" \| "error" \| "info" \| "wa...` | `info` | Status determines default icon and color |
| `title` | `React.ReactNode` | — | Title text |
| `subtitle` | `React.ReactNode` | — | Subtitle / description |
| `icon` | `React.ReactNode` | — | Custom icon override |
| `extra` | `React.ReactNode` | — | Extra content (e.g., action buttons) |
| ...rest | `React.ComponentProps<"div">` | — | Inherits all `<div>` attributes |

# GlobalLoading 全局加载

> 反馈组件 / Feedback components for user interaction and status.

全局加载 — 全局加载指示器，用于路由切换与全屏加载。
GlobalLoading — Global loading indicator for route switches and full-page loads.

## 何时使用 / When to Use

需要全局加载时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

Use GlobalLoading when you need: GlobalLoading: enterprise business component — see Storybook for interactive demos. Use for ERP/domain workflows (docs, pickers, approval, settlement) with business semantics.

# Skeleton 骨架屏

> 反馈组件 / Feedback and status communication to the user.

在数据加载完成前提供占位,降低等待焦虑感。
Placeholder shown while data loads; reduces perceived wait time.

## 何时使用 / When to Use

数据加载中时占位,降低感知等待时间。

Provide placeholder UI while data loads to reduce perceived wait time.

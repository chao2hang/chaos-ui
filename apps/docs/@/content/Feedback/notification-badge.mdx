# NotificationBadge 通知角标

> 反馈组件 / Feedback components for user interaction and status.

通知角标 — Header 通知角标，显示未读数小圆点。
NotificationBadge — Header notification badge with unread count dot.

## 何时使用 / When to Use

当你需要「通知角标」相关能力时使用：Header 通知角标，显示未读数小圆点。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use NotificationBadge when you need: Header notification badge with unread count dot. Use for loading, confirmation, warnings, and notifications—never fail silently.

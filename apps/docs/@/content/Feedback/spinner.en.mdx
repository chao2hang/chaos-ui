# Spinner

> Feedback and status communication to the user.

Local loading spinner indicating content is being fetched.

## When to Use

Indicate an in-progress operation.


## Examples

### Basic Usage

<CodeBlock code={`import { Spinner } from "@/components/ui/spinner"

export function Basic() {
  return <Spinner>Default</Spinner>
}`} lang="tsx" />

<CodeBlock code={`{/* 展示 variant 变体 */}

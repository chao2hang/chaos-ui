# Notification 通知提醒

> 反馈组件 / Feedback and status communication to the user.

右上角弹出的通知卡片,比 Toast 内容更丰富、驻留更久。
Top-right notification card with richer content and longer duration than toast.

## 何时使用 / When to Use

需要通知提醒功能时使用本组件。

Use when you need a notification.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant       | 说明 / Description                       | 默认 |
| ------------- | ---------------------------------------- | ---- |
| `default`     | 默认样式 / Default style                 | ✓    |
| `info`        | 信息提示 / Informational style           |      |
| `success`     | 成功状态 / Success style                 |      |
| `warning`     | 警告状态 / Warning style                 |      |
| `destructive` | 危险/删除样式 / Destructive/danger style |      |

## API / Props

### Notification

| Prop          | Type                          | Default | 说明 / Description                                          |
| ------------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className`   | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `title`       | `React.ReactNode`             | —       | Title                                                       |
| `description` | `React.ReactNode`             | —       | Description content                                         |
| `open`        | `boolean`                     | `true`  | Whether the notification is visible                         |
| `onClose`     | `() => void`                  | —       | Called when close button is clicked                         |
| `duration`    | `number`                      | `4500`  | Auto-close duration in ms (0 to disable)                    |
| `icon`        | `React.ReactNode`             | —       | Icon to show                                                |
| ...rest       | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

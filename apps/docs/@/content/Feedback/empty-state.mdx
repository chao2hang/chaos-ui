# EmptyState 空状态

> 反馈组件 / Feedback and status communication to the user.

当前没有数据时的占位展示,可搭配引导操作。
Placeholder for empty data states, optionally with a call-to-action.

## 何时使用 / When to Use

当你需要「空状态」相关能力时使用：当前没有数据时的占位展示,可搭配引导操作。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use EmptyState when you need: Placeholder for empty data states, optionally with a call-to-action. Use for loading, confirmation, warnings, and notifications—never fail silently.

## API / Props

### EmptyState

| Prop          | Type                          | Default | 说明 / Description                                          |
| ------------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className`   | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `icon`        | `React.ReactNode`             | —       | Icon element                                                |
| `title`       | `React.ReactNode`             | —       | Title text                                                  |
| `description` | `React.ReactNode`             | —       | Description text                                            |
| `action`      | `React.ReactNode`             | —       | Action element (button, link, etc.)                         |
| ...rest       | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# Dialog 对话框

> 反馈组件 / Feedback and status communication to the user.

模态弹出框,用于需要用户处理事务的重要场景,不支持多框叠加。
Modal popup for critical user interactions; does not support stacking.

## 何时使用 / When to Use

需要用户处理重要事务,且不希望跳转页面打断工作流时使用模态框。

Use a modal when the user must handle a critical task without leaving the current workflow.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription } from "@/components/ui/dialog"

export function Basic() {
  return (

    </Dialog>

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 9 个子组件(DialogClose, DialogContent, DialogDescription, DialogFooter, DialogHeader…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# Popconfirm 气泡确认框

> 反馈组件 / Feedback and status communication to the user.

气泡确认框 — Chaos UI 的反馈组件。
Popconfirm — a feedback and status communication to the user. from Chaos UI.

## 何时使用 / When to Use

需要气泡确认框时使用本组件。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

Use Popconfirm when you need: Popconfirm — a feedback and status communication to the user. from Chaos UI. Use for loading, confirmation, warnings, and notifications—never fail silently.

## API / Props

### Popconfirm

| Prop           | Type                          | Default   | 说明 / Description                                          |
| -------------- | ----------------------------- | --------- | ----------------------------------------------------------- |
| `className`    | `string`                      | —         | 自定义样式类名 <br/> Custom CSS class                       |
| `open`         | `boolean`                     | —         | Whether the popconfirm is open                              |
| `onOpenChange` | `() => void`                  | —         | Called when open state changes                              |
| `title`        | `React.ReactNode`             | —         | Title text                                                  |
| `description`  | `React.ReactNode`             | —         | Description text                                            |
| `confirmText`  | `string`                      | `Confirm` | Text for confirm button                                     |
| `cancelText`   | `string`                      | `Cancel`  | Text for cancel button                                      |
| `onConfirm`    | `() => void`                  | —         | Called when confirmed                                       |
| `onCancel`     | `() => void`                  | —         | Called when cancelled                                       |
| ...rest        | `React.ComponentProps<"div">` | —         | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

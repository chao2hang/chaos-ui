# ModalProvider 模态提供者

> 反馈组件

模态提供者 — Chaos UI 的反馈组件。

## 何时使用

需要模态提供者时使用本组件。用于过程反馈与结果提示（加载、确认、警告、通知）；避免静默失败。

## 代码示例

### 基础用法

<CodeBlock code={`import { ModalProvider, Modal, ModalContext } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <div className="space-y-4">
  <ModalProvider />
  <Modal />
  <ModalContext />
    </div>
  )
}`} lang="tsx" />

## API / Props

### ModalProvider

| Prop        | Type                          | Default | 说明                  |
| ----------- | ----------------------------- | ------- | --------------------- |
| `className` | `string`                      | —       | 自定义样式类名        |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

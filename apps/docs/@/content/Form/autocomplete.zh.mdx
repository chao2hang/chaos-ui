# Autocomplete 自动完成

> 表单控件

输入框联想下拉,输入时实时建议匹配项。

## 何时使用

当你需要「自动完成」相关能力时使用：输入框联想下拉,输入时实时建议匹配项。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

## 样式变体

**变体**

| Variant   | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认样式 | ✓    |
| `outline` | 描边样式 |      |

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |
| `sm`      | 小号     |      |
| `lg`      | 大号     |      |

## API / Props

### Autocomplete

| Prop            | Type                   | Default | 说明                                 |
| --------------- | ---------------------- | ------- | ------------------------------------ |
| `className`     | `string`               | —       | 自定义样式类名                       |
| `options`       | `string[]`             | —       | Options to suggest                   |
| `value`         | `string`               | —       | Controlled value                     |
| `onValueChange` | `() => void`           | —       | Change handler                       |
| `minChars`      | `number`               | `1`     | Min chars before showing suggestions |
| ...rest         | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                    |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

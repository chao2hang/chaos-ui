| `accept`    | `Record<string, string[]>` | —       |                                                           |
| `maxFiles`  | `number`                   | `5`     |                                                           |
| `maxSize`   | `number`                   | —       |                                                           |
| `disabled`  | `boolean`                  | —       | 是否禁用 <br/> Whether disabled                           |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(FileList),使用组合模式按需导入。

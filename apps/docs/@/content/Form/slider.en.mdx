# Slider

> Form controls for user input and data submission.

Drag to select a numeric value within a range; more visual than an input.

## When to Use

Select a value from a continuous or discrete range via dragging.


## Examples

### Basic Usage

<CodeBlock code={`import { Slider, SliderControl, SliderTrack, SliderIndicator, SliderThumb } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Slider>
      <SliderControl>Content</SliderControl>
    </Slider>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 4 个子组件(SliderControl, SliderTrack, SliderIndicator, SliderThumb),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

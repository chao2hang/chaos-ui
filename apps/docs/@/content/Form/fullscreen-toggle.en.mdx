Use when you need a fullscreen toggle.

## Style Variants

**Variant**

| Variant   | Description                 | 默认 |
| --------- | --------------------------- | ---- |
| `default` | Default style               |      |
| `outline` | Outlined style              |      |
| `ghost`   | Ghost (no background) style | ✓    |

**Size**

| Size      | Description      | 默认 |
| --------- | ---------------- | ---- |
| `default` | Default size     |      |
| `sm`      | Small            |      |
| `lg`      | Large            |      |
| `icon`    | Icon-only square | ✓    |

## API / Props

### FullscreenToggle

| Prop        | Type                             | Default | Description                                                        |
| ----------- | -------------------------------- | ------- | ------------------------------------------------------------------ |
| `className` | `string`                         | —       | Custom CSS class                                                   |
| `target`    | `React.RefObject<HTMLElement>`   | —       | Target element to fullscreen. Defaults to document.documentElement |
| ...rest     | `React.ComponentProps<"button">` | —       | Inherits all `<button>` attributes                                 |

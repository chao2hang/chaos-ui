# Cascader

> Form controls for user input and data submission.

Select from hierarchical linked data such as Province/City/District.

## When to Use

Use when you need a cascader.

## API / Props

### Cascader

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `options` | `CascaderOption[]` | — | Available options |
| `value` | `string[]` | — | Current value (controlled) |
| `onChange` | `() => void` | — | Callback when value changes |

| `disabled` | `boolean` | — | Whether disabled |

# Label 标签文本

> 表单控件 / Form controls for user input and data submission.

表单字段标签,支持 required 星号、字号、对齐。
Form field label supporting required asterisk, sizes, and alignment.

## 何时使用 / When to Use

为表单控件提供可访问的文本标签。

Provide an accessible text label for a form control.

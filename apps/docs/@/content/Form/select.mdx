# Select 选择器

> 表单控件 / Form controls for user input and data submission.

下拉选项选择器,支持单选/搜索/自定义选项渲染。
Dropdown option selector supporting single-select, search, and custom option rendering.

## 何时使用 / When to Use

从一组预设选项中选择单个值,选项过多时配合搜索使用。

Pick a single value from a predefined list; use search when options are many.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem } from "@/components/ui/select"

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 9 个子组件(SelectContent, SelectGroup, SelectItem, SelectLabel, SelectScrollDownButton…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

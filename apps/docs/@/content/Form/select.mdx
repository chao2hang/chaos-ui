# Select 选择器

Chaos UI 下拉选择器组件，基于 Base UI Select，支持单选、搜索、分组等。
A Base UI-powered dropdown select with single selection, search, and grouping.

## 何时使用 / When to use

用于从预定义选项列表中选择一项。当选项 ≤5 个时考虑 RadioGroup，当选项很多或需要搜索时使用 Select。

Use for selecting one item from a predefined list. Consider RadioGroup for ≤5 options; use Select when options are many or search is needed.

## API

### Select

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| defaultValue | `string` | — | 默认选中的值 | Default selected value |
| value | `string` | — | 受控：当前选中的值 | Controlled selected value |
| onValueChange | `(value: string) => void` | — | 选择回调 | Callback on value change |
| disabled | `boolean` | `false` | 禁用整个选择器 | Disable entire select |
| name | `string` | — | 表单字段名（用于原生表单提交） | Form field name |
| ...rest | `SelectPrimitive.Root.Props` | — | 透传 Base UI 属性 | Passthrough Base UI props |

### SelectTrigger

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| size | `'sm' \| 'default'` | `'default'` | 触发器尺寸 | Trigger size |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### SelectContent

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| align | `'start' \| 'center' \| 'end'` | `'center'` | 弹出对齐方式 | Popup alignment |
| side | `'top' \| 'bottom'` | `'bottom'` | 弹出方向 | Popup side |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### SelectItem

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| value | `string` | — | **必填**，选项值 | **Required** option value |
| disabled | `boolean` | `false` | 禁用该选项 | Disable this option |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

## 注意事项

- **弹出层定位**：`SelectContent` 使用 Portal 渲染到 `<body>`，通过 `align`/`side` 控制定位。如果弹出层被父容器裁剪，检查父容器的 `overflow` 样式。
- **SelectValue**：使用 `placeholder` 属性设置未选择时的显示文字。已选择时自动显示对应 `SelectItem` 的 children。
- **无障碍**：Base UI Select 遵循 WAI-ARIA Listbox Pattern，键盘支持 Enter/Space 打开、Arrow keys 导航、Escape 关闭。
- **与 Form 集成**：设置 `name` 属性后 Select 可参与原生表单提交。

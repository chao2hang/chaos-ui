# Calendar 日历

> 表单控件

按照日历形式展示数据的容器,可用于日期选择或日程展示。

## 何时使用

以日历面板方式展示/选择日期。


## 代码示例

### 基础用法

<CodeBlock code={`import { Calendar, CalendarDayButton } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Calendar>
      <CalendarDayButton>Content</CalendarDayButton>
    </Calendar>
  )
}`} lang="tsx" />

### 状态

<CodeBlock code={`<Calendar>Normal</Calendar>
<Calendar disabled>Disabled</Calendar>`} lang="tsx" />

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(CalendarDayButton),使用组合模式按需导入。

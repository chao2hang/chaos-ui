# BrowseInput 浏览输入框

> 表单控件 / Form controls for user input and data submission.

输入框右侧带浏览按钮,常用于选择文件/目录/业务对象。
Input with a browse button on the right; used for selecting files/directories/business objects.

## 何时使用 / When to Use

当你需要「浏览输入框」相关能力时使用：输入框右侧带浏览按钮,常用于选择文件/目录/业务对象。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use BrowseInput when you need: Input with a browse button on the right; used for selecting files/directories/business objects. Use to collect or edit data with validation and submit flows.

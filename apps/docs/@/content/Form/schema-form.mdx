# SchemaForm Schema 表单

## 何时使用 / When to Use

适用于Schema 表单相关场景。

Use when you need a schema form.

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **表单校验**: 集成 `zod` + `react-hook-form`,支持类型安全的表单校验与提交。

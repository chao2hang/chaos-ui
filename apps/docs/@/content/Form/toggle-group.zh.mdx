# ToggleGroup 开关组

> 表单控件

开关组 — Chaos UI 的表单控件。

## 何时使用

需要开关组功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { ToggleGroup, ToggleGroupItem } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <ToggleGroup>
      <ToggleGroupItem>Content</ToggleGroupItem>
    </ToggleGroup>
  )
}`} lang="tsx" />

### 状态

<CodeBlock
  code={`<ToggleGroup>Normal</ToggleGroup>

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(ToggleGroupItem),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# ChatInput Chat Input

> 表单控件

Chat Input — Chaos UI 的表单控件。

## 何时使用

需要采集或编辑业务字段，并进入校验/提交链路时使用。（组件：Chat Input）

## 代码示例

### 基础用法

<CodeBlock code={`import { ChatInput } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <ChatInput />
}`} lang="tsx" />

## API / Props

### ChatInput

| Prop          | Type                   | Default | 说明              |
| ------------- | ---------------------- | ------- | ----------------- |
| `className`   | `string`               | —       | 自定义样式类名    |
| `value`       | `string`               | —       | 当前输入值        |
| `onChange`    | `() => void`           | —       | 值变化回调        |
| `onSend`      | `() => void`           | —       | 发送回调          |
| `placeholder` | `string`               | —       | 占位提示          |
| `disabled`    | `boolean`              | —       | 是否禁用          |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

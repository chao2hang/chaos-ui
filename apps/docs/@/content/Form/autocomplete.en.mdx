# Autocomplete

> Form controls for user input and data submission.

Input with real-time suggestion dropdown as user types.

## When to Use

Use Autocomplete when you need: Input with real-time suggestion dropdown as user types. Use to collect or edit data with validation and submit flows.

## Style Variants

**Variant**

| Variant   | Description    | 默认 |
| --------- | -------------- | ---- |
| `default` | Default style  | ✓    |
| `outline` | Outlined style |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

## API / Props

### Autocomplete

| Prop            | Type                   | Default | Description                          |
| --------------- | ---------------------- | ------- | ------------------------------------ |
| `className`     | `string`               | —       | Custom CSS class                     |
| `options`       | `string[]`             | —       | Options to suggest                   |
| `value`         | `string`               | —       | Controlled value                     |
| `onValueChange` | `() => void`           | —       | Change handler                       |
| `minChars`      | `number`               | `1`     | Min chars before showing suggestions |
| ...rest         | `React.HTMLAttributes` | —       | Inherits standard HTML attributes    |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

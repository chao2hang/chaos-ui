# ColorPicker 颜色选择器

> 表单控件

选择颜色的面板控件,支持色板、输入色值、透明度。

## 何时使用

需要颜色选择器功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { ColorPicker } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <ColorPicker />
}`} lang="tsx" />

### 状态

<CodeBlock code={`<ColorPicker>Normal</ColorPicker>
<ColorPicker disabled>Disabled</ColorPicker>`} lang="tsx" />

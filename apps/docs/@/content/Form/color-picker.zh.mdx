# ColorPicker 颜色选择器

> 表单控件

选择颜色的面板控件,支持色板、输入色值、透明度。

## 何时使用

当你需要「颜色选择器」相关能力时使用：选择颜色的面板控件,支持色板、输入色值、透明度。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

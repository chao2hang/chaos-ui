# Combobox

> Form controls for user input and data submission.

Combines input with a dropdown list; supports custom input and option matching.

## When to Use

Use when you need a combobox.

## Style Variants

**Variant**

| Variant   | Description    | 默认 |
| --------- | -------------- | ---- |
| `default` | Default style  | ✓    |
| `outline` | Outlined style |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

## API / Props

### Combobox

| Prop           | Type                   | Default             | Description                       |
| -------------- | ---------------------- | ------------------- | --------------------------------- |
| `className`    | `string`               | —                   | Custom CSS class                  |
| `options`      | `ComboboxOption[]`     | —                   | Available options                 |
| `value`        | `string`               | —                   | Current value (controlled)        |
| `onChange`     | `() => void`           | —                   | Callback when value changes       |
| `placeholder`  | `string`               | `Select option...`  | Placeholder text                  |
| `searchable`   | `boolean`              | `true`              |                                   |
| `emptyMessage` | `string`               | `No results found.` |                                   |
| ...rest        | `React.HTMLAttributes` | —                   | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

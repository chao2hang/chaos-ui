# Switch

> Form controls for user input and data submission.

Toggle switch representing binary state — more visually intuitive than a checkbox.

## When to Use

Toggle between two states, typically on/off or enabled/disabled.

## Examples

### Basic Usage

<CodeBlock code={`import { Switch } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Switch />
}`} lang="tsx" />

### States

<CodeBlock
  code={`<Switch>Normal</Switch>

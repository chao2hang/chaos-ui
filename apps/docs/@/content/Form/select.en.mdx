# Select

> Form controls for user input and data submission.

Dropdown option selector supporting single-select, search, and custom option rendering.

## When to Use

Pick a single value from a predefined list; use search when options are many.

## Examples

### Basic Usage

<CodeBlock code={`import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Select defaultValue="1">
      <SelectTrigger><SelectValue /></SelectTrigger>
      <SelectContent>
        <SelectItem value="1">Option 1</SelectItem>
        <SelectItem value="2">Option 2</SelectItem>
      </SelectContent>
    </Select>
  )
}`} lang="tsx" />

### States

<CodeBlock
  code={`<Select>Normal</Select>
<Select disabled>Disabled</Select>`}

## API / Props

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 9 个子组件(SelectContent, SelectGroup, SelectItem, SelectLabel, SelectScrollDownButton…),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

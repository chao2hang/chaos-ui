# OtpField

> Form controls for user input and data submission.

One-time password input with individual cells for each character; auto-focuses next cell.

## When to Use

One-time password / SMS verification code input.

## Examples

### Basic Usage

<CodeBlock code={`import { OTPField, OTPFieldSeparator } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <OTPField>
      <OTPFieldSeparator>Content</OTPFieldSeparator>
    </OTPField>
  )
}`} lang="tsx" />

## API / Props

### OTPField

| Prop             | Type                   | Default | Description                       |
| ---------------- | ---------------------- | ------- | --------------------------------- |
| `className`      | `string`               | —       | Custom CSS class                  |
| `length`         | `number`               | —       |                                   |
| `value`          | `string`               | —       | Current value (controlled)        |
| `onValueChange`  | `() => void`           | —       |                                   |
| `mask`           | `boolean`              | `false` |                                   |
| `inputClassName` | `string`               | —       |                                   |
| ...rest          | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(OTPFieldSeparator),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# Combobox 组合框

> 表单控件

结合输入框和下拉列表的组合选择控件,支持自定义输入和选项匹配。

## 何时使用

需要组合框功能时使用本组件。

## 样式变体

**变体**

| Variant   | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认样式 | ✓    |
| `outline` | 描边样式 |      |

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `default` | 默认尺寸 | ✓    |
| `sm`      | 小号     |      |
| `lg`      | 大号     |      |

## API / Props

### Combobox

| Prop           | Type                   | Default             | 说明              |
| -------------- | ---------------------- | ------------------- | ----------------- |
| `className`    | `string`               | —                   | 自定义样式类名    |
| `options`      | `ComboboxOption[]`     | —                   | 可选项列表        |
| `value`        | `string`               | —                   | 当前值(受控)      |
| `onChange`     | `() => void`           | —                   | 值变化回调        |
| `placeholder`  | `string`               | `Select option...`  | 占位文本          |
| `searchable`   | `boolean`              | `true`              |                   |
| `emptyMessage` | `string`               | `No results found.` |                   |
| ...rest        | `React.HTMLAttributes` | —                   | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

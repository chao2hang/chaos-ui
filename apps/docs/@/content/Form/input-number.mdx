# InputNumber 数字输入

> 表单控件 / Form controls for user input and data submission.

专门输入数字的输入框,支持增减按钮、精度、范围限制。
Numeric input with increment/decrement buttons, precision, and range limits.

## 何时使用 / When to Use

当你需要「数字输入」相关能力时使用：专门输入数字的输入框,支持增减按钮、精度、范围限制。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use InputNumber when you need: Numeric input with increment/decrement buttons, precision, and range limits. Use to collect or edit data with validation and submit flows.

## API / Props

### InputNumber

| Prop          | Type                   | Default | 说明 / Description                                        |
| ------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`   | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `value`       | `number`               | —       | 当前值(受控) <br/> Current value (controlled)             |
| `onChange`    | `() => void`           | —       | 值变化回调 <br/> Callback when value changes              |
| `min`         | `number`               | —       | 最小值 <br/> Minimum value                                |
| `max`         | `number`               | —       | 最大值 <br/> Maximum value                                |
| `step`        | `number`               | `1`     | 步长 <br/> Step value                                     |
| `showStepper` | `boolean`              | `true`  | Show stepper buttons                                      |
| `allowEmpty`  | `boolean`              | `false` | Allow empty value                                         |
| `suffix`      | `React.ReactNode`      | —       | Suffix text                                               |
| ...rest       | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

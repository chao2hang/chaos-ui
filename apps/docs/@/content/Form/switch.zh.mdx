# Switch 开关

> 表单控件

开关选择器,代表两种状态之间的切换,比 Checkbox 更直观的开关形态。

## 何时使用

两种状态之间的切换,典型用于开/关、启用/禁用。


## 代码示例

### 基础用法

<CodeBlock code={`import { Switch } from "@/components/ui/switch"

export function Basic() {
  return <Switch />
}`} lang="tsx" />

### 状态

<CodeBlock code={`<Switch>Normal</Switch>
<Switch disabled>Disabled</Switch>`} lang="tsx" />

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

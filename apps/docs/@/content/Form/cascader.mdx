# Cascader 级联选择

> 表单控件 / Form controls for user input and data submission.

从多级层级关联数据中进行选择,如省/市/区。
Select from hierarchical linked data such as Province/City/District.

## 何时使用 / When to Use

当你需要「级联选择」相关能力时使用：从多级层级关联数据中进行选择,如省/市/区。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use Cascader when you need: Select from hierarchical linked data such as Province/City/District. Use to collect or edit data with validation and submit flows.

## API / Props

### Cascader

| Prop          | Type                   | Default         | 说明 / Description                                        |
| ------------- | ---------------------- | --------------- | --------------------------------------------------------- |
| `className`   | `string`               | —               | 自定义样式类名 <br/> Custom CSS class                     |
| `options`     | `CascaderOption[]`     | —               | 可选项列表 <br/> Available options                        |
| `value`       | `string[]`             | —               | 当前值(受控) <br/> Current value (controlled)             |
| `onChange`    | `() => void`           | —               | 值变化回调 <br/> Callback when value changes              |
| `placeholder` | `string`               | `Please select` | 占位文本 <br/> Placeholder text                           |
| `disabled`    | `boolean`              | —               | 是否禁用 <br/> Whether disabled                           |
| ...rest       | `React.HTMLAttributes` | —               | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

# MaskedInput 格式化输入

> 表单组件 / Form components for data entry and collection.

格式化输入 — 格式化掩码输入，支持身份证、银行卡、IP 等。
MaskedInput — Format-masked input for ID cards, bank cards, IP addresses, etc.

## 何时使用 / When to Use

当你需要「格式化输入」相关能力时使用：格式化掩码输入，支持身份证、银行卡、IP 等。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

Use MaskedInput when you need: Format-masked input for ID cards, bank cards, IP addresses, etc. Use to collect or edit data with validation and submit flows.

### react-hook-form 兼容 / react-hook-form Compatibility

`onChange` 同时接受 `(value: string) => void` 和标准 `React.ChangeEventHandler<HTMLInputElement>` 签名。这意味着你可以直接展开 react-hook-form 的 field 对象：

The `onChange` prop accepts both a simplified `(value: string) => void` and a standard `React.ChangeEventHandler<HTMLInputElement>`. This allows you to spread react-hook-form field props directly:

```tsx
// ✅ Works with react-hook-form spread
<PasswordInput {...field} placeholder="Enter password" />

// ✅ Also works with simplified callback
<PasswordInput value={pwd} onChange={setPwd} />
```

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **react-hook-form**: `onChange` 支持双签名,可直接展开 `{...field}`,name、ref、onBlur 全部自动带上。

# FileUpload

> Form controls for user input and data submission.

Click/drag-to-upload supporting multi-select, type restrictions, and preview.

## When to Use

File upload supporting drag & drop, batch upload, and progress feedback.


## Examples

### Basic Usage

<CodeBlock code={`import { FileUpload, FileList } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <FileUpload>
      <FileList>Content</FileList>
    </FileUpload>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(FileList),使用组合模式按需导入。

# Checkbox 复选框

> 表单控件

在一组选项中进行多选,或单独表示两种状态切换。

## 何时使用

在一组选项中进行多选,或标记单个二元状态。

## 代码示例

### 基础用法

<CodeBlock code={`import { Checkbox } from "@/components/ui/checkbox"

export function Basic() {
  return <Checkbox />
}`} lang="tsx" />

### 状态

<CodeBlock
  code={`<Checkbox>Normal</Checkbox>
<Checkbox disabled>Disabled</Checkbox>`}
  lang="tsx"
/>

## API / Props

### Checkbox

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

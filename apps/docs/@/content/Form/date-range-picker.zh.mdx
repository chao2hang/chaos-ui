# DateRangePicker 日期范围选择

> 表单组件

日期范围选择 — 日期范围选择器，支持起止日期选择。

## 何时使用

需要日期范围选择器时使用本组件。用于 ERP/业务域场景（单据、选择器、审批、对账等）；按业务语义接入而不是当纯 UI 积木。

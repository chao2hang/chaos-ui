## API / Props

### NativeSelect

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `options` | `(NativeSelectOption \| NativeSelectGr...` | — | Flat options or grouped options / 扁平选项或分组选项 |
| `placeholder` | `string` | — | Placeholder option text / 占位选项文本 |
| `size` | `"sm" \| "default"` | `default` | Size variant / 尺寸变体 |
| `error` | `boolean` | — | Whether to show error styling / 是否显示错误样式 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

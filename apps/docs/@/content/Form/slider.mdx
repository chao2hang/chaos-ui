# Slider 滑块

Chaos UI 滑块组件，基于 Base UI Slider，用于在固定范围内选择一个数值或范围。
A Base UI-powered slider for selecting a value or range within a fixed interval.

## 何时使用 / When to use

用于在连续或离散范围内选择数值：音量调节、价格范围筛选、亮度调整、评分等。适合不需要精确数值的场景。

Use for selecting a value in a continuous or discrete range: volume control, price range filter, brightness, rating. Best when precise numeric input isn't needed.

## API

### Slider

| Prop | Type | Default | Description (zh) | Description (en) |
|---|---|---|---|---|
| defaultValue | `number \| number[]` | — | 默认值 | Default value |
| value | `number \| number[]` | — | 受控：当前值 | Controlled value |
| onValueChange | `(value: number \| number[]) => void` | — | 值变化回调 | Callback on value change |
| min | `number` | `0` | 最小值 | Minimum value |
| max | `number` | `100` | 最大值 | Maximum value |
| step | `number` | `1` | 步长 | Step interval |
| disabled | `boolean` | `false` | 禁用状态 | Disabled state |
| className | `string` | — | 自定义样式类名 | Custom CSS class |

### Slider 子组件

| 组件 | Props | 描述 |
|---|---|---|
| Slider | `SliderPrimitive.Root.Props` | 滑块根容器 |
| SliderControl | `SliderPrimitive.Control.Props` | 滑块控制区 |
| SliderTrack | `SliderPrimitive.Track.Props` | 滑块轨道 |
| SliderIndicator | `SliderPrimitive.Indicator.Props` | 已填充轨道 |
| SliderThumb | `SliderPrimitive.Thumb.Props` | 滑块手柄 |

## 注意事项

- **单值 vs 范围**：传入单个数值为单点滑块，传入 `[min, max]` 数组为范围滑块。
- **步长精度**：`step` 控制拖拽时的精度，同时也影响键盘操作的步进量。设置 `step={0.01}` 可支持小数精度。
- **无障碍**：Base UI Slider 使用 `<div role="slider">` 实现，键盘支持 Arrow Left/Right 调整数值、Home/End 跳至最小/最大值。确保提供 `aria-label` 或 `aria-labelledby`。
- **子组件**：一般直接使用 `Slider` 即可，默认包含完整的 Track + Thumb。仅在需要自定义布局时才使用子组件组合。

## 样式变体 / Style Variants

**变体 / Variant**

| Variant   | 说明 / Description        | 默认 |
| --------- | ------------------------- | ---- |
| `default` | 默认样式 / Default style  | ✓    |
| `outline` | 描边样式 / Outlined style |      |

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |

## API / Props

### Combobox

| Prop           | Type                   | Default             | 说明 / Description                                        |
| -------------- | ---------------------- | ------------------- | --------------------------------------------------------- |
| `className`    | `string`               | —                   | 自定义样式类名 <br/> Custom CSS class                     |
| `options`      | `ComboboxOption[]`     | —                   | 可选项列表 <br/> Available options                        |
| `value`        | `string`               | —                   | 当前值(受控) <br/> Current value (controlled)             |
| `onChange`     | `() => void`           | —                   | 值变化回调 <br/> Callback when value changes              |
| `placeholder`  | `string`               | `Select option...`  | 占位文本 <br/> Placeholder text                           |
| `searchable`   | `boolean`              | `true`              |                                                           |
| `emptyMessage` | `string`               | `No results found.` |                                                           |
| ...rest        | `React.HTMLAttributes` | —                   | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

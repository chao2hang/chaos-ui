# Form

> Form controls for user input and data submission.

Form container enabling data collection, validation, and submission with FormField.

## When to Use

Collect, validate, and submit structured user input.

## Examples

### Basic Usage

<CodeBlock code={`import { Form, FormItem, FormField } from "@/components/ui/form"

export function Basic() {
  return (
    <Form>
      <FormItem>Content</FormItem>
    </Form>
  )
}`} lang="tsx" />

## API / Props

### Form

| Prop        | Type                           | Default | Description                                         |
| ----------- | ------------------------------ | ------- | --------------------------------------------------- |
| `className` | `string`                       | —       | Custom CSS class                                    |
| `onSubmit`  | `() => void`                   | —       | Called on submit with form data                     |
| `errors`    | `Record<string, string>`       | —       | Validation errors map (field name -> error message) |
| ...rest     | `React.ComponentProps<"form">` | —       | Inherits all `<form>` attributes                    |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(FormItem, FormField),使用组合模式按需导入。

# FullscreenToggle 全屏切换

> 表单控件

全屏切换 — Chaos UI 的表单控件。

## 何时使用

需要全屏切换功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { FullscreenToggle } from "@/components/ui/fullscreen-toggle"

export function Basic() {
  return <FullscreenToggle />
}`} lang="tsx" />

<CodeBlock

## 样式变体

**变体**

| Variant   | 说明             | 默认 |
| --------- | ---------------- | ---- |
| `default` | 默认样式         |      |
| `outline` | 描边样式         |      |
| `ghost`   | 幽灵(无背景)样式 | ✓    |

**尺寸**

| Size      | 说明       | 默认 |
| --------- | ---------- | ---- |
| `default` | 默认尺寸   |      |
| `sm`      | 小号       |      |
| `lg`      | 大号       |      |
| `icon`    | 图标正方形 | ✓    |

## API / Props

### FullscreenToggle

| Prop        | Type                             | Default | 说明                                                               |
| ----------- | -------------------------------- | ------- | ------------------------------------------------------------------ |
| `className` | `string`                         | —       | 自定义样式类名                                                     |
| `target`    | `React.RefObject<HTMLElement>`   | —       | Target element to fullscreen. Defaults to document.documentElement |
| ...rest     | `React.ComponentProps<"button">` | —       | 透传 `<button>` 原生属性                                           |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<button>` 原生 HTML 属性。

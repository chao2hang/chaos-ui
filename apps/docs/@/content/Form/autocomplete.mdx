# Autocomplete 自动完成

> 表单控件 / Form controls for user input and data submission.

输入框联想下拉,输入时实时建议匹配项。
Input with real-time suggestion dropdown as user types.

## 何时使用 / When to Use

需要自动完成功能时使用本组件。

Use when you need an autocomplete.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant   | 说明 / Description        | 默认 |
| --------- | ------------------------- | ---- |
| `default` | 默认样式 / Default style  | ✓    |
| `outline` | 描边样式 / Outlined style |      |

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |

## API / Props

### Autocomplete

| Prop            | Type                   | Default | 说明 / Description                                        |
| --------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`     | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `options`       | `string[]`             | —       | Options to suggest                                        |
| `value`         | `string`               | —       | Controlled value                                          |
| `onValueChange` | `() => void`           | —       | Change handler                                            |
| `minChars`      | `number`               | `1`     | Min chars before showing suggestions                      |
| ...rest         | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

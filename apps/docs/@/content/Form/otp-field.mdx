# OtpField 验证码输入

> 表单控件 / Form controls for user input and data submission.

一次性验证码输入框,多格子分别输入数字/字母,自动聚焦下一格。
One-time password input with individual cells for each character; auto-focuses next cell.

## 何时使用 / When to Use

一次性验证码/短信验证码输入。

One-time password / SMS verification code input.

## API / Props

### OTPField

| Prop             | Type                   | Default | 说明 / Description                                        |
| ---------------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className`      | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| `length`         | `number`               | —       |                                                           |
| `value`          | `string`               | —       | 当前值(受控) <br/> Current value (controlled)             |
| `onValueChange`  | `() => void`           | —       |                                                           |
| `mask`           | `boolean`              | `false` |                                                           |
| `inputClassName` | `string`               | —       |                                                           |
| ...rest          | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 1 个子组件(OTPFieldSeparator),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

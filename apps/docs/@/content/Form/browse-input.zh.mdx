# BrowseInput 浏览输入框

> 表单控件

输入框右侧带浏览按钮,常用于选择文件/目录/业务对象。

## 何时使用

当你需要「浏览输入框」相关能力时使用：输入框右侧带浏览按钮,常用于选择文件/目录/业务对象。用于采集或编辑业务数据；需要校验、提交、草稿时优先用表单控件族。

## API / Props

### BrowseInput

| Prop               | Type                   | Default     | 说明              |
| ------------------ | ---------------------- | ----------- | ----------------- |
| `className`        | `string`               | —           | 自定义样式类名    |
| `value`            | `string`               | —           | 当前值(受控)      |
| `defaultValue`     | `string`               | —           | 默认值(非受控)    |
| `placeholder`      | `string`               | `Select...` | 占位文本          |
| `disabled`         | `boolean`              | —           | 是否禁用          |
| `readOnly`         | `boolean`              | —           | 是否只读          |
| `required`         | `boolean`              | —           | 是否必填          |
| `onBrowse`         | `() => void`           | —           |                   |
| `onChange`         | `() => void`           | —           | 值变化回调        |
| `onClear`          | `() => void`           | —           |                   |
| `showClearButton`  | `boolean`              | `true`      |                   |
| `showBrowseButton` | `boolean`              | `true`      |                   |
| ...rest            | `React.HTMLAttributes` | —           | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

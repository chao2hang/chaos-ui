## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 5 个子组件(InputGroupAddon, InputGroupButton, InputGroupText, InputGroupInput, InputGroupTextarea),使用组合模式按需导入。
- **forwardRef**: `InputGroupInput` 和 `InputGroupTextarea` 均支持 `React.forwardRef`,可直接与 react-hook-form 的 `field.ref` 配合使用,实现焦点管理和字段注册。

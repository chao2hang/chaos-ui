# Checkbox 复选框

> 表单控件 / Form controls for user input and data submission.

在一组选项中进行多选,或单独表示两种状态切换。
Multi-select from a group, or standalone toggle for binary state.

## 何时使用 / When to Use

在一组选项中进行多选,或标记单个二元状态。

Select multiple items from a group, or toggle a single binary state.

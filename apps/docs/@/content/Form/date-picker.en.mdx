# DatePicker

> Form controls for user input and data submission.

Select single date, date ranges, or multiple dates.

## When to Use

Select a single date.

## API / Props

### DatePicker

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `value` | `Date` | — | Selected date |
| `onChange` | `() => void` | — | Change handler |
| `placeholder` | `string` | `Pick a date` | Placeholder text |
| `format` | `string` | — | Date format for display |
| `minDate` | `Date` | — | Min selectable date |
| `maxDate` | `Date` | — | Max selectable date |
| `disabled` | `boolean` | — | Whether disabled |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

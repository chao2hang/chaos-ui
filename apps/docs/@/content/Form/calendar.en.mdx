# Calendar

> Form controls for user input and data submission.

Calendar-view container for date selection or schedule display.

## When to Use

Display or select dates via a calendar panel.

## Examples

### Basic Usage

<CodeBlock code={`import { Calendar, CalendarDayButton } from "@/components/ui/calendar"

export function Basic() {
  return (
    <Calendar>
      <CalendarDayButton>Content</CalendarDayButton>
    </Calendar>
  )
}`} lang="tsx" />

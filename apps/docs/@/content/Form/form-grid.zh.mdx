# FormGrid 表单栅格

> 表单控件

表单栅格 — Chaos UI 的表单控件。

## 何时使用

需要表单栅格功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { FormGrid, FormGridItem } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <FormGrid>
      <FormGridItem>Content</FormGridItem>
    </FormGrid>
  )
}`} lang="tsx" />

## 样式变体

**columns**

| columns | 说明 | 默认 |
| ------- | ---- | ---- |
| `1`     |      | ✓    |
| `2`     |      |      |
| `3`     |      |      |
| `4`     |      |      |

**gap**

| gap       | 说明 | 默认 |
| --------- | ---- | ---- |
| `sm`      |      |      |
| `default` |      | ✓    |
| `lg`      |      |      |

## API / Props

### FormGrid

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(FormGridItem),使用组合模式按需导入。

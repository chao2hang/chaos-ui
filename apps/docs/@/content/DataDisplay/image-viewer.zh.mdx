## 样式变体

**open**

| open    | 说明 | 默认 |
| ------- | ---- | ---- |
| `true`  |      | ✓    |
| `false` |      |      |

## API / Props

### ImageViewer

| Prop           | Type                              | Default | 说明                                |
| -------------- | --------------------------------- | ------- | ----------------------------------- |
| `className`    | `string`                          | —       | 自定义样式类名                      |
| `open`         | `boolean`                         | —       | Whether the viewer is open          |
| `onClose`      | `() => void`                      | —       | Called when the viewer should close |
| `src`          | `string`                          | —       | Image source URL                    |
| `alt`          | `string`                          | ``      | Image alt text                      |
| `images`       | `{ src: string; alt?: string }[]` | —       | List of images for gallery mode     |
| `initialIndex` | `number`                          | `0`     | Initial index for gallery mode      |
| ...rest        | `React.ComponentProps<"div">`     | —       | 透传 `<div>` 原生属性               |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

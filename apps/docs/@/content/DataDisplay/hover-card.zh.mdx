# HoverCard 悬浮卡片

> 数据展示

鼠标悬停时出现的卡片浮层,比 Tooltip 承载更多内容。

## 何时使用

当你需要「悬浮卡片」相关能力时使用：鼠标悬停时出现的卡片浮层,比 Tooltip 承载更多内容。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

## 代码示例

### 基础用法

<CodeBlock code={`import { Card } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Card>
      <CardHeader>

    </Card>
  )
}`} lang="tsx" />

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(HoverCardTrigger, HoverCardContent),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

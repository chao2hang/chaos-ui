# Timeline 时间线

> 数据展示

垂直展示时间流信息,如操作日志、进程跟踪。

## 何时使用

需要时间线功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { Timeline, TimelineItem, TimelineDot, TimelineConnector, TimelineContent, TimelineTitle } from "@/components/ui/timeline"

export function Basic() {
  return (
    <Timeline>
      <TimelineContent>Content</TimelineContent>
    </Timeline>
  )
}`} lang="tsx" />

### 变体

<CodeBlock code={`{/* 展示 variant 变体 */}
<div className="flex flex-wrap items-center gap-3">
  <Timeline variant="default">Default</Timeline>
  <Timeline variant="success">Success</Timeline>
  <Timeline variant="warning">Warning</Timeline>
  <Timeline variant="destructive">Destructive</Timeline>
  <Timeline variant="info">Info</Timeline>
</div>`} lang="tsx" />


## 样式变体

**变体**

| Variant | 说明 | 默认 |
|---|---|---|
| `default` | 默认样式 | ✓ |
| `success` | 成功状态 |  |
| `warning` | 警告状态 |  |
| `destructive` | 危险/删除样式 |  |
| `info` | 信息提示 |  |

## API / Props

### Timeline

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 7 个子组件(TimelineItem, TimelineDot, TimelineConnector, TimelineContent, TimelineTitle…),使用组合模式按需导入。

# Card 卡片

> 数据展示 / Components for presenting data and content.

将信息聚合在卡片容器中展示,最通用的内容分组容器。
Groups content in a container; the most common content-grouping primitive.

## 何时使用 / When to Use

当你需要「卡片（Card）」解决具体交互/展示问题时使用；优先看下方示例与 API，按业务字段接入。

Use when you need Card for a concrete interaction; follow the examples and API below.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Card, CardHeader, CardTitle, CardContent, CardFooter } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
      </CardHeader>
      <CardContent>Card content goes here.</CardContent>
    </Card>
  )
}`} lang="tsx" />

## API / Props

### Card

| Prop        | Type                                                                      | Default     | 说明 / Description                                        |
| ----------- | ------------------------------------------------------------------------- | ----------- | --------------------------------------------------------- |
| `className` | `string`                                                                  | —           | 自定义样式类名 <br/> Custom CSS class                     |
| `size`      | `"default" \| "sm"`                                                       | `"default"` | 卡片尺寸 <br/> Card size                                  |
| `flush`     | `boolean`                                                                 | `false`     | 内容通栏 <br/> Edge-to-edge content                       |
| `shadow`    | `"none" \| "default" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "brand"` | `"default"` | 阴影变体（1.2.0+） <br/> Shadow variant                   |
| ...rest     | `React.HTMLAttributes`                                                    | —           | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

### shadow 阴影变体 / Shadow Variants

`shadow` prop 控制卡片的阴影深度，`brand` 变体使用品牌色阴影，适用于登录页等需要强调的场景：

<CodeBlock code={`import { Card, CardHeader, CardTitle, CardContent } from "@chaos_team/chaos-ui/ui"

export function ShadowDemo() {
  return (
    <Card shadow="brand">
      <CardHeader>
        <CardTitle>Brand Shadow Card</CardTitle>
      </CardHeader>
      <CardContent>登录页卡片，使用品牌色阴影。</CardContent>
    </Card>
  )
}`} lang="tsx" />

## 注意事项 / Notes

- **复合组件**: 包含 6 个子组件(CardHeader, CardFooter, CardTitle, CardAction, CardDescription…),使用组合模式按需导入。

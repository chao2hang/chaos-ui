# Card 卡片

Chaos UI 卡片容器组件，用于信息分组和内容容器化展示。
A card container component for information grouping and content presentation.

## 何时使用 / When to use

用于信息分块展示：用户资料卡、产品卡片、统计卡片、设置面板等。Card 提供统一的外观和可选的操作区。

Use for chunked information display: user profiles, product cards, stats cards, settings panels. Card provides consistent styling and optional action areas.

## 注意事项

- **卡片布局**：Card 渲染为 `<div>`，宽度默认撑满父容器。建议通过 `style={{ maxWidth }}` 或 Tailwind 类限制宽度。
- **CardAction**：可选的操作区，通常放更多操作菜单（`⋯` 按钮）。渲染在 CardHeader 的右上角。
- **嵌套**：Card 可以在 CardContent 内嵌套其他组件（Form、Table 等），不受限制。
- **无障碍**：Card 使用纯 `<div>` 容器，无特殊 ARIA 角色。如需交互式卡片，在外部包裹 `<button>` 或 `<a>` 或使用 `tabIndex` 和键盘事件。

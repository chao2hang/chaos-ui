# Descriptions

> Components for presenting data and content.

Key-value list of read-only fields; common on detail pages.

## When to Use

Use Descriptions when you need: Key-value list of read-only fields; common on detail pages. Use to present lists, status, metrics, and detail content for quick scanning.

**layout**

| layout       | Description | 默认 |
| ------------ | ----------- | ---- |
| `horizontal` |             | ✓    |

**Size**

| Size      | Description  | 默认 |

## API / Props

### Descriptions

| Prop         | Type                          | Default | Description                     |
| ------------ | ----------------------------- | ------- | ------------------------------- |
| `className`  | `string`                      | —       | Custom CSS class                |
| `items`      | `DescriptionItem[]`           | —       | Items / data list               |
| `column`     | `number`                      | `3`     | Column count                    |
| `labelWidth` | `string \| number`            | —       | Label width                     |
| ...rest      | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes |

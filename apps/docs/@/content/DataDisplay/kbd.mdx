# Kbd 键盘按键

> 数据展示 / Components for presenting data and content.

展示键盘按键样式的组件,用于快捷键提示。
Renders a keyboard-key style, used for shortcut hints.

## 何时使用 / When to Use

需要键盘按键功能时使用本组件。

Use when you need a kbd.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Kbd, KbdGroup } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Kbd>
      <KbdGroup>Content</KbdGroup>
    </Kbd>
  )
}`} lang="tsx" />

### 尺寸 / Sizes

<CodeBlock code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <Kbd size="sm">Sm</Kbd>
  <Kbd size="default">Default</Kbd>
  <Kbd size="lg">Lg</Kbd>
</div>`} lang="tsx" />


## 样式变体 / Style Variants

**尺寸 / Size**

| Size | 说明 / Description | 默认 |
|---|---|---|
| `sm` | 小号 / Small |  |
| `default` | 默认尺寸 / Default size | ✓ |
| `lg` | 大号 / Large |  |

## API / Props

### Kbd

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(KbdGroup),使用组合模式按需导入。

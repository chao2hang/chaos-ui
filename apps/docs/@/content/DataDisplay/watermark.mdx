# Watermark 水印

> 数据展示 / Components for presenting data and content.

在内容上叠加指定文字/图片水印,用于版权标识。
Overlays specified text/image watermark on content for copyright.

## 何时使用 / When to Use

当你需要「水印」相关能力时使用：在内容上叠加指定文字/图片水印,用于版权标识。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use Watermark when you need: Overlays specified text/image watermark on content for copyright. Use to present lists, status, metrics, and detail content for quick scanning.

## 样式变体 / Style Variants

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |

## API / Props

### Watermark

| Prop        | Type                          | Default | 说明 / Description                                          |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `content`   | `string`                      | —       | Text content for the watermark                              |
| `rotate`    | `number`                      | `-22`   | Rotation angle in degrees                                   |
| `gap`       | `number`                      | `100`   | Gap between repeated marks                                  |
| `opacity`   | `number`                      | `0.1`   | Opacity of the watermark                                    |
| `color`     | `string`                      | `#000`  | Text color                                                  |
| `fontSize`  | `number`                      | `14`    | Font size override                                          |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

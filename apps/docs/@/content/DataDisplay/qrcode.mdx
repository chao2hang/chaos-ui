# Qrcode 二维码

> 数据展示 / Components for presenting data and content.

二维码 — Chaos UI 的数据展示。
Qrcode — a components for presenting data and content. from Chaos UI.

## 何时使用 / When to Use

当你需要「二维码」相关能力时使用：二维码 — Chaos UI 的数据展示。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use Qrcode when you need: Qrcode — a components for presenting data and content. from Chaos UI. Use to present lists, status, metrics, and detail content for quick scanning.

## API / Props

### QRCode

| Prop               | Type                          | Default      | 说明 / Description                                          |
| ------------------ | ----------------------------- | ------------ | ----------------------------------------------------------- |
| `className`        | `string`                      | —            | 自定义样式类名 <br/> Custom CSS class                       |
| `value`            | `string`                      | —            | URL or text to encode                                       |
| `size`             | `number`                      | `200`        | Size in pixels                                              |
| `fgColor`          | `string`                      | `#000000`    | Foreground color                                            |
| `bgColor`          | `string`                      | `#ffffff`    | Background color                                            |
| `downloadable`     | `boolean`                     | `false`      | Include download button                                     |
| `downloadFileName` | `string`                      | `qrcode.png` | Filename for download                                       |
| `loading`          | `boolean`                     | `false`      | Loading state                                               |
| `error`            | `boolean`                     | `false`      | Error state                                                 |
| ...rest            | `React.ComponentProps<"div">` | —            | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

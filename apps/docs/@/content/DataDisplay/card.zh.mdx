# Card 卡片

> 数据展示

将信息聚合在卡片容器中展示,最通用的内容分组容器。

## 何时使用

将相关信息聚合展示,如商品、文章、用户等。


## 代码示例

### 基础用法

<CodeBlock code={`import { Card, CardHeader, CardTitle, CardContent, CardFooter } from "@/components/ui/card"

export function Basic() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
      </CardHeader>
      <CardContent>Card content goes here.</CardContent>
    </Card>
  )
}`} lang="tsx" />


## API / Props

### Card

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **复合组件**: 包含 6 个子组件(CardHeader, CardFooter, CardTitle, CardAction, CardDescription…),使用组合模式按需导入。

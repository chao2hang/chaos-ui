# EditableTable 可编辑表格

> 数据展示 / Data display components for presenting information.

可编辑表格 — 行内可编辑表格，支持增删改与批量操作。
EditableTable — Inline editable table with row add, edit, delete and batch operations.

## 何时使用 / When to Use

当你需要「可编辑表格」相关能力时使用：行内可编辑表格，支持增删改与批量操作。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use EditableTable when you need: Inline editable table with row add, edit, delete and batch operations. Use to present lists, status, metrics, and detail content for quick scanning.

# Statistic 统计数值

> 数据展示 / Components for presenting data and content.

突出展示某个统计数字,支持千分位、前缀/后缀、动画。
Prominently displays a statistic with thousands separators, prefix/suffix, and animation.

## 何时使用 / When to Use

当你需要「统计数值」相关能力时使用：突出展示某个统计数字,支持千分位、前缀/后缀、动画。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use Statistic when you need: Prominently displays a statistic with thousands separators, prefix/suffix, and animation. Use to present lists, status, metrics, and detail content for quick scanning.

## 样式变体 / Style Variants

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |

## API / Props

### Statistic

| Prop        | Type                          | Default | 说明 / Description                                          |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `value`     | `React.ReactNode`             | —       | The value to display                                        |
| `title`     | `React.ReactNode`             | —       | Label / title                                               |
| `prefix`    | `React.ReactNode`             | —       | Prefix before value                                         |
| `suffix`    | `React.ReactNode`             | —       | Suffix after value                                          |
| `precision` | `number`                      | —       | Precision for number formatting                             |
| `loading`   | `boolean`                     | `false` | Loading state shows skeleton                                |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

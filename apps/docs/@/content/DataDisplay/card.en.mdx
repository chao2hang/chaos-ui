# Card

> Components for presenting data and content.

Groups content in a container; the most common content-grouping primitive.

## When to Use

Group related information such as products, articles, or users.

## Examples

### Basic Usage

<CodeBlock code={`import { Card, CardHeader, CardTitle, CardContent, CardFooter } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
      </CardHeader>
      <CardContent>Card content goes here.</CardContent>
    </Card>
  )
}`} lang="tsx" />

## API / Props

### Card

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **复合组件**: 包含 6 个子组件(CardHeader, CardFooter, CardTitle, CardAction, CardDescription…),使用组合模式按需导入。

# Timeline

> Components for presenting data and content.

Vertical display of a chronological sequence such as logs or processes.

## When to Use

Use Timeline when you need: Vertical display of a chronological sequence such as logs or processes. Use to present lists, status, metrics, and detail content for quick scanning.

## Examples

### Basic Usage

<CodeBlock code={`import { Timeline, TimelineItem, TimelineDot, TimelineConnector, TimelineContent, TimelineTitle } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Timeline>
      <TimelineContent>Content</TimelineContent>
    </Timeline>
  )
}`} lang="tsx" />

## Style Variants

**Variant**

| Variant       | Description              | 默认 |
| ------------- | ------------------------ | ---- |
| `default`     | Default style            | ✓    |
| `success`     | Success style            |      |
| `warning`     | Warning style            |      |
| `destructive` | Destructive/danger style |      |
| `info`        | Informational style      |      |

## API / Props

### Timeline

| Prop        | Type                   | Default | Description                       |

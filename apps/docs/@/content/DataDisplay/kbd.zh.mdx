# Kbd 键盘按键

> 数据展示

展示键盘按键样式的组件,用于快捷键提示。

## 何时使用

需要键盘按键功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { Kbd, KbdGroup } from "@/components/ui/kbd"

export function Basic() {
  return (
    <Kbd>
      <KbdGroup>Content</KbdGroup>
    </Kbd>
  )
}`} lang="tsx" />

### 尺寸

<CodeBlock code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <Kbd size="sm">Sm</Kbd>
  <Kbd size="default">Default</Kbd>
  <Kbd size="lg">Lg</Kbd>
</div>`} lang="tsx" />


## 样式变体

**尺寸**

| Size | 说明 | 默认 |
|---|---|---|
| `sm` | 小号 |  |
| `default` | 默认尺寸 | ✓ |
| `lg` | 大号 |  |

## API / Props

### Kbd

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 1 个子组件(KbdGroup),使用组合模式按需导入。

# KpiPanel KPI 面板

> 数据展示

关键绩效指标面板,展示核心业务数值和趋势。

## 何时使用

需要把经营指标或趋势做成图表供扫读时使用。（组件：KPI 面板）

## 代码示例

### 基础用法

<CodeBlock code={`import { KPIPanel } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <KPIPanel>Default</KPIPanel>
}`} lang="tsx" />

### 尺寸

<CodeBlock
  code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <KPIPanel size="sm">Sm</KPIPanel>
  <KPIPanel size="default">Default</KPIPanel>
  <KPIPanel size="lg">Lg</KPIPanel>
</div>`}
  lang="tsx"
/>

## 样式变体

**尺寸**

| Size      | 说明     | 默认 |
| --------- | -------- | ---- |
| `sm`      | 小号     |      |
| `default` | 默认尺寸 | ✓    |
| `lg`      | 大号     |      |

## API / Props

### KPIPanel

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| `items`     | `KPIItem[]`            | —       | 选项/数据项列表   |
| `columns`   | `1 \| 2 \| 3 \| 4`     | `4`     | 列配置            |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。

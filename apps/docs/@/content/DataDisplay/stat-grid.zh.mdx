# StatGrid 指标网格

> 数据展示

指标网格 — 响应式指标网格，支持自动排列与趋势指示。

## 何时使用

当你需要「指标网格」相关能力时使用：响应式指标网格，支持自动排列与趋势指示。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

# Image

> Components for presenting data and content.

Enhanced image with preview, error fallback, and lazy loading.

## When to Use

Use when you need an image.

## Examples

### Basic Usage

<CodeBlock code={`import { Image } from "@/components/ui/image"

  return <Image />
}`} lang="tsx" />

## API / Props

### Image

| Prop        | Type                          | Default | Description                                      |
| ----------- | ----------------------------- | ------- | ------------------------------------------------ |
| `className` | `string`                      | —       | Custom CSS class                                 |
| `src`       | `string`                      | —       | Source URL                                       |
| `alt`       | `string`                      | —       | Alternative text                                 |
| `fallback`  | `React.ReactNode`             | —       | Fallback element shown while loading or on error |
| `width`     | `number \| string`            | —       | Width                                            |
| `height`    | `number \| string`            | —       | Height                                           |
| `onError`   | `() => void`                  | —       | Called when image fails to load                  |
| `onLoad`    | `() => void`                  | —       | Called when image loads successfully             |
| ...rest     | `React.ComponentProps<"div">` | —       | Inherits all `<div>` attributes                  |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

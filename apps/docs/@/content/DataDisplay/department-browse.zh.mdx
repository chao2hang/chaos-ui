# DepartmentBrowse 部门浏览

> 数据展示

从组织架构中选择部门/人员,支持树状展开、搜索、多选。

## 何时使用

需要部门浏览功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { DepartmentBrowse } from "@/components/ui/department-browse"

export function Basic() {
  return <DepartmentBrowse />
}`} lang="tsx" />

## API / Props

### DepartmentBrowse

| Prop           | Type                         | Default                | 说明              |
| -------------- | ---------------------------- | ---------------------- | ----------------- |
| `className`    | `string`                     | —                      | 自定义样式类名    |
| `value`        | `Department \| Department[]` | —                      | 当前值(受控)      |
| `defaultValue` | `Department \| Department[]` | —                      | 默认值(非受控)    |
| `placeholder`  | `string`                     | `Select department...` | 占位文本          |
| `disabled`     | `boolean`                    | —                      | 是否禁用          |
| `multiple`     | `boolean`                    | `false`                | 是否多选          |
| `maxCount`     | `number`                     | —                      |                   |
| `departments`  | `Department[]`               | —                      |                   |
| `onChange`     | `() => void`                 | —                      | 值变化回调        |
| ...rest        | `React.HTMLAttributes`       | —                      | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

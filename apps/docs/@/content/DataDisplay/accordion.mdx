# Accordion 手风琴

> 数据展示 / Components for presenting data and content.

可折叠/展开的内容面板,垂直堆叠,每次可展开一个或多个。
Vertically stacked collapsible panels; can expand one or more at a time.

## 何时使用 / When to Use

折叠面板展示大量可分组内容,按需展开查看。

Display grouped content that users can expand or collapse as needed.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Accordion type="single">
      <AccordionItem value="item-1">
        <AccordionTrigger>Section 1</AccordionTrigger>
        <AccordionContent>Content 1</AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}`} lang="tsx" />

## API / Props

### Accordion

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **复合组件**: 包含 3 个子组件(AccordionItem, AccordionTrigger, AccordionContent),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

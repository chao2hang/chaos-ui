## API / Props

### Countdown

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `target` | `Date \| number \| string` | — | Target date/time (ISO string or timestamp) |
| `onFinish` | `() => void` | — | Called when countdown reaches 0 |
| `format` | `string` | `HH:mm:ss` | Format: D=days, H=hours, M=minutes, S=seconds |
| ...rest | `React.ComponentProps<"span">` | — | 透传 `<span>` 原生属性 <br/> Inherits all `<span>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<span>` 原生 HTML 属性。

# Table 表格

Chaos UI 表格组件，基于原生 HTML `<table>` 元素，提供统一的样式和语义化结构。
A semantic table component built on native HTML `<table>` with unified styling.

## 何时使用 / When to use

用于展示结构化数据列表。适合简单的数据展示场景。对于复杂功能（排序、筛选、分页），使用 `DataTable` 或 `AdvancedDataTable`。

Use for displaying structured data. For complex features (sorting, filtering, pagination), use DataTable or AdvancedDataTable.

## API

| 子组件 | Props | 渲染为 | Description (zh) | Description (en) |
|---|---|---|---|---|
| Table | `React.ComponentProps<'table'>` | `<table>` | 表格根容器 | Table root container |
| TableHeader | `React.ComponentProps<'thead'>` | `<thead>` | 表头容器 | Table head container |
| TableBody | `React.ComponentProps<'tbody'>` | `<tbody>` | 表体容器 | Table body container |
| TableFooter | `React.ComponentProps<'tfoot'>` | `<tfoot>` | 表脚容器 | Table footer container |
| TableRow | `React.ComponentProps<'tr'>` | `<tr>` | 表格行 | Table row |
| TableHead | `React.ComponentProps<'th'>` | `<th>` | 表头单元格 | Table head cell |
| TableCell | `React.ComponentProps<'td'>` | `<td>` | 数据单元格 | Table data cell |
| TableCaption | `React.ComponentProps<'caption'>` | `<caption>` | 表格标题 | Table caption |

所有子组件均接受 `className` 自定义样式。

## 注意事项

- **vs DataTable**：`Table` 是纯展示组件，不含排序/筛选/分页逻辑。需要这些功能时请使用 `DataTable`（业务组件）。
- **colSpan / rowSpan**：TableCell 支持原生 `colSpan` 和 `rowSpan` 属性，用于合并单元格（如空状态行）。
- **响应式**：Table 默认不横向滚动。内容较宽时，建议包裹在 `ScrollArea` 中或使用 `overflow-x: auto` 容器。
- **无障碍**：使用原生 `<table>` 语义元素，屏幕阅读器自动识别表格结构。`TableCaption` 提供表格摘要。表头使用 `<th>` + `scope="col"` 提升语义。

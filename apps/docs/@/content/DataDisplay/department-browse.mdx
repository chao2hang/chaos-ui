# DepartmentBrowse 部门浏览

> 数据展示 / Components for presenting data and content.

从组织架构中选择部门/人员,支持树状展开、搜索、多选。
Browse org structure to select departments/people; supports tree expand, search, multi-select.

## 何时使用 / When to Use

需要部门浏览功能时使用本组件。

Use when you need a department browse.

# TreeView 树形视图

> 数据展示

树形视图 — Chaos UI 的数据展示。

## 何时使用

当你需要「树形视图」相关能力时使用：树形视图 — Chaos UI 的数据展示。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

# Tooltip 文字提示

> 数据展示

鼠标悬停时出现的简单文本提示,用于解释按钮/图标含义。

## 何时使用

鼠标悬停时展示补充说明信息,不占用常驻版面。

## 代码示例

### 基础用法

<CodeBlock code={`import { Tooltip, TooltipProvider, TooltipTrigger, TooltipContent } from "@/components/ui/tooltip"

export function Basic() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger>Hover me</TooltipTrigger>
        <TooltipContent>Tooltip content</TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}`} lang="tsx" />

## API / Props

### Tooltip

| Prop        | Type                   | Default | 说明              |
| ----------- | ---------------------- | ------- | ----------------- |
| `className` | `string`               | —       | 自定义样式类名    |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 3 个子组件(TooltipTrigger, TooltipContent, TooltipProvider),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

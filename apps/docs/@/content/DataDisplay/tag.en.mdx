## Style Variants

**Variant**

| Variant       | Description              | 默认 |
| ------------- | ------------------------ | ---- |
| `default`     | Default style            | ✓    |
| `secondary`   | Secondary action style   |      |
| `destructive` | Destructive/danger style |      |
| `outline`     | Outlined style           |      |
| `success`     | Success style            |      |
| `warning`     | Warning style            |      |
| `info`        | Informational style      |      |

**Size**

| Size      | Description  | 默认 |
| --------- | ------------ | ---- |
| `default` | Default size | ✓    |
| `sm`      | Small        |      |
| `lg`      | Large        |      |

**rounded**

| rounded   | Description | 默认 |
| --------- | ----------- | ---- |
| `default` |             | ✓    |
| `full`    |             |      |
| `none`    |             |      |

## API / Props

### Tag

| Prop        | Type                           | Default | Description                           |
| ----------- | ------------------------------ | ------- | ------------------------------------- |
| `className` | `string`                       | —       | Custom CSS class                      |
| `closable`  | `boolean`                      | `false` | Whether the tag can be closed/removed |
| `onClose`   | `() => void`                   | —       | Called when close button is clicked   |
| `icon`      | `React.ReactNode`              | —       | Icon element                          |
| ...rest     | `React.ComponentProps<"span">` | —       | Inherits all `<span>` attributes      |

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

# Avatar 头像

Chaos UI 头像组件，基于 Base UI Avatar，用于展示用户头像、图标或首字母缩写。
A Base UI-powered avatar for displaying user avatars, icons, or initials.

## 何时使用 / When to use

用于用户身份标识：个人资料、评论列表、团队成员展示、导航栏用户入口等。图片加载失败时自动回退到首字母缩写。

Use for user identification: profiles, comment lists, team member display, nav bar user entry. Falls back to initials when image fails to load.

## 注意事项

- **Fallback 机制**：`AvatarImage` 加载失败或未提供 `src` 时自动显示 `AvatarFallback`。通常填入用户名的首字母缩写。
- **Badge 定位**：`AvatarBadge` 自动定位在头像右下角，通过 `className` 设置背景色表达不同状态（在线/离开/离线）。
- **Group 叠放**：`AvatarGroup` 中的头像自动重叠排列，`AvatarGroupCount` 显示剩余数量。
- **无障碍**：Base UI Avatar 提供 `role="img"` 和 `aria-label` 支持。`AvatarImage` 的 `alt` 属性作为 fallback 时的描述文本。

# Heatmap 热力图

> 数据展示

热力图 — GitHub 风格日历热力图，支持色阶与单元格点击。

## 何时使用

当你需要「热力图」相关能力时使用：GitHub 风格日历热力图，支持色阶与单元格点击。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

# EditableDescriptions 可编辑描述

> 数据展示 / Data display components for presenting information.

可编辑描述 — 可编辑描述列表，支持字段行内编辑。
EditableDescriptions — Editable descriptions list with inline field editing.

## 何时使用 / When to Use

当你需要「可编辑描述」相关能力时使用：可编辑描述列表，支持字段行内编辑。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use EditableDescriptions when you need: Editable descriptions list with inline field editing. Use to present lists, status, metrics, and detail content for quick scanning.

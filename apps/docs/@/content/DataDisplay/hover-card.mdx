# HoverCard 悬浮卡片

> 数据展示 / Components for presenting data and content.

鼠标悬停时出现的卡片浮层,比 Tooltip 承载更多内容。
Card that appears on hover; holds more content than Tooltip.

## 何时使用 / When to Use

需要悬浮卡片功能时使用本组件。

Use when you need a hover card.

## 代码示例 / Examples

### 基础用法 / Basic Usage

<CodeBlock code={`import { Card } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
      </CardHeader>
      <CardContent>Card content goes here.</CardContent>
    </Card>
  )
}`} lang="tsx" />

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(HoverCardTrigger, HoverCardContent),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

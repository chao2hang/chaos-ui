# Descriptions 描述列表

> 数据展示

以键值对形式展示多个字段的只读详情,常见于详情页。

## 何时使用

需要描述列表功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { Descriptions } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return <Descriptions>Default</Descriptions>
}`} lang="tsx" />

### 尺寸

<CodeBlock code={`{/* 尺寸对照 */}
<div className="flex flex-wrap items-center gap-3">
  <Descriptions size="sm">Sm</Descriptions>
  <Descriptions size="default">Default</Descriptions>
  <Descriptions size="lg">Lg</Descriptions>
</div>`} lang="tsx" />

## API / Props

### Descriptions

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `items` | `DescriptionItem[]` | — | 选项/数据项列表 |
| `column` | `number` | `3` | Column count |
| `labelWidth` | `string \| number` | — | Label width |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 |

## 注意事项

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

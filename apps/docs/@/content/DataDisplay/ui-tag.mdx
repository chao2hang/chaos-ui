## 样式变体 / Style Variants

**变体 / Variant**

| Variant       | 说明 / Description                       | 默认 |
| ------------- | ---------------------------------------- | ---- |
| `default`     | 默认样式 / Default style                 | ✓    |
| `secondary`   | 次要样式 / Secondary action style        |      |
| `destructive` | 危险/删除样式 / Destructive/danger style |      |
| `outline`     | 描边样式 / Outlined style                |      |
| `success`     | 成功状态 / Success style                 |      |
| `warning`     | 警告状态 / Warning style                 |      |
| `info`        | 信息提示 / Informational style           |      |

**尺寸 / Size**

| Size      | 说明 / Description      | 默认 |
| --------- | ----------------------- | ---- |
| `default` | 默认尺寸 / Default size | ✓    |
| `sm`      | 小号 / Small            |      |
| `lg`      | 大号 / Large            |      |

**rounded / rounded**

| rounded   | 说明 / Description | 默认 |
| --------- | ------------------ | ---- |
| `default` |                    | ✓    |
| `full`    |                    |      |
| `none`    |                    |      |

## API / Props

### Tag

| Prop        | Type                           | Default | 说明 / Description                                            |
| ----------- | ------------------------------ | ------- | ------------------------------------------------------------- |
| `className` | `string`                       | —       | 自定义样式类名 <br/> Custom CSS class                         |
| `closable`  | `boolean`                      | `false` | Whether the tag can be closed/removed                         |
| `onClose`   | `() => void`                   | —       | Called when close button is clicked                           |
| `icon`      | `React.ReactNode`              | —       | Icon element                                                  |
| ...rest     | `React.ComponentProps<"span">` | —       | 透传 `<span>` 原生属性 <br/> Inherits all `<span>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **原生属性**: 继承所有 `<span>` 原生 HTML 属性。

# FilterChips 筛选条件条

> 数据展示 / Data display components for presenting information.

筛选条件条 — 已选筛选条件 chip 条，支持逐条删除与清空。
FilterChips — Active filter chips bar with per-chip removal and clear all.

## 何时使用 / When to Use

当你需要「筛选条件条」相关能力时使用：已选筛选条件 chip 条，支持逐条删除与清空。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use FilterChips when you need: Active filter chips bar with per-chip removal and clear all. Use to present lists, status, metrics, and detail content for quick scanning.

# JSONViewerUI JSON查看器

> 数据展示 / Data display components for presenting information.

JSON查看器 — 轻量 JSON 树查看器，支持折叠节点与语法高亮。
JSONViewerUI — Lightweight JSON tree viewer with collapsible nodes and syntax highlighting.

## 何时使用 / When to Use

当你需要「JSON查看器」相关能力时使用：轻量 JSON 树查看器，支持折叠节点与语法高亮。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use JSONViewerUI when you need: Lightweight JSON tree viewer with collapsible nodes and syntax highlighting. Use to present lists, status, metrics, and detail content for quick scanning.

# Badge

> Components for presenting data and content.

Small status/number indicator pinned to a corner, often used on notification icons, avatars, or list items.

## When to Use

Display status, tags, counts, or categorizations as compact pills.

## Examples

### Basic Usage

<CodeBlock code={`import { Badge } from "@/components/ui/badge"

export function Basic() {
  return <Badge>Default</Badge>
}`} lang="tsx" />

### Variants

<CodeBlock
  code={`{/* 展示 variant 变体 */}

## Style Variants

**Variant**

| Variant       | Description                 | 默认 |
| ------------- | --------------------------- | ---- |
| `default`     | Default style               | ✓    |
| `secondary`   | Secondary action style      |      |
| `destructive` | Destructive/danger style    |      |
| `outline`     | Outlined style              |      |
| `ghost`       | Ghost (no background) style |      |
| `link`        | Link style                  |      |

## API / Props

### Badge

| Prop        | Type                   | Default | Description                       |
| ----------- | ---------------------- | ------- | --------------------------------- |
| `className` | `string`               | —       | Custom CSS class                  |
| ...rest     | `React.HTMLAttributes` | —       | Inherits standard HTML attributes |

## Notes

- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

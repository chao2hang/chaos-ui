# ProgressiveImage 渐进式图片

> 数据展示 / Data display components for presenting information.

渐进式图片 — 渐进式图片加载，支持模糊到清晰交叉渐显。
ProgressiveImage — Progressive image loading with blur-up cross-fade.

## 何时使用 / When to Use

当你需要「渐进式图片」相关能力时使用：渐进式图片加载，支持模糊到清晰交叉渐显。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use ProgressiveImage when you need: Progressive image loading with blur-up cross-fade. Use to present lists, status, metrics, and detail content for quick scanning.

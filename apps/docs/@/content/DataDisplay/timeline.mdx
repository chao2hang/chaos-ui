# Timeline 时间线

> 数据展示 / Components for presenting data and content.

垂直展示时间流信息,如操作日志、进程跟踪。
Vertical display of a chronological sequence such as logs or processes.

## 何时使用 / When to Use

需要时间线功能时使用本组件。

Use when you need a timeline.

## 样式变体 / Style Variants

**变体 / Variant**

| Variant       | 说明 / Description                       | 默认 |
| ------------- | ---------------------------------------- | ---- |
| `default`     | 默认样式 / Default style                 | ✓    |
| `success`     | 成功状态 / Success style                 |      |
| `warning`     | 警告状态 / Warning style                 |      |
| `destructive` | 危险/删除样式 / Destructive/danger style |      |
| `info`        | 信息提示 / Informational style           |      |

## API / Props

### Timeline

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **复合组件**: 包含 7 个子组件(TimelineItem, TimelineDot, TimelineConnector, TimelineContent, TimelineTitle…),使用组合模式按需导入。

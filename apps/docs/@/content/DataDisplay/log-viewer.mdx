# LogViewer 日志查看器

> 数据展示 / Data display components for presenting information.

日志查看器 — 流式日志查看器，支持搜索、过滤与自动滚动。
LogViewer — Streaming log viewer with search, filter and auto-scroll.

## 何时使用 / When to Use

当你需要「日志查看器」相关能力时使用：流式日志查看器，支持搜索、过滤与自动滚动。用于展示列表、状态、统计与内容详情；让用户先看懂数据再操作。

Use LogViewer when you need: Streaming log viewer with search, filter and auto-scroll. Use to present lists, status, metrics, and detail content for quick scanning.

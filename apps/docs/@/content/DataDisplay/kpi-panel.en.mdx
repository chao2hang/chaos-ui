## API / Props

### KPIPanel

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | — | Custom CSS class |
| `items` | `KPIItem[]` | — | Items / data list |
| `columns` | `1 \| 2 \| 3 \| 4` | `4` | Column configuration |
| ...rest | `React.HTMLAttributes` | — | Inherits standard HTML attributes |

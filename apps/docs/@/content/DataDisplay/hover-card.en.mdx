# HoverCard

> Components for presenting data and content.

Card that appears on hover; holds more content than Tooltip.

## When to Use

Use HoverCard when you need: Card that appears on hover; holds more content than Tooltip. Use to present lists, status, metrics, and detail content for quick scanning.

## Examples

### Basic Usage

<CodeBlock code={`import { Card } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
      </CardHeader>
      <CardContent>Card content goes here.</CardContent>
    </Card>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(HoverCardTrigger, HoverCardContent),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

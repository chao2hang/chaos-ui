## 样式变体 / Style Variants

**变体 / Variant**

| Variant       | 说明 / Description                             | 默认 |
| ------------- | ---------------------------------------------- | ---- |
| `default`     | 默认样式 / Default style                       | ✓    |
| `secondary`   | 次要样式 / Secondary action style              |      |
| `destructive` | 危险/删除样式 / Destructive/danger style       |      |
| `outline`     | 描边样式 / Outlined style                      |      |
| `ghost`       | 幽灵(无背景)样式 / Ghost (no background) style |      |
| `link`        | 链接样式 / Link style                          |      |

## API / Props

### Badge

| Prop        | Type                   | Default | 说明 / Description                                        |
| ----------- | ---------------------- | ------- | --------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名 <br/> Custom CSS class                     |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **CVA 变体**: 通过 `class-variance-authority` 管理样式变体,类型安全。变体对象可导出复用。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

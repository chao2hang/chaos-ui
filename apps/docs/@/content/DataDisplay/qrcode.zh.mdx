## API / Props

### QRCode

| Prop               | Type                          | Default      | 说明                    |
| ------------------ | ----------------------------- | ------------ | ----------------------- |
| `className`        | `string`                      | —            | 自定义样式类名          |
| `value`            | `string`                      | —            | URL or text to encode   |
| `size`             | `number`                      | `200`        | Size in pixels          |
| `fgColor`          | `string`                      | `#000000`    | Foreground color        |
| `bgColor`          | `string`                      | `#ffffff`    | Background color        |
| `downloadable`     | `boolean`                     | `false`      | Include download button |
| `downloadFileName` | `string`                      | `qrcode.png` | Filename for download   |
| `loading`          | `boolean`                     | `false`      | Loading state           |
| `error`            | `boolean`                     | `false`      | Error state             |
| ...rest            | `React.ComponentProps<"div">` | —            | 透传 `<div>` 原生属性   |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

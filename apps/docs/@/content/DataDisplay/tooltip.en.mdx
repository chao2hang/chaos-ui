# Tooltip

> Components for presenting data and content.

Simple text bubble appearing on hover; used to explain button/icon meanings.

## When to Use

Reveal supplementary information on hover without occupying permanent space.


## Examples

### Basic Usage

<CodeBlock code={`import { Tooltip, TooltipProvider, TooltipTrigger, TooltipContent } from "@chaos_team/chaos-ui/ui"

export function Basic() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger>Hover me</TooltipTrigger>
        <TooltipContent>Tooltip content</TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}`} lang="tsx" />

## Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 3 个子组件(TooltipTrigger, TooltipContent, TooltipProvider),使用组合模式按需导入。
- **Base UI**: 基于 `@base-ui/react` 无头(headless)原语构建,无障碍(a11y)开箱即用。

# RegionLayout 区域布局

> 系统布局

区域布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

当你需要「区域布局」相关能力时使用：区域布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

## API / Props

### RegionLayout

| Prop           | Type                          | Default | 说明                  |
| -------------- | ----------------------------- | ------- | --------------------- |
| `className`    | `string`                      | —       | 自定义样式类名        |
| `header`       | `React.ReactNode`             | —       |                       |
| `left`         | `React.ReactNode`             | —       |                       |
| `right`        | `React.ReactNode`             | —       |                       |
| `footer`       | `React.ReactNode`             | —       |                       |
| `leftWidth`    | `string`                      | `240px` |                       |
| `rightWidth`   | `string`                      | `240px` |                       |
| `headerHeight` | `string`                      | `56px`  |                       |
| `footerHeight` | `string`                      | `48px`  |                       |
| ...rest        | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

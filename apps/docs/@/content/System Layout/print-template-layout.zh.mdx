# PrintTemplateLayout Print Template Layout

> 系统布局

Print Template Layout — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要统一后台页面框架或分步/分区布局时使用。（组件：Print Template Layout）

## 代码示例

### 基础用法

<CodeBlock code={`import { PrintTemplateLayout } from "@chaos_team/chaos-ui/layout"

export function Basic() {
  return <PrintTemplateLayout />
}`} lang="tsx" />

### PrintTemplateLayout

| Prop        | Type                   | Default | 说明                              |

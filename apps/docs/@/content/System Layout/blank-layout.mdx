# BlankLayout 空白布局

> 系统布局 / System-level layout shells for admin pages.

空白布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
BlankLayout — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

当你需要「空白布局」相关能力时使用：空白布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

Use BlankLayout when you need: BlankLayout — a system-level layout shell for admin/dashboard/auth pages. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

## API / Props

### BlankLayout

| Prop        | Type                          | Default | 说明 / Description                                          |
| ----------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className` | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `centered`  | `boolean`                     | `false` |                                                             |
| `padded`    | `boolean`                     | `true`  |                                                             |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

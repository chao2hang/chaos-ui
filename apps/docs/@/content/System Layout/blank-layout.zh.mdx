# BlankLayout 空白布局

> 系统布局

空白布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要空白布局功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { BlankLayout } from "@/components/layout/blank-layout"

export function Basic() {
  return <BlankLayout />
}`} lang="tsx" />


## API / Props

### BlankLayout

| Prop | Type | Default | 说明 |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 |
| `centered` | `boolean` | `false` |  |
| `padded` | `boolean` | `true` |  |
| ...rest | `React.ComponentProps<"div">` | — | 透传 `<div>` 原生属性 |

## 注意事项

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

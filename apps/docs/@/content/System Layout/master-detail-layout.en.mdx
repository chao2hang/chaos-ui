# MasterDetailLayout

> System-level layout shells for admin pages.

MasterDetailLayout — a system-level layout shell for admin/dashboard/auth pages.

## When to Use

Use when you need a master detail layout.

## Examples

### Basic Usage

<CodeBlock code={`import { MasterDetailLayout } from "@chaos_team/chaos-ui/layout"

| `detail`             | `React.ReactNode`             | —       |                                 |
| `masterWidth`        | `number`                      | —       |                                 |

| `minMasterWidth`     | `number`                      | `240`   |                                 |

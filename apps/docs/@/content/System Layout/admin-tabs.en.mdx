# AdminTabs

> System-level layout shells for admin pages.

AdminTabs — a system-level layout shell for admin/dashboard/auth pages.

## When to Use

Use AdminTabs when you need: AdminTabs — a system-level layout shell for admin/dashboard/auth pages. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

## Examples

### Basic Usage

<CodeBlock code={`import { NavigationTabsBar, AdminTabs, NavigationTabsBarTabItem, AdminTabItem } from "@chaos_team/chaos-ui/layout"

## API / Props

### ArticleLayout

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `title` | `React.ReactNode` | — | Optional article header title rendered inside an `<header>`. |
| `toc` | `React.ReactNode` | — | Optional table-of-contents rendered as a sticky `<aside>`. |
| `footer` | `React.ReactNode` | — | Optional article footer. |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

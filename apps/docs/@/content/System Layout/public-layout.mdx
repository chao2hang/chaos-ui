# PublicLayout 公共布局

> 系统布局 / System-level layout shells for admin pages.

公共布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
PublicLayout — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

需要公共布局功能时使用本组件。

Use when you need a public layout.

## API / Props

### PublicLayout

| Prop            | Type                          | Default | 说明 / Description                                          |
| --------------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className`     | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `logo`          | `React.ReactNode`             | —       |                                                             |
| `nav`           | `Array<T>`                    | —       |                                                             |
| `footer`        | `React.ReactNode`             | —       |                                                             |
| `headerActions` | `React.ReactNode`             | —       |                                                             |
| ...rest         | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

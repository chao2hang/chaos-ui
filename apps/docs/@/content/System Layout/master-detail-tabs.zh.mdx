# MasterDetailTabs 主次标签布局

> 系统布局

主次标签布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要主次标签布局功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { MasterDetailTabs } from "@/components/layout/master-detail-tabs"

export function Basic() {
  return <MasterDetailTabs />
}`} lang="tsx" />

## API / Props

### MasterDetailTabs

| Prop               | Type                          | Default | 说明                  |
| ------------------ | ----------------------------- | ------- | --------------------- |
| `className`        | `string`                      | —       | 自定义样式类名        |
| `items`            | `MasterDetailTabItem[]`       | —       | 选项/数据项列表       |
| `activeKey`        | `string`                      | —       |                       |
| `defaultActiveKey` | `string`                      | —       |                       |
| `onChange`         | `() => void`                  | —       | 值变化回调            |
| ...rest            | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# AuthLayout 认证布局

> 系统布局

认证布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

当你需要「认证布局」相关能力时使用：认证布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

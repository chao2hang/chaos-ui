# NavigationTabsBar Navigation Tabs Bar

> 系统布局 / System-level layout shells for admin pages.

Tab items / Tab 项列表
NavigationTabsBar — navigation tabs bar component.

## 何时使用 / When to Use

当你需要「Navigation Tabs Bar」相关能力时使用：Tab items / Tab 项列表。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

Use NavigationTabsBar when this capability is required. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

## API / Props

### NavigationTabsBar

| Prop               | Type                         | Default | 说明 / Description                                                   |
| ------------------ | ---------------------------- | ------- | -------------------------------------------------------------------- |
| `className`        | `string`                     | —       | 自定义样式类名 <br/> Custom CSS class                                |
| `items`            | `NavigationTabsBarTabItem[]` | —       | Tab items / Tab 项列表                                               |
| `activeKey`        | `string`                     | —       | Active tab key / 当前激活的 tab key                                  |
| `defaultActiveKey` | `string`                     | —       | Default active tab key (uncontrolled) / 默认激活的 tab key（非受控） |
| `onChange`         | `() => void`                 | —       | Tab change callback / Tab 切换回调                                   |
| `onClose`          | `() => void`                 | —       | Tab close callback / Tab 关闭回调                                    |
| `onCloseAll`       | `() => void`                 | —       | Close all tabs callback / 关闭全部回调                               |
| `onCloseOthers`    | `() => void`                 | —       | Close other tabs callback / 关闭其他回调                             |
| `onCloseToRight`   | `() => void`                 | —       | Close tabs to the right callback / 关闭右侧回调                      |
| `onRefresh`        | `() => void`                 | —       | Refresh tab callback / 刷新回调                                      |
| ...rest            | `React.HTMLAttributes`       | —       | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes            |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

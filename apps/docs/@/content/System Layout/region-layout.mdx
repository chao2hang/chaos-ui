# RegionLayout 区域布局

> 系统布局 / System-level layout shells for admin pages.

区域布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
RegionLayout — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

需要区域布局功能时使用本组件。

Use when you need a region layout.

## API / Props

### RegionLayout

| Prop           | Type                          | Default | 说明 / Description                                          |
| -------------- | ----------------------------- | ------- | ----------------------------------------------------------- |
| `className`    | `string`                      | —       | 自定义样式类名 <br/> Custom CSS class                       |
| `header`       | `React.ReactNode`             | —       |                                                             |
| `left`         | `React.ReactNode`             | —       |                                                             |
| `right`        | `React.ReactNode`             | —       |                                                             |
| `footer`       | `React.ReactNode`             | —       |                                                             |
| `leftWidth`    | `string`                      | `240px` |                                                             |
| `rightWidth`   | `string`                      | `240px` |                                                             |
| `headerHeight` | `string`                      | `56px`  |                                                             |
| `footerHeight` | `string`                      | `48px`  |                                                             |
| ...rest        | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 <br/> Inherits all `<div>` attributes |

## 注意事项 / Notes

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

# PublicLayout

> System-level layout shells for admin pages.

PublicLayout — a system-level layout shell for admin/dashboard/auth pages.

## When to Use

Use PublicLayout when you need: PublicLayout — a system-level layout shell for admin/dashboard/auth pages. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

## API / Props

### SplitScreen

| Prop        | Type                         | Default      | 说明 / Description                                        |
| ----------- | ---------------------------- | ------------ | --------------------------------------------------------- |
| `className` | `string`                     | —            | 自定义样式类名 <br/> Custom CSS class                     |
| `left`      | `React.ReactNode`            | —            | Left (or top) pane content.                               |
| `right`     | `React.ReactNode`            | —            | Right (or bottom) pane content.                           |
| `direction` | `"horizontal" \| "vertical"` | `horizontal` | Layout direction. Defaults to `horizontal`.               |
| ...rest     | `React.HTMLAttributes`       | —            | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

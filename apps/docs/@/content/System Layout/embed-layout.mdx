# EmbedLayout Embed Layout

> 系统布局 / System layout

嵌入第三方宿主（iframe / 卡片 / 抽屉）的全高框架：默认 **填满父级高度**（`h-full`），可选顶栏 + 可滚动主体。  
Full-height frame for third-party hosts: fills the **parent** (`h-full`) by default.

## 何时使用 / When to Use

需要把业务视图嵌进固定高度的宿主容器，且顶栏固定、正文独立滚动时使用。  
Use when embedding into a fixed-height host with a sticky header and scrollable body.

## API / Props

### EmbedLayout

| Prop        | Type              | Default | 说明 / Description                                                 |
| ----------- | ----------------- | ------- | ------------------------------------------------------------------ |
| `className` | `string`          | —       | 根节点类名；可覆盖高度（如 `h-svh`） / Root class; override height |
| `header`    | `React.ReactNode` | —       | 可选顶栏 / Optional top bar                                        |
| `children`  | `React.ReactNode` | —       | 主体（`flex-1` + `overflow-auto`） / Body                          |

## 注意事项 / Notes

- **高度由宿主决定 / Host sets height**：默认 `h-full`，不是 `h-screen`。
- 视口全高 / Viewport full height：`className="h-svh"` 或 `h-screen`。
- **Client Component**: `"use client"`.

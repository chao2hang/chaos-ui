# AdminSider 管理后台侧边栏

> 系统布局

管理后台侧边栏 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要管理后台侧边栏功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { AdminSider } from "@/components/layout/admin-sider"

export function Basic() {
  return <AdminSider />
}`} lang="tsx" />

## API / Props

### AdminSider

| Prop        | Type                            | Default | 说明                    |
| ----------- | ------------------------------- | ------- | ----------------------- |
| `className` | `string`                        | —       | 自定义样式类名          |
| `items`     | `AdminSiderItem[]`              | —       | 选项/数据项列表         |
| `activeKey` | `string`                        | —       |                         |
| `onSelect`  | `() => void`                    | —       | 选中回调                |
| ...rest     | `React.ComponentProps<"aside">` | —       | 透传 `<aside>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<aside>` 原生 HTML 属性。

# AdminSider 管理后台侧边栏

> 系统布局

管理后台侧边栏 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

当你需要「管理后台侧边栏」相关能力时使用：管理后台侧边栏 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

## API / Props

### AdminSider

| Prop                 | Type                            | Default | 说明                    |
| -------------------- | ------------------------------- | ------- | ----------------------- |
| `className`          | `string`                        | —       | 自定义样式类名          |
| `menuItems`          | `MenuItem[]`                    | —       | 菜单项列表              |
| `selectedKey`        | `string`                        | —       | 选中的菜单 key          |
| `onItemClick`        | `(item: MenuItem) => void`      | —       | 菜单点击回调            |
| `collapsed`          | `boolean`                       | `false` | 是否折叠                |
| `onCollapse`         | `(collapsed: boolean) => void`  | —       | 折叠变更回调            |
| `logo`               | `ReactNode`                     | —       | Logo 内容               |
| `footer`             | `ReactNode`                     | —       | 底部内容                |
| `width`              | `number`                        | `240`   | 展开宽度                |
| `collapsedWidth`     | `number`                        | `64`    | 折叠宽度                |
| `mobileOpen`         | `boolean`                       | `false` | 移动端是否打开          |
| `onMobileOpenChange` | `(open: boolean) => void`       | —       | 移动端打开变更回调      |
| ...rest              | `React.ComponentProps<"aside">` | —       | 透传 `<aside>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<aside>` 原生 HTML 属性。

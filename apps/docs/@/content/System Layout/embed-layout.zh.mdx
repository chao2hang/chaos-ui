# EmbedLayout Embed Layout

> 系统布局

Embed Layout — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

适用于Embed Layout相关场景。

## 代码示例

### 基础用法

<CodeBlock code={`import { EmbedLayout } from "@/components/layout/embed-layout"

export function Basic() {
  return <EmbedLayout />
}`} lang="tsx" />

## API / Props

### EmbedLayout

| Prop        | Type                   | Default | 说明                                                           |
| ----------- | ---------------------- | ------- | -------------------------------------------------------------- |
| `className` | `string`               | —       | 自定义样式类名                                                 |
| `header`    | `React.ReactNode`      | —       | Optional top bar (title/actions) rendered inside a `<header>`. |
| ...rest     | `React.HTMLAttributes` | —       | 透传原生 DOM 属性                                              |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

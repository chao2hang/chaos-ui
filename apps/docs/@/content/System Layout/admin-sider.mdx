# AdminSider 管理后台侧边栏

> 系统布局 / System-level layout shells for admin pages.

管理后台侧边栏 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
AdminSider — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

需要管理后台侧边栏功能时使用本组件。

Use when you need an admin sider.

## API / Props

### AdminSider

| Prop                 | Type                            | Default | 说明 / Description                                              |
| -------------------- | ------------------------------- | ------- | --------------------------------------------------------------- |
| `className`          | `string`                        | —       | 自定义样式类名 <br/> Custom CSS class                           |
| `menuItems`          | `MenuItem[]`                    | —       | 菜单项列表 <br/> Menu items                                     |
| `selectedKey`        | `string`                        | —       | 选中的菜单 key <br/> Selected menu key                          |
| `onItemClick`        | `(item: MenuItem) => void`      | —       | 菜单点击回调 <br/> Menu click handler                           |
| `collapsed`          | `boolean`                       | `false` | 是否折叠 <br/> Whether collapsed                                |
| `onCollapse`         | `(collapsed: boolean) => void`  | —       | 折叠变更回调 <br/> Collapse change callback                     |
| `logo`               | `ReactNode`                     | —       | Logo 内容 <br/> Logo content                                    |
| `footer`             | `ReactNode`                     | —       | 底部内容 <br/> Footer content                                   |
| `width`              | `number`                        | `240`   | 展开宽度 <br/> Expanded width                                   |
| `collapsedWidth`     | `number`                        | `64`    | 折叠宽度 <br/> Collapsed width                                  |
| `mobileOpen`         | `boolean`                       | `false` | 移动端是否打开 <br/> Mobile open state                          |
| `onMobileOpenChange` | `(open: boolean) => void`       | —       | 移动端打开变更回调 <br/> Mobile open change callback            |
| ...rest              | `React.ComponentProps<"aside">` | —       | 透传 `<aside>` 原生属性 <br/> Inherits all `<aside>` attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<aside>` 原生 HTML 属性。

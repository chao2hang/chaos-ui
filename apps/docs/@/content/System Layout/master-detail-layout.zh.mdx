# MasterDetailLayout 主次布局

> 系统布局

主次布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要主次布局功能时使用本组件。

## 代码示例

### 基础用法

<CodeBlock code={`import { MasterDetailLayout } from "@/components/layout/master-detail-layout"

export function Basic() {
  return <MasterDetailLayout />
}`} lang="tsx" />

## API / Props

### MasterDetailLayout

| Prop                 | Type                          | Default | 说明                  |
| -------------------- | ----------------------------- | ------- | --------------------- |
| `className`          | `string`                      | —       | 自定义样式类名        |
| `master`             | `React.ReactNode`             | —       |                       |
| `detail`             | `React.ReactNode`             | —       |                       |
| `masterWidth`        | `number`                      | —       |                       |
| `defaultMasterWidth` | `number`                      | `320`   |                       |
| `minMasterWidth`     | `number`                      | `240`   |                       |
| `maxMasterWidth`     | `number`                      | `480`   |                       |
| `resizable`          | `boolean`                     | `true`  |                       |
| ...rest              | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

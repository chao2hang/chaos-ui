# DashboardLayout 仪表盘布局

> 系统布局

仪表盘布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

仪表盘页面整体布局(侧边栏+头部+内容区)。

## 代码示例

### 基础用法

<CodeBlock code={`import { DashboardLayout } from "@/components/layout/dashboard-layout"

  return <DashboardLayout />

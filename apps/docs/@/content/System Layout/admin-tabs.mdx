# AdminTabs 管理后台标签页

> 系统布局 / System-level layout shells for admin pages.

管理后台标签页 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
AdminTabs — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

需要管理后台标签页功能时使用本组件。

Use when you need an admin tabs.

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(NavigationTabsBarTabItem, AdminTabItem),使用组合模式按需导入。

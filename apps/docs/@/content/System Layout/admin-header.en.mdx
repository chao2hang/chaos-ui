# AdminHeader

> System-level layout shells for admin pages.

AdminHeader — a system-level layout shell for admin/dashboard/auth pages.

## When to Use

Use AdminHeader when you need: AdminHeader — a system-level layout shell for admin/dashboard/auth pages. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

# PrintLayout 打印布局

> 系统布局 / System-level layout shells for admin pages.

打印布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。
PrintLayout — a system-level layout shell for admin/dashboard/auth pages.

## 何时使用 / When to Use

当你需要「打印布局」相关能力时使用：打印布局 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

Use PrintLayout when you need: PrintLayout — a system-level layout shell for admin/dashboard/auth pages. Use for admin shells (sider, tabs, header) to keep multi-page navigation consistent.

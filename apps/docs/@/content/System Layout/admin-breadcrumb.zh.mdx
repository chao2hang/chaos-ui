# AdminBreadcrumb 管理后台面包屑

> 系统布局

管理后台面包屑 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

当你需要「管理后台面包屑」相关能力时使用：管理后台面包屑 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。用于后台壳层与系统级布局（侧栏、页签、顶栏）；保证多页应用导航一致。

## API / Props

### AdminBreadcrumb

| Prop        | Type                          | Default | 说明                  |
| ----------- | ----------------------------- | ------- | --------------------- |
| `className` | `string`                      | —       | 自定义样式类名        |
| `items`     | `AdminBreadcrumbItem[]`       | —       | 选项/数据项列表       |
| `separator` | `React.ReactNode`             | `/`     |                       |
| ...rest     | `React.ComponentProps<"div">` | —       | 透传 `<div>` 原生属性 |

## 注意事项

- **原生属性**: 继承所有 `<div>` 原生 HTML 属性。

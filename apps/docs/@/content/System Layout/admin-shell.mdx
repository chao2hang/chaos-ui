## API / Props

### AdminShell

| Prop | Type | Default | 说明 / Description |
|---|---|---|---|
| `className` | `string` | — | 自定义样式类名 <br/> Custom CSS class |
| `menuItems` | `MenuItem[]` | — | Menu items for the sidebar / 侧栏菜单项 |
| `selectedMenuKey` | `string` | — | Selected menu key / 选中菜单 key |
| `onMenuItemClick` | `() => void` | — | Menu item click callback / 菜单项点击回调 |
| `logo` | `React.ReactNode` | — | Logo displayed in header and sider / 头部和侧栏的 Logo |
| `siderFooter` | `React.ReactNode` | — | Footer content in sidebar / 侧栏底部内容 |
| `sidebarCollapsible` | `boolean` | `true` | Whether sidebar is collapsible / 侧栏是否可折叠 |
| `defaultCollapsed` | `boolean` | `false` | Default collapsed state / 默认折叠状态 |
| `sidebarWidth` | `number` | `240` | Sidebar expanded width / 展开宽度 |
| `collapsedWidth` | `number` | `64` | Sidebar collapsed width / 折叠宽度 |
| `breadcrumb` | `AdminHeaderProps["breadcrumb"]` | — | Breadcrumb items / 面包屑项 |
| `showSearch` | `boolean` | `true` | Whether to show search in header / 是否在头部显示搜索 |
| `searchPlaceholder` | `string` | `Search...` | Search placeholder / 搜索占位文本 |
| `onSearch` | `() => void` | — | Search callback / 搜索回调 |
| `headerActions` | `React.ReactNode` | — | Extra actions in header / 头部额外操作 |
| `user` | `UserMenuUser` | — | Current user info / 当前用户信息 |
| ...rest | `React.HTMLAttributes` | — | 透传原生 DOM 属性 <br/> Inherits standard HTML attributes |

## 注意事项 / Notes

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。

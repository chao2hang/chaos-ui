# AdminTabs 管理后台标签页

> 系统布局

管理后台标签页 — 系统级布局外壳,用于后台/仪表盘/认证等整页布局。

## 何时使用

需要管理后台标签页功能时使用本组件。


## 代码示例

### 基础用法

<CodeBlock code={`import { NavigationTabsBar, AdminTabs, NavigationTabsBarTabItem, AdminTabItem } from "@/components/layout/admin-tabs"

export function Basic() {
  return (
    <AdminTabs>
      <AdminTabs>Content</AdminTabs>
    </AdminTabs>
  )
}`} lang="tsx" />


## API / Props

### AdminTabs

| Prop | Type | Default | 说明 |

## 注意事项

- **Client Component**: 使用 `"use client"`,依赖 React hooks/状态,需在客户端渲染。
- **复合组件**: 包含 2 个子组件(NavigationTabsBarTabItem, AdminTabItem),使用组合模式按需导入。

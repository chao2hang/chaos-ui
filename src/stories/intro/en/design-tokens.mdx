# Design Tokens

Chaos UI uses CSS custom properties for all design tokens, enabling consistent theming and easy customization.

## Color Tokens

### Semantic Colors

| Token | Usage | CSS Variable |
|-------|-------|--------------|
| `background` | Page background | `--background` |
| `foreground` | Default text | `--foreground` |
| `primary` | Primary actions | `--primary` |
| `primary-foreground` | Text on primary | `--primary-foreground` |
| `secondary` | Secondary actions | `--secondary` |
| `secondary-foreground` | Text on secondary | `--secondary-foreground` |
| `muted` | Subtle backgrounds | `--muted` |
| `muted-foreground` | Subtle text | `--muted-foreground` |
| `accent` | Hover states | `--accent` |
| `accent-foreground` | Text on accent | `--accent-foreground` |
| `destructive` | Destructive actions | `--destructive` |
| `destructive-foreground` | Text on destructive | `--destructive-foreground` |

### Status Colors

| Token | Usage | CSS Variable |
|-------|-------|--------------|

| `success-foreground` | Text on success | `--success-foreground` |
| `warning` | Warning states | `--warning` |

### Border & Input

| Token | Usage | CSS Variable |
|-------|-------|--------------|

## Spacing Scale

Chaos UI uses Tailwind's default spacing scale:

| Class | Size | Pixels |
|-------|------|--------|
| `p-1` | 1 | 4px |
| `p-2` | 2 | 8px |
| `p-3` | 3 | 12px |
| `p-4` | 4 | 16px |
| `p-6` | 6 | 24px |
| `p-8` | 8 | 32px |

## Border Radius

| Class | Value | Usage |
|-------|-------|-------|
| `rounded-sm` | `radius * 0.6` | Small elements |
| `rounded-md` | `radius * 0.8` | Inputs, selects |
| `rounded-lg` | `radius` | Cards, dialogs |
| `rounded-xl` | `radius * 1.4` | Modals |
| `rounded-2xl` | `radius * 1.8` | Large surfaces |
| `rounded-full` | 9999px | Pills, avatars |

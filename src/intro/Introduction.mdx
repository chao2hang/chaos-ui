# Chaos UI Design System

Enterprise-grade component design system built with React 19, TypeScript 5.9, and Tailwind CSS 4.

## Overview

Chaos UI is a comprehensive component library designed for building modern web applications. It provides:

- **60+ UI Components** - Production-ready base components
- **14+ Business Components** - Domain-specific components
- **3+ Layout Components** - Page-level layouts
- **Full TypeScript Support** - Strict type safety
- **Dark Mode Ready** - Built-in dark mode support
- **AI-Friendly Documentation** - Optimized for LLM consumption

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React 19 | UI framework |
| TypeScript 5.9 | Type safety |
| Tailwind CSS 4 | Styling |
| @base-ui/react | Base UI primitives |
| shadcn/ui | Component patterns |
| Lucide React | Icons |
| Storybook 10 | Documentation |

## Quick Start

```bash
# Install dependencies
npm install

# Start Storybook
npm run dev

# Build Storybook for production
npm run build-storybook
```
